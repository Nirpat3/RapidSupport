# How to add item in item movement_

**How to add item in Item Movement?**

Item movement is a module for adding item directly open order or move the item in department, sub-department, vendor, group, tax, tag, EBT, etc.

But for moving item in different places first need to add item in item movement list.

- Open rapid RMS application.

- Go to item movement module.

- Select search.

![](media/image1.png){width="6.5in" height="4.694444444444445in"}

- All items available in store will display.

- Select items from the list.

- Apply to add.

![](media/image2.png){width="6.5in" height="4.75in"}

- Items will display in item movement list.

![](media/image3.png){width="6.5in" height="4.75in"}
