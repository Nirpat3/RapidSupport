# How to delete QSR group

**How to delete QSR group ?**

When user want to delete group then it is possible by deleting QSR group from inventory management module.

Follow below procedure to delete qsr group.

- Open inventory management.

- Go to QSR group tab.

- There should be list of available qsr group.

- Open qsr group.

- At bottom there is one delete button available.

- Click on delete button.

- QSR group will delete successfully.

![](media/image1.png){width="6.5in" height="4.736111111111111in"}

- Deleted group will disappear from QSR group list.

- As qsr group deleted, deleted qsr group should remove from linked modifier item and from qsr sub group.
