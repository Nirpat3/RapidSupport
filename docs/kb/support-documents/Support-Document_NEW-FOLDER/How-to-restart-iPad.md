# How to restart iPad

**How to restart the iPad?**

- Hit setting icon available on iPad dashboard.

![](media/image2.png){width="6.5in" height="4.174930008748906in"}

- From the side, panel click on general.

- Go in sequence accessibility -switch control - menu items- device-restart

- Set restart option selected.

![](media/image3.png){width="6.5in" height="4.502567804024497in"}

- Now go to the dashboard option and click on the shortcut menu.

- Go to sequence device-restart.

![](media/image1.png){width="6.5in" height="4.835557742782152in"}

- When hit restart then popup would display keep allow option.

- Ipad would restart.
