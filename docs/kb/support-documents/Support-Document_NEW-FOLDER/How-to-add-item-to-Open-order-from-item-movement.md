# How to add item to Open order from item movement

**How to add an item to open order from item movement?**

When the user wants to move the item to purchase order then one of the possible ways is from item movement.

- Open rapid RMS application.

- Go to item movement.

- Add items in item movement list from search option.

![](media/image1.png){width="6.5in" height="4.666666666666667in"}

- From list select items and apply option add to open order.

![](media/image2.png){width="6.5in" height="4.763888888888889in"}

- Pop up will blink with all available open order in the system.

- Select anyone from that.

- At a time you can select only one open order.

- Now apply yes.

![](media/image3.png){width="6.5in" height="4.777777777777778in"}

- Continue with process.

- Item will be added to open order.

- Items which are added to open orders will display with PO symbol.

![](media/image4.png){width="6.5in" height="4.763888888888889in"}
