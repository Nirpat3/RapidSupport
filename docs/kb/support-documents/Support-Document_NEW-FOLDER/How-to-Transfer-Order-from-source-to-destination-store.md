# How to Transfer Order from source to destination store

**How to Transfer Order from source to destination store?**

**-\>** It Is used when source store wants to send items to destination store.

To Transfer Order follow below steps :

- Open Rapid RMS app.

- Go to the Inventory Transfer module.

- Go to the Open Transfer Order section.

- Now open any transfer order which was Generated from Generate order or created by shortcut from Open Transfer Order.

![](media/image5.png){width="6.5in" height="4.722222222222222in"}

- From Transfer order items page, tap on 'send Qty' for set send Qty which source store want to send.

![](media/image6.png){width="6.5in" height="4.666666666666667in"}

- After Set Qty selects items and clicks on Delivery Button.

- Now Enter Invoice number and tap on save button.

![](media/image2.png){width="6.5in" height="4.680555555555555in"}

- Now you will see transfer Order In 'In Route Order' module of inventory transfer and 'Receiving pending' of PO module.

-\> Inventory Transfer -\> In-route order -\> In route transfer order list.

![](media/image7.png){width="6.5in" height="4.666666666666667in"}
