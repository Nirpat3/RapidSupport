# How do I delete multiple items at once from the imported edi file

**How do I delete multiple items at once from the imported EDI file?**

When stores want to delete multiple items at once from reconcile page then they can use remove option.

Follow below steps to delete multiple items.

- Open any browser, Go to [[https://www.rapidrms.com/Account/Branchlogin]{.underline}](https://www.rapidrms.com/Account/Branchlogin)

- In the Sign in page, Login with your store credential.

- In the Left side menu, in inventory go to EDI File option.

- In the EDI list, either import any one new EDI file or open existing EDI in edit mode by action -\> open order option.

- In the reconcile page, in the imported items list, select items which you want to delete.

- Click on the 'Remove' button.

- You will see, selected items will delete and successful delete message will display.
