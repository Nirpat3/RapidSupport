# How to set memo pop up for reconcile manual entry

**How to set memo pop up for reconcile manual entry ?**

When user want to add memo pop up for reconcile manual entry then it can be set from BOF.

Follow below procedure to set memo for reconcile manual entry.

- Log in with web URL.

- Go to setting tab.

- Select memo option.

- When memo opens then many of ON/OFF switch display for different options.

- Review there is one option for reconcile manual entry.

- From BOF keep reconcile manual entry option OFF.

![](media/image2.png){width="6.5in" height="3.688967629046369in"}

- Now go to rapid rms application and open manual entry module.

- Go to recall.

- Reconcile any order.

![](media/image1.png){width="6.5in" height="4.777777777777778in"}

- There is no memo pop up open and order reconcile without any memo pop up.

- Now from BOF keep reconcile hold invoice option ON.

![](media/image3.png){width="6.5in" height="3.8906255468066493in"}

- Now restart application as any change from BOF will reflect in Pos after restart application.

- Open manual entry module.

- Open any recall entry.

- Now reconcile that invoice.

- When hit on reconcile button then there would be one pop up open for memo.

![](media/image4.png){width="6.5in" height="4.861111111111111in"}

- Either select text from suggestion or entry any valid text.

- Hit save button.

- Entry will reconcile with entered text in memo option.
