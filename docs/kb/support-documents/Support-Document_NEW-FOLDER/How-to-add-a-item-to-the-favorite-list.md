# How to add a item to the favorite list

**[How to add an item to the favorite list?]{.underline}**

**Please follow the below steps:-**

1.  Open rapid RMS application.

2.  Go to the cash register module from Dashboard.

![](media/image1.png){width="6.5in" height="4.597222222222222in"}

3.  To add an item to the favorite list, click on the **+** icon available next to the favorite window.

![](media/image2.png){width="6.5in" height="4.375in"}

4.  Select the **Add Favorite Items** option from the drop-down.

5.  It will open **Add Favorite Items** pop-up window**.**

6.  It will appear all existing items in the list.

7.  Select the item that you wish to add to the favorite list.

8.  To select the item, click on the **Heart** option available next to the item name.

9.  Click on the **Add** button.

![](media/image3.png){width="6.5in" height="4.736111111111111in"}

10. The selected item will appear in the favorite item list in Cash Register.

11. It will prompt the message for the item added to the favorite list successfully.

![](media/image4.png){width="6.5in" height="4.833333333333333in"}
