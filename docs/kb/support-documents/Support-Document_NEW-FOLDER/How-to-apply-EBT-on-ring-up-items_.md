# How to apply EBT on ring up items_

**[How to apply EBT on ring up items?]{.mark}**

- **[EBT option should ON from BOF before applying EBT on ring up items.]{.mark}**

- [Steps to **ON EBT option**]{.mark}

<!-- -->

- [Sign in with your log in id -\> Go to settings -\> Store -\>Payment Type -\>Payment setting -\> EBT -\> Set EBT ON.]{.mark}

![](media/image1.png){width="6.5in" height="3.486111111111111in"}

- **[Steps to apply EBT on ring up items.]{.mark}**

<!-- -->

- [Open Rapidrms application.]{.mark}

- [Go to Cash Register module.]{.mark}

- [Ring up items.]{.mark}

- [In the left side of the pos screen, tap on main menu icon.]{.mark}

- [Now, from list tap on EBT option.]{.mark}

![](media/image2.png){width="6.5in" height="4.401042213473316in"}

- [Now you will see EBT popup for Apply EBT and Remove EBT.]{.mark}

- [Now select Either 'All items' or 'EBT Items' option.]{.mark}

- [After selecting any one option and according to that in bill screen item appear selected.]{.mark}

- [Now click on the 'Apply EBT' button.]{.mark}

![](media/image3.png){width="6.5in" height="3.838542213473316in"}

- [Now In the bottom of the bill screen, you will see EBT Amount.]{.mark}

![](media/image4.png){width="6.5in" height="3.888888888888889in"}
