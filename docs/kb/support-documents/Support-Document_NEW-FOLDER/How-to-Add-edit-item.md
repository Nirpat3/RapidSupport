# How to Add edit item

**How to Add/edit an item?**

The new item can be added from inventory management. Also, the added item can be edited from the same place.

**ADD ITEM**

- Open the RapidRMS application on your iPad.

- Select Inventory Management from the dashboard.

- Enter your valid passcode to access Inventory Management Module.

- For adding a new item apply to add a new option.

![](media/image1.png){width="6.5in" height="4.680555555555555in"}

- When adding a new item then below the screen display.

- Item barcode is a compulsory field to add.

- Without a barcode new item can not be created.

![](media/image2.png){width="6.5in" height="4.75in"}

- Enter the item name.

- Set item barcode.

- Enter item #.

- Enter another detail.

![](media/image3.png){width="6.5in" height="4.694444444444445in"}

- Apply to save.

- New items create and listed in the item list.

**EDIT ITEM**

- Open any item in edit mode.

- Apply changes.

- Click on save.

- Applied changes will remain to save.

![](media/image3.png){width="6.5in" height="4.694444444444445in"}
