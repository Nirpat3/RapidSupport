# How to creating an User for Rapid who can only extract data from Sales

**[How to creating an User for Rapid who can only extract data from Sales?]{.underline}**

- **Please follow the below steps:-**

1.  Login to BOS with your user credentials.

2.  Tap menu ![hamburger-menu-icon-png-white-24.jpg](media/image1.png){width="0.2075404636920385in" height="0.2075404636920385in"} (located on the top left side)

3.  Click on "**SETTINGS**".

4.  In Settings, click on "**STAFF**".

5.  In Staff, click on "**USER LIST**".

![](media/image2.png){width="6.5in" height="2.959767060367454in"}

6.  In User List, click on "**ADD**" Section then click on "**Add New**" button to create new User.

![](media/image3.png){width="6.5in" height="2.4106900699912512in"}

7.  Please fillup the "**PROFILE**" details.

![](media/image4.png){width="6.5in" height="3.2626585739282588in"}

![](media/image5.png){width="6.5in" height="3.111571522309711in"}

![](media/image6.png){width="6.5in" height="2.626350612423447in"}

8.  After fillup the Profile, Go into "**USER RIGHTS**".

Here, you can select only "**SALES**" option in "**Analytics Module**".

![](media/image7.png){width="6.5in" height="3.3312707786526685in"}

9.  After that, Tap on the "**SAVE"** button in last.

![](media/image8.png){width="6.5in" height="2.5435739282589678in"}
