# How to delete an existing sub department

**[How to delete an existing sub-department from Sub-department Master?]{.underline}**

- **It is highly recommended to check below steps before deleting sub-department.**

1.  Please make sure that you have not assigned the sub-department to any item.

2.  Please make sure that you have moved items to another Sub-Department.

- Open RAPID RMS application.

- Select inventory management module from Dashboard.

![](media/image1.png){width="6.5in" height="4.597222222222222in"}

- Click on three line option available from the top left side corner of the screen.

- Select sub-department master from the list.

- It will appear sub-department list.

![](media/image2.png){width="6.5in" height="4.239680664916885in"}

- Select any sub-department to open in edit mode.

![C:\\Users\\siya\\Downloads\\MicrosoftTeams-image (69).png](media/image3.png){width="6.5in" height="4.184199475065617in"}

- Click on the Delete button available at the bottom of the screen.

- The confirmation message will appear.

![](media/image4.png){width="6.5in" height="4.722222222222222in"}

- Select the Yes option.

- If you have assigned the sub-department to any items, it will not allow you to delete the sub-department and will appear the error message as shown in the below image.

![](media/image5.png){width="6.5in" height="4.694444444444445in"}
