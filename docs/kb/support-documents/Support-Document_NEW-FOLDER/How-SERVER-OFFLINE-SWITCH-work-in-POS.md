# How SERVER OFFLINE SWITCH work in POS

**[How "SERVER OFFLINE SWITCH" work in POS?]{.underline}**

# **What is Server Offline Switch ?**

The Server Offline Switch in a POS (Point of Sale) system is a functionality that allows the system to continue operating even when the connection to the central server is lost or disrupted.

It serves as a failover mechanism to ensure uninterrupted service and transaction processing in case of server downtime or network issues.

- **How "Server Offline Switch" work in POS ?**

With the help of "Server Offline Switch", RapidRMS supports Cash Register (RCR) module working in offline mode. Rest of the modules work only in online mode.

- **When App goes in offline mode, only following icons are available in Dashboard:**

1.  Cash Register

- Retail

- Petro

2.  Clock-In/Out

3.  Sign-In/Out

4.  Settings

- **Please follow the below steps:-**

1.  Login to **RapidRMS Application** with your credentials.

- Enter your Email OR User ID.

- Enter your Password.

- Click on SIGN IN button.

![](media/image1.png){width="6.5in" height="3.008707349081365in"}

2.  In Rightside upper corner, you will find "**SERVER OFFLINE SWITCH**". Click on it.

![](media/image2.png){width="6.5in" height="3.9217596237970254in"}

3.  Once you click on "**Server Offline Switch**", it will give you popup message for confirmation.

Please click on "**OFFLINE**" button.

![](media/image3.png){width="6.5in" height="3.627178477690289in"}

4.  Once you click on "**OFFLINE**" button then RapidRMS move to **OFFLINE mode** with successfully popup message.
