How to Create Third-Party Promotions for Altria Scan Data
What is Scan Data?

Scan data refers to transaction records captured by retailers when consumers make purchases. It is
primarily used to:
    Create discounts or promotions.
    Track sales records for each day.
    Transmit data weekly to a vendor, agent, contractor, affiliate, or service provider via SFTP for
business analysis and strategy.

Note:

Before creating third-party Scan Data promotions, ensure that the Scan Data module is enabled in
BackOffice (BOS).
For assistance, contact the support team.



How to Create Third-Party Promotions

    1. Log in to RapidRMS BackOffice using your credentials.

    2. Navigate to the Scan Data module.

    3. Click on Third-Party Promotion under the Scan Data module.

How to Submit the Altria Scan Data File from BackOffice
There are two ways to submit the Altria Scan Data file:

    1. Automatic Submission

    2. Manual Submission




Automatic Submission (Auto Upload)
    1. Log in to BackOffice using your credentials.

    2. Navigate to Settings.

    3. Under Settings, go to the Modules section.

    4. Enable the "Scan Data Auto Upload" module.




www.rapidrms.com                                          Page 14
Manual Submission
   1. Log in to BackOffice using your credentials.

   2. Navigate to the Scan Data module.

   3. Click on "Altria File" under Scan Data.




   4. Verify the Scan Data file week period (For Altria, it should be Sunday to Saturday).

   5. Click on "Upload 2022 File".

   6. A confirmation message will appear. Click "Upload" to proceed.




   7. To view previously submitted files, go to the "File" section.
