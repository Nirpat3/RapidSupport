# How synchronization work_

**How does synchronization work?**

Sync is for taking sync of application data.

In rapid RMS application in all module at the top bar there is one sync option.

By applying that option if due to the internet some data or live update does not sync then all data sync by this option.

![](media/image3.png){width="6.5in" height="4.75in"}

- Open setting option.

- There is one button for synchronization.

- ![](media/image2.png){width="6.5in" height="4.666666666666667in"}

- There are three options for database synchronization.

- When applying for any option it will ask for confirmation. When applying yes then a database for that particular time duration will sync.
