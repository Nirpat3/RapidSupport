# How to clock out

**How to clock out?**

When already some user is working in shift and now want to close shift or want to clock out there are two possible ways.

- Open rapid RMS application.

- Go to clock in-out module.

- When there is already one record continues then clock out option is enabled.

- Apply clock out the option to close entry.

![](media/image1.png){width="6.5in" height="4.708333333333333in"}

- One another way to clock out is by edit.

- Swipe entry to the right.

- Apply edit.

![](media/image2.png){width="6.5in" height="4.638888888888889in"}

- In the popup set out time.

- Apply done.

- Clock out will be done in this way also.

![](media/image4.png){width="3.40625in" height="3.875in"}

![](media/image3.png){width="6.5in" height="4.791666666666667in"}

- In out both times can be set by edit option.

By "add" button also clock in and clock out can be set.
