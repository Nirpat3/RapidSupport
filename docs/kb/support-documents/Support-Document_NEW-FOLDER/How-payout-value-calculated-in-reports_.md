# How payout value calculated in reports_

**How is payout value calculated in reports?**

In rapid there is option available for service in POS.

Services are considered as payout.

There is separate department of payout is available in inventory.

In POS if there is any item ring up with payout then that is consider as payout sales value.

Payout department/ items are always with negative amount.

In report that payout value is consider in payout field.
