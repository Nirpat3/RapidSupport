# How to set memo for qty OH changed

**How to set memo for qty OH changed ?**

When user want to add memo pop up for qtyOH change then it can be set from BOF.

Follow below procedure to set memo for qtyOH change.

- Log in with web URL.

- Go to setting tab.

- Select memo option.

- When memo opens then many of ON/OFF switch display for different options.

- Review there is one option for no sale.

- Keep memo option for qty oh change OFF.

![](media/image4.png){width="6.5in" height="3.3472222222222223in"}

- Open rapid rms application.

- Go to inventory management module.

- Open any item in edit mode.

- There is one field qtyOH.

- Click on that for change qtyOH for that item.

- There is no pop up for memo, without any memo pop up qty OH would be change.

![](media/image5.png){width="6.5in" height="4.722222222222222in"}

- Now from BOF keep qtyOH changed memo option ON.

![](media/image2.png){width="6.5in" height="3.606438101487314in"}

- Restart application.

- Now open inventory management application.

- Open any item in edit mode.

- Now in qty OH field try to change quantity.

- When click on qty OH field then memo pop up for qty OH change reason would be open.

- Either select text from given default text or enter your valid reason for qtyOH change.

![](media/image3.png){width="6.5in" height="4.75in"}

- Hit save.

- Now change qty.

- Memo text would be save at the time of qty change.

![](media/image1.png){width="6.5in" height="4.694444444444445in"}

- Thus, by this way qty OH change memo can be set.
