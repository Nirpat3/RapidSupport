# How to manage barcode in inventory item

**[How to manage barcodes in an inventory item?]{.underline}**

In inventory, item facilitates adding a barcode to identify different items in store.

- Open RAPID RMS application in IOS.

- Select inventory management from the Dashboard.

![](media/image1.png){width="6.5in" height="4.472222222222222in"}

- Create a new item or Open any item in edit mode.

- Click on the barcode field.

![](media/image2.png){width="6.5in" height="4.777777777777778in"}

- In the item, the barcode can be added differently.

- In a single case and pack, you can add various barcodes. And also in one field multiple barcode addition is possible.

- There are two ways mentioned below to add barcodes

i)  Generate barcode

ii) Manually enter the barcode

**Generate barcode**

- Click on the barcode/UPC field. One window will open.

- Select any from single/case/pack.

- Click on Generate. One barcode will generate automatically by the system.

- Click on add button.

- Click on the done button.

- Barcode will add to the item.

![](media/image3.png){width="6.5in" height="4.638888888888889in"}

**Manually enter the barcode**

- Click on the barcode field.

- One window will open.

- Select any from single/case/pack.

- Enter data into the barcode field. It could be numerical value and also alphabetical value possible.

- Click on add button.

- Click on the save button.

![](media/image4.png){width="6.5in" height="4.032608267716536in"}

- Duplication of barcode

- It is possible to add a duplicate barcode to the item.

- There is one button for the duplicate barcode.

1)  **When the barcode button is set OFF.**

![](media/image5.png){width="6.5in" height="3.1956517935258093in"}

When the barcode button is set off then you cannot add a duplicate barcode to the item. Every time you have to add a different barcode that is not present anywhere in the system.

2)  **When the barcode is set to ON**

![](media/image6.png){width="6.5in" height="4.75in"}

- When the barcode button is set to ON, you can add a duplicate barcode to the item. It is possible to add multiple-time duplicate barcodes.

**Delete barcode**

![](media/image7.png){width="6.5in" height="4.666666666666667in"}

- To delete a barcode click on the barcode field. The window for the barcode will open.

- Click on the delete button of the barcode, which barcode you want to remove from the item.

- Click on the save button.

- Barcode will remove from that item.

> **Adding barcode by swipe item**

- Barcode addition is also possible by swiping an item.

- Open inventory management.

<!-- -->

- Swipe item.

![](media/image8.png){width="6.5in" height="4.861111111111111in"}

> **There are three individual options single, case, and pack for barcode addition.**

- Click on one option in which you want to add a barcode.

- i.e if you want to add a barcode in a case then click on the case.

- One window will open add a barcode and click on done

- Barcode will add in case of the item.
