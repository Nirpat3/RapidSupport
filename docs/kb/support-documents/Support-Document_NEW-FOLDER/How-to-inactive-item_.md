# How to inactive item_

**How to an inactive item?**

All the items which are available in-store display in the inventory management module Items section.

Sometimes it happens that for some season some items not require temporary in-store. Rapid RMS gives a facility to inactive those items. Whenever you want then those items can be again activated.

- Open rapid RMS application.

- Go to item management and open item list.

- Open the item in edit mode which item you want to inactive.

![](media/image1.png){width="6.5in" height="4.964173228346457in"}

- Keep active button OFF.

![](media/image2.png){width="6.5in" height="4.873101487314086in"}

- Item will now display in the inactive list.

![](media/image3.png){width="6.5in" height="4.856502624671916in"}

- From swipe, you can again active that item.

![](media/image4.png){width="6.5in" height="4.75in"}

Another way to inactive that item is via swipe.

- Swipe that item which you want to inactive.

- Hit inactive option.

![](media/image5.png){width="6.5in" height="4.694444444444445in"}

- Item will now remove from item list and move to inactive list.

Note: Department/Sub dept default items can not be inactive.
