# How to view shift report user wise_

**[How can I view shift reports user-wise?]{.underline}**

- If you need to check the shift history data according to the user wise you can review the shift history report by filtering the user.

<!-- -->

- Open Rapid RMS application.

- Go to the Daily report module.

![](media/image1.png){width="6.5in" height="4.75in"}

- Open the shift report and click on the user filter.

- It will display all users and current users.

- Select from any one of them.

![](media/image2.png){width="6.5in" height="4.819444444444445in"}

- Data will filter according to the selected user.

![](media/image2.png){width="6.5in" height="4.819444444444445in"}
