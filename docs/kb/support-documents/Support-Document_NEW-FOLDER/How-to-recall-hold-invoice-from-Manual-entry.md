# How to recall hold invoice from Manual entry

**[How to recall hold invoices from Manual entry?]{.underline}**

- Open the Manual entry module from DashBoard.

- Enter the Quick access password.

- Click on the Recall option.

- It will appear the list of hold invoices.

![](media/image1.png){width="6.5in" height="4.652777777777778in"}

- You can see hold invoices with vendor name (if present), invoice number, date received and date started.

  ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------
    **Fields**    **Description**
  --------------- -----------------------------------------------------------------------------------------------------------------------------------------------------------
   Date Started   It describes starting date when you create a new invoice from manual entry. It appears automatically, you need not enter it while creating a new invoice.

    Vendor Name   It appears vendor name if you have selected it from the vendor list while creating a new invoice.

    Invoice \#    It appears entered the invoice number that you entered while creating a new invoice.

   Date Received  It appears the date on which you expect to receive or deliver the purchased product.
  ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------

![](media/image2.png){width="6.5in" height="4.736111111111111in"}

- Select any invoice entry from Recall Manual Entry.

- It will be prompted confirmation message. Select the **Yes** option.

![](media/image3.png){width="6.5in" height="4.708333333333333in"}

- It will open invoice manual order entry.

- To add an item click on the search button at the bottom bar.
