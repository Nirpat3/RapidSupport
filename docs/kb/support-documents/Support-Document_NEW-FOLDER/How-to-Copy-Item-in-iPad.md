# How to Copy Item in iPad

**How to Copy-Item on iPad:**

If you want to create a new Item that has almost the same information as the previously created Item then RapidRMS give you the functionality to copy the existing Item detail for creating a new Item. You don't need to enter every detail. Just change the detail which defers and provide a unique barcode and save it.

Here are the steps on how to copy existing item detail.

- Open the RapidRMS application on your iPad.

- Select Inventory Management from the dashboard.

- Enter your valid passcode to access Inventory Management Module.

- Slide any Item from the list.

- Option bar has inactive, history, single, case, pack, copy and delete options.

![](media/image1.png){width="6.5in" height="4.680555555555555in"}

- Click on the Copy button.

- All the details of the existing Item will copy except the barcode.

- Change the details according to you and save the detail.
