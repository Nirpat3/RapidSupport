# How to print invoice_

**[How to print an invoice?]{.underline}**

When a user wants to print invoice then invoice can be printed. Follow the below steps.

- Open rapid RMS application.

- Go to the cash register module.

- Open invoice.

- At the bottom bar, there is one option named as **PRINT**.

- Hit that button.

![](media/image1.png){width="6.5in" height="4.736111111111111in"}

- One pop up for print selection will display.

- Select the option according to print required.

- Hit print option.

- Invoice print.

![](media/image2.png){width="6.5in" height="4.680555555555555in"}

**Note**: One condition is there if the printer is connected then the only invoice will print.
