# How payment type calculation define in report_

**How payment type calculations define in report?**

In report all kind of payment display with selling amount and count.

In payment section all type of payment display.

![](media/image1.png){width="6.5in" height="4.847222222222222in"}

![](media/image2.png){width="6.5in" height="4.847222222222222in"}

There are three columns available.

Tender Type: Payment name are display.

Amount: Sales value of that transactions which is completed by that payment type.

Cash: Item discounted price with Tax - Refund or payout + fees (Only those in which payment made through Cash payment)

Check: Item Discounted price with Tax - Refund or payout + fees (Only those in which payment made through check payment) + Check Cash amount

Credit: Item discounted price with Tax - Refund or payout + fees (Only those in which payment made through Credit card payment)

Debit: Item discounted price with Tax - Refund or payout + fees (Only those in which payment made through Debit card payment)

Gift card: Item discounted price with Tax - Refund or payout + fees (Only those in which payment made through Gift card payment)

EBT: Total of discounted price of Item in which EBT applied (EBT not apply on Check cash, Payout, charge type ) - Total refund item in which EBT applied
