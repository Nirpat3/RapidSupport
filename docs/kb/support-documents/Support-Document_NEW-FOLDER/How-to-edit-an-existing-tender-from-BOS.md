# How to edit an existing tender from BOS

**[How to edit an existing tender from BOS?]{.underline}**

- Log in to the Back office with your credentials.

- Now go to settings. You may need to scroll down to select the settings option.

- Select the Store option from the list.

- Click on the payment type option from the list.

<!-- -->

- To update an existing tender, Click on the action button.

- Select the Edit button.

![](media/image1.png){width="6.5in" height="3.024254155730534in"}

- Change the details that you want to update.

- Click on the Save button.

- If you do not want to save any changes click on the close button.
