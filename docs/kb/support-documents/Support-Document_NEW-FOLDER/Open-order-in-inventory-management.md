# Open order in inventory management

**Open order in inventory management**

When entering items from in which are for the received record and put an order on hold then that order display in open order.

- Go to the inventory management module.

- Hit open order.

![](media/image5.png){width="6.5in" height="4.694444444444445in"}

- All the hold invoices list display here, which are hold from IN/Out.

- You can view or email order by action button.

![](media/image2.png){width="6.5in" height="4.694444444444445in"}

- In open order, it displays in type column that open order is from IN or OUT.

- Open any of order.

- In open order you can add an item and also added item can delete via swipe.

- The invoice can be preview and mail by export option.

- Now after change save that invoice.

![](media/image4.png){width="6.5in" height="4.736111111111111in"}

![](media/image1.png){width="6.5in" height="4.805555555555555in"}

- When saving invoice then pop up appear.

- Enter text and apply to save.

![](media/image3.png){width="6.5in" height="4.791666666666667in"}

- When saving then that invoice goes into close order.
