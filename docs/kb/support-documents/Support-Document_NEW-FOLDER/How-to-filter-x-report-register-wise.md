# How to filter x report register wise

**How to filter x report register wise?**

In x report, all the shift data display until z is not applied. In x report, particular one(self) register data display.

But if the user wants to check the detail of another register then it can be possible.

- Open rapid RMS application.

- Go to Daily report module.

![](media/image3.png){width="6.5in" height="4.75in"}

- Open x report.

- It will display all data of that particular register.

- If you want to check another register data then select filter.

![](media/image2.png){width="6.5in" height="4.722222222222222in"}

- Select register from the list.

![](media/image1.png){width="6.5in" height="4.638888888888889in"}

- Selected register data will display in x report.

![](media/image4.png){width="6.5in" height="4.638888888888889in"}

- From clear all option all register can be removed from filter.
