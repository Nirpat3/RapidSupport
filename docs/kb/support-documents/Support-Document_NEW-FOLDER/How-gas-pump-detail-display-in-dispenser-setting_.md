# How gas pump detail display in dispenser setting_

**How gas pump detail display in dispenser setting ?**

In gas pump section there are detail about tank fuel grade and all pumps.

![](media/image1.png){width="6.5in" height="4.888888888888889in"}

- In tank it will display all available tank with litter and capacity.

![](media/image3.png){width="6.5in" height="4.708333333333333in"}

- In fuel grade all fuel grade type display with description.

- When select any fuel type then list of all variance will display and from list anyone you can select.

![](media/image5.png){width="6.5in" height="4.722222222222222in"}

![](media/image7.png){width="6.5in" height="4.75in"}

![](media/image6.png){width="6.5in" height="4.541666666666667in"}

- In pump all pump display display with enable/disable option.

- If pump option is enable then that pump is display in RCR and if option OFF then that particular pump will not display in RCR.
