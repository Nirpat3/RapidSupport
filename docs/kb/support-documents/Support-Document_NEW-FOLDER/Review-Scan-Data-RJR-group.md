# Review Scan Data RJR group

**[How to review Scan Data for the RJR group?]{.underline}**

1.  Login to BOS with your user credentials.

2.  In the top right, click on the three-line hamburger ![hamburger-menu-icon-png-white-24.jpg](media/image1.png){width="0.2075404636920385in" height="0.2075404636920385in"} menu.

3.  Choose the Outlet Detail Beta option from the settings.

![](media/image2.png){width="6.5in" height="3.011420603674541in"}

4.  At the top right side, select RJR vendor from the drop down list.

![](media/image3.png){width="6.5in" height="3.0062259405074365in"}

5.  At the top left side, select the weekly pre define option from the Date filter.

![](media/image4.png){width="6.5in" height="3.0367541557305335in"}\
**Note:-** If you have taken any scan data transaction that will only appear every Sunday.

6.  After you select a filter, It will review all transactions of scan data sold items.

![](media/image5.png){width="6.5in" height="3.008455818022747in"}

7.  It appears circular ORG flow chart for all created scan data types percentage wise.

![](media/image6.png){width="6.5in" height="3.011420603674541in"}

8.  It also appears a summary of different promotions sales and sold records.

![](media/image7.png){width="6.5in" height="3.011420603674541in"}
