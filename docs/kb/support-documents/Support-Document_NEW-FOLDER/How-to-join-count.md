# How to join count

**How to join the count?**

In inventory count when counting of the item is already started and another user wants to join that counting from another place in-store at that time join count module comes in the picture.

- Open rapid RMS application.

- Go to the dashboard and select inventory count module.

![](media/image1.png){width="6.5in" height="4.569444444444445in"}

- On the inventory count module dashboard click on join count.

![](media/image2.png){width="6.5in" height="4.208333333333333in"}

- Already started counting list displays in the join page.

- Open any invoice in which you want to start counting.

![](media/image3.png){width="6.5in" height="4.777777777777778in"}

- Enter the place name.

![](media/image4.png){width="6.5in" height="4.777777777777778in"}

- Add the items which you have found from the store at the time of counting.

- Enter qty and put an invoice on hold or reconcile.

![](media/image5.png){width="6.5in" height="4.736111111111111in"}

![](media/image6.png){width="6.5in" height="4.763888888888889in"}

By this way from differ places counting can be started.
