# How to logout from inventory management_

**How to logout from inventory management?**

For logout from an inventory management module go follow bellow procedure.

- At the bottom bar, there is on the button available in the left corner.

- Hit logout.

- It will redirect to the dashboard and you are now logged out from the inventory module.
