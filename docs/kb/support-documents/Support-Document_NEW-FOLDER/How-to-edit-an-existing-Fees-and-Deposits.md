# How to edit an existing Fees and Deposits

> **[How to update existing Fees and Deposits?]{.underline}**

- [Open Rapidrms application.]{.mark}

- [Go to the Inventory management module.]{.mark}

- [Enter your quick access password.]{.mark}

- [At the top left, click on the three lines (Hamburger) button]{.mark} ![C:\\Users\\mit shah\\AppData\\Local\\Microsoft\\Windows\\INetCache\\IE\\ZY2HW6WI\\Menu-thin-circular-outlined-button.svg\[1\].png](media/image1.png){width="0.2247561242344707in" height="0.2247561242344707in"}. []{.mark}

- Select the Fee and Deposit option from the Item Master menu.

![](media/image2.png){width="6.5in" height="4.888888888888889in"}

- [Select the Fee/Deposit option from the list.]{.mark}

- [It will appear the existing Fee/Deposit list.]{.mark}

![C:\\Users\\siya\\Downloads\\MicrosoftTeams-image (75).png](media/image3.png){width="6.5in" height="4.871271872265967in"}

- [Select Any Fee or Deposit.]{.mark}

- Change the details that you want to update.

- Click on the Save button.

- If you do not want to save any changes click on the close button.
