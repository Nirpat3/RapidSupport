# How to turn on Scan data module from Back office

**[How to turn on the Scan data module from the Back office?]{.underline}**

If the third party promotion option does not appear, please follow the below steps:-

1.  Login to BOS with your user credentials.

2.  Tap menu ![hamburger-menu-icon-png-white-24.jpg](media/image1.png){width="0.2075404636920385in" height="0.2075404636920385in"} (located in the top right side)

3.  Tap settings.

4.  Tap modules

5.  Scroll to see the Scan data option.

![](media/image2.png){width="6.5in" height="3.0150918635170605in"}

6.  Select the 'ON' Radio button.

![](media/image3.png){width="6.5in" height="3.026492782152231in"}
