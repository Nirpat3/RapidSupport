# How to create new invoice in inventory count_

**How to create a new invoice in inventory count?**

Inventory count is a module for item collection record for available items holding on different places.

For creating a new module follow the below procedure.

- Open rapid RMS application.

- Go to the dashboard and select inventory count module.

![](media/image1.png){width="6.5in" height="4.569444444444445in"}

- On inventory count module dashboard click on new.

![](media/image2.png){width="6.5in" height="4.555555555555555in"}

- When creating a new invoice that means for a particular store start counting items from any place of the store.

- In remarks enter any text.

- Click on save.

![](media/image3.png){width="6.5in" height="4.791666666666667in"}

- One pop up open for entering that place name from where you need to start counting.

- Enter the place name.

- Click on done.

![](media/image4.png){width="6.5in" height="4.694444444444445in"}

- One page for item adding will open.

- Item(s) you found, You can enter from the search option.

- After selecting an item from the list click on add.

- Items will be added to our counting list.

- Items will display with Qty, barcode, description, quantity available in-store and with price/cost.

![](media/image5.png){width="6.5in" height="4.763888888888889in"}

- In the qty field enter qty which you found from that location.

- The quantity you can enter is a quantity in the form of single, case and pack.

![](media/image6.png){width="6.5in" height="4.416666666666667in"}

- After adding an item you can hold or reconcile the invoice and proceed further.

![](media/image7.png){width="6.5in" height="4.722222222222222in"}

![](media/image8.png){width="6.5in" height="4.777777777777778in"}

- By this way, the new invoice can be created in inventory count.
