# How to add qsr group in modifier item

**How to add qsr group in modifier item ?**

There are two ways to add qsr group into modifier item.

First way is, from qsr group master modifier item can be link and that way qsr group can be added in modifier item.

Another way is from modifier item group can be added.

Follow below procedure to add qsr group into modifier item.

- Open rapid rms application.

- Go to inventory management module.

- Open modifier items tab.

- Open any of modifier item in edit mode.

- Now scroll that item there is one field available named as qsr group.

- When create modifier item then initially it display blank.

![](media/image1.png){width="6.5in" height="4.888888888888889in"}

- To add qsr group click on arrow available in that field.

- All available qsr group in system will be listed.

- Select any of one radio button.

![](media/image2.png){width="6.5in" height="4.75in"}

- Hit save button, qsr group will be added to item.

- There is search bar available to search qsr group from list.

- Note that at a time only one qsr group can be added in one modifier item.

- Added qsr group will be display in modifier item.

![](media/image3.png){width="6.5in" height="4.791666666666667in"}

- Added qsr group can be remove by cancel button available behind qsr group name.
