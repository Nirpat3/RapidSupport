# How do I set up stock level in the items

## **How do I set up stock level in the items?**

In rapid RMS stock can manage in inventory management module. In item, there is one filed in pricing table named as QTY OH. Stock level can manage from the options tab. There are two fields' available minimum inventory level and maximum inventory level. Maximum inventory level must be greater than the minimum inventory level.

Set up a stock level:

- Open inventory management level.

- Open any item in edit mode.

- Go to the Options tab of that item.

- There are two option minimum inventory level and maximum inventory level.

- From that, both option stock levels can set up.
