# How to discontinue items from item movement_

**How to discontinue items from item movement ?**

When item is discontinue from item movement then item will inactive from list. For discontinue item follow below procedure.

- Open rapid rms application.

- Go to item movement module.

- Add items.

![](media/image2.png){width="6.5in" height="4.736111111111111in"}

- Select items from list.

- Apply discontinue.

![](media/image4.png){width="6.5in" height="4.694444444444445in"}

![](media/image1.png){width="6.5in" height="4.791666666666667in"}

- Items you discontinue now will move to item inactive list.

![](media/image3.png){width="6.5in" height="4.722222222222222in"}
