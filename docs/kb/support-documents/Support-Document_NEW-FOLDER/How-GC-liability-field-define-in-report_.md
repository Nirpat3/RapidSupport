# How GC liability field define in report_

**How GC liability field define in report?**

The GC liability field is related to gift card facility which is available in the shift report.

GC Liabilities is total Amount or balance of Gift card sold to customer in all the register.

When there is no data in register means if it's starting of any shift for particular register then also in GC liability there is some value available if universal value is there in store that means in all reports GC liability value remain same according to calculation.
