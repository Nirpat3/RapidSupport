# How to define price change count_

**How to define price change count ?**

Price change is a count shows how many number of time item price is changed during the report scope.

![](media/image2.png){width="6.5in" height="4.791666666666667in"}

In RCR when item is ring up and price changed by swipe item then this count is consider in price change count in report.

In short it is \# of price changes occurred on sales screen.
