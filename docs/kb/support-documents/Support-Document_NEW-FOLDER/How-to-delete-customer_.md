# How to delete customer_

**How to delete a customer?**

If any customer is no more in use in application and if you want to delete that customer then follow the below procedure.

- Open rapid RMS application.

- Go to the customer loyalty module.

![](media/image2.png){width="6.5in" height="4.763888888888889in"}

- All existing customer display in the list.

- Swipe left.

- There is one delete button.

- Apply that button.

![](media/image1.png){width="6.5in" height="4.777777777777778in"}

- Give yes to the confirmation message.

![](media/image3.png){width="6.5in" height="4.763888888888889in"}

- Customer will remove from list.
