# How to reconcile invoice in inventory count_

**How to reconcile invoice in inventory count?**

In inventory count when from one or other places counting complete then from that individual place want to stop counting then invoice can be reconciled.

For reconcile invoice follow below steps :

- Open rapid RMS application.

- Go to the inventory count module.

- Open recall module.

![](media/image1.png){width="6.5in" height="4.5in"}

- All the invoices which are on hold display in the recall module.

- Open any of invoice which individual counting is complete.

- In the recall list, in one invoice all the places display from where counting is started.

![](media/image2.png){width="6.5in" height="4.722222222222222in"}

- Now apply to reconcile. If you want to change in item qty or want to add item in counting then also you can add and after then you can reconcile.

![](media/image3.png){width="6.5in" height="4.694444444444445in"}

- Give yes.

![](media/image4.png){width="6.5in" height="4.722222222222222in"}

- After complete reconcile, an invoice will move to reconcile list.

- Go to reconcile module.

![](media/image5.png){width="6.5in" height="4.805555555555555in"}

- In reconcile, list invoice is added with status as incomplete.

- You can complete invoice but if there is still any invoice entry

- From reconcile, you can complete invoice but there is still from somewhere counting is not complete till then you cannot complete invoice.

- After complete all places counting, the invoice can complete.
