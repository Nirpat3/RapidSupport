# How to open drawer without key

**How to open the drawer without a key?**

- Select the cash register icon. Enter the 4-digit password.

- Click the menu button in the top left-hand corner.

- Select No sale.

- A message will prompt you if you would like to open the cash drawer. Select Yes. Your cash drawer will open.
