# How to setup receipt print option

**[How to setup receipt print option (last invoice, invoice print and change due screen) from merchant, sale and other receipt pop up.]{.mark}**

- **[This option is used when the user wants to print only selected receipt.]{.mark}**

- [Follow the below steps to set up receipt print]{.mark}

<!-- -->

- [Open RapidRMS application.]{.mark}

- [Go to Cash register module.]{.mark}

- [At there you have 3 places to setup receipt print (last invoice, invoice print and change due screen).]{.mark}

- [Tap on 'print' icon in the bottom of the POS screen.]{.mark}

![](media/image1.png){width="6.5in" height="4.875in"}

- [At there, you will see Receipt types popup for print.]{.mark}

- [At that popup, you can setup receipt setting.]{.mark}

- [Tap on any options which you want to select and after selection, it will automatically save the setting.]{.mark}

> **[Same functionality in change due screen and invoice screen.]{.mark}**
