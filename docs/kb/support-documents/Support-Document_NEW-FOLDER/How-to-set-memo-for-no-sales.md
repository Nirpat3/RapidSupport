# How to set memo for no sales

**How to set memo for no sales?**

When user want to add memo pop up for no sales then it can be set from BOF.

Follow below procedure to set memo for no sales.

- Log in with web URL.

- Go to setting tab.

- Select memo option.

- When memo opens then many of ON/OFF switch display for different options.

- Review there is one option for no sale.

![](media/image1.png){width="6.5in" height="3.3472222222222223in"}

- Keep no sales memo option off.

- Now open rapid rms application and go to cash register.

- Click on no sales.

- There is no memo pop up open for no sales.

![](media/image2.png){width="6.5in" height="4.736111111111111in"}

- Now from web log in keep memo option for no sales ON.

![](media/image3.png){width="6.5in" height="3.2222222222222223in"}

- Restart application, As any change from BOF will reflect after restart application.

![](media/image4.png){width="6.5in" height="4.583333333333333in"}

- Pop up for memo would open.

![](media/image5.png){width="6.5in" height="4.430555555555555in"}

- Select any one suggestion.(Suggestion would default given by system).

- Click on save.

- Memo would save for that action.
