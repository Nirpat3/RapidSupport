# How to apply ZZ_

**How to apply ZZ?**

ZZ report is for monthly selling data.

Usually, z is applied at the end of the month.

When z is applied then all z which are applied before are considered in ZZ.

When apply ZZ then shift is close.

- Open rapid RMS application.

- Go to daily report module.

- At bottom bar there is one option named as "ZZ report".

![](media/image3.png){width="6.5in" height="4.763888888888889in"}

- Click on ZZ report for apply ZZ.

![](media/image1.png){width="6.5in" height="4.75in"}

- Apply yes. ZZ will apply properly.

- At the time of ZZ apply shift will close and ask for the close amount.

![](media/image2.png){width="6.5in" height="4.805555555555555in"}

- When applying ZZ then it will ask for print that report.

- If you want to print then you can print with it from here.

![](media/image4.png){width="6.5in" height="4.708333333333333in"}

- When applying ZZ then x is reset. And the new shift open.
