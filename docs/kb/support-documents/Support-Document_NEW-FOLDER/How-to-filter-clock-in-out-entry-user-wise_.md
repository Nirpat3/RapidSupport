# How to filter clock in out entry user wise_

**How to filter the clock in our entry user wise?**

In clock In Out when wanted to filter by the user then it can be possible in a Shift In- Out module.

- Open rapid RMS application.

- Go to the clock In-Out module.

- At the top bar, there is one option in which the user name is displayed.

- By user, you can search the clock in out entry.

- Select any user from the list.
