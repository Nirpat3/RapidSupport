# How to set up a gateway from Back

**[How to set up a gateway from Back-office?]{.underline}**

- A **gateway** is a network that provides a connection between networks and contains hardware devices such as Pax, Ingenico, Bridgepay, etc.

<!-- -->

- **Please follow the below steps:-**

1.  Login to BOS with your user credentials.

2.  Tap menu ![hamburger-menu-icon-png-white-24.jpg](media/image1.png){width="0.2075404636920385in" height="0.2075404636920385in"} (located on the top right side)

3.  Tap settings.

4.  Tap store.

5.  Scroll to see the Gateway option. Select the Gateway option.

![](media/image2.png){width="6.5in" height="2.9866404199475065in"}

6.  Select the Add option.

![](media/image3.png){width="6.5in" height="2.9767989938757657in"}

7.  Click on the Payment drop menu to select a payment gateway. When you select payment from the payment option, It will appear other fields details that you need to enter for the payment gateway.

![](media/image4.png){width="6.5in" height="4.535982064741908in"}

- The **different fields** will appear **automatically based on the selected gateway** from the payment option.

For Pax, **please click here.**

For Bridgepay, **please click here.**

For Bridgepay guardian, **please click here.**

For EPS, **please click here.**

For Ingenico, **please click here.**

8.  Enter all details.

9.  Tap on the **Save** button.
