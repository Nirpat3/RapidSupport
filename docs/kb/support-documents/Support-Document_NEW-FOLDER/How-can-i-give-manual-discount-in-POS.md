# How can i give manual discount in POS

**[How can I give a manual discount on POS?]{.underline}**

- In bill Ring up item(S).

- From bottom bar select option discount.

- Select items from the bill in which you want to apply the discount manually.

- Now from discount window select bill/item discount.

- ![](media/image1.png){width="6.490972222222222in" height="3.9895833333333335in"}Discount may be applied in % or \$ according to selection.

- Click on Enter option.

- Discount will apply on the selected item.
