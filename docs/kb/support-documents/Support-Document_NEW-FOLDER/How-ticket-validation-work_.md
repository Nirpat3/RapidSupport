# How ticket validation work_

**How does ticket validation work?**

For ticket validation application follow the below steps.

- Go to the cash register module.

- open left side pane.

- Apply ticket validation.

![](media/image1.png){width="6.5in" height="4.777777777777778in"}

- The screen for ticket validation will open.

- From that, you can scan the QR code and also use inquiry for ticket information.
