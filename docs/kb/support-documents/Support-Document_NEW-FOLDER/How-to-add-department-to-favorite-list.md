# How to add department to favorite list

**[How to add a department to the favorite list?]{.underline}**

**Please follow the below steps:-**

1.  Open rapid RMS application.

2.  Go to the cash register module from Dashboard.

![](media/image1.png){width="6.5in" height="4.597222222222222in"}

3.  To add Department to the favorite list, click on the **+** icon available next to the favorite window.

![](media/image2.png){width="6.5in" height="4.75in"}

4.  Select the **Add Favorite Departments** option from the drop-down.

5.  It will open **Add Favorite Departments** pop-up window**.**

6.  It will appear all existing Favorite/unfavorite departments in the list.

7.  The department which is already added to the favorite list will appear as selected with a **Yellow heart** symbol next to the department name in the list.

8.  Select the department that you wish to add to your favorite list.

9.  Click on the **Heart** option next to the department name to select the department.

![](media/image3.png){width="6.5in" height="4.680555555555555in"}

10. Click on Add button.

11. The selected department will appear in the favorite list in the Cash register.

12. When you deselect any department, it will disappear from the favorite list.

13. To deselect any department, click on the **Heart** option again.

14. It will prompt the message for the item added to the favorite list successfully.

![](media/image4.png){width="6.5in" height="4.763888888888889in"}
