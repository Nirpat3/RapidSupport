# How to load balance in gift card_

**How to load balance in a gift card?**

The gift card is a type of payment by which user can payment. For generating gift card make sure in payment type rapid RMS gift card payment type should be there in payment list.

- Open rapid RMS application.

- Go to the cash register module.

- Click on gift card option available at the bottom bar.

- Select the Load Balance option.

.

![](media/image1.png){width="6.5in" height="3.838542213473316in"}

- Enter the gift card number and confirm it again.

![](media/image2.png){width="6.5in" height="4.130208880139983in"}

- Add the amount in \$ of which amount you want to generate gift card.

- Click on save.

- A message will prompt 'Would you like to add \$500.00 to card #23322222222256?' click on yes option.

- Now you can see Rapid RMS Gift Card in ring up the screen and complete transaction.

- New gift card will generate.
