# How to calculate line item void_

**How to calculate line item void?**

Line item void is a calculation of how many number of times items are deleted by cashier during the sale or during the reporting time scope.

![](media/image1.png){width="6.5in" height="4.791666666666667in"}

When you ring up and then delete that item by swiping from the bill screen, the total counts of line item voice will be added in the shift report but the count will added once you complete next successful transaction.

![](media/image2.png){width="6.5in" height="4.541666666666667in"}
