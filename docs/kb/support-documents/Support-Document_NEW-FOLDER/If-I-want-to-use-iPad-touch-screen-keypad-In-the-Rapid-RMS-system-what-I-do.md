# If I want to use iPad touch screen keypad In the Rapid RMS system what I do

**If I want to use the iPad touch screen keypad In the Rapid RMS system what do I do?**

- When you use a Bluetooth scanner. The scanner works/acts as a keyboard on the iPad. So when you press the scanner trigger button twice continually it will open the apple keyboard on the iPad. In the same way, you can hide the keyboard. **In rapid RMS.** We have a keyboard button. That will lock and unlock or open and close the keyboard.
