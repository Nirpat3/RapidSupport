# How to Synchronize Database

**[How to Synchronize Database]{.underline}**

- **How does Synchronization work ?.**

**\
Sync is for taking sync of application data.**

**\
In rapid RMS application in all module at the top bar there is one sync option.\
By applying that option if due to the internet issue some data or live update does not sync then all data sync by this option.\
\**
![](media/image1.png){width="6.5in" height="4.748611111111111in"}

- **If you want to do it whole database sncyronization process then please follow the below steps:-**

1.  Login to RapidRMS Application with your credentials.

- Enter your "**Email OR User ID**".

- Enter your "**Password**".

- Click on "**SIGN IN**" button.

![](media/image2.png){width="6.5in" height="2.9370034995625547in"}

2.  Click on "**SETTINGS**" option.

![](media/image3.png){width="6.5in" height="3.273818897637795in"}

3.  Enter your quick password and Click on "**ENTER**" button.

![](media/image4.png){width="6.5in" height="3.312424540682415in"}

4.  Click on the "**SYCHRONIZE**" option.

There are three options for database synchronization.

![](media/image5.png){width="6.5in" height="4.263888888888889in"}

5.  When applying for any option it will ask for confirmation. When applying yes then a database for that particular time duration will sync.

![](media/image6.png){width="6.5in" height="3.7222222222222223in"}

6.  Once you click on YES then it will start processing with loading page and it will take 10-15 minutes base on your database size.

![](media/image7.png){width="6.5in" height="3.670138888888889in"}

7.  Once syncing process will be finish then it will give you confirmation message.

![](media/image8.png){width="6.5in" height="3.704861111111111in"}
