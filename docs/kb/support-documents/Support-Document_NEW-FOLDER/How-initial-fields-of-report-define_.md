# How initial fields of report define_

**How initial fields of report define?**

In all report at initial there are some fields available in all kind of report like shift report, X report, Z history, ZZ history, centralized Z, centralized ZZ etc.

![](media/image1.png){width="6.5in" height="4.777777777777778in"}

They all fields define as per below.

**Current Day**: In report this field is available on top left side of report and it display name of present day within week.

**Shift user/ Current user**: This field display current log in user.

**Register**: In register field it display name of register which is available in ipad. In BOF report in register field it display name of register by which register report is filtered.

**Print date and time**: It display date and time (in 12 hr format ) of report generation.

Start date and time: It display date and time when shift start or we can say date and time when report data start.

End time and date: It display date and time when shift is close or we can say its end date of report.
