# How to approve Pump_

**How to approve Pump?**

When pump is approve then on pump fueling can start or any further process can start.

Initially pump is in ideal mode.

![](media/image1.png){width="6.5in" height="4.916666666666667in"}

- Click on pump which you want to authorised.

<!-- -->

- Now one window will open.

<!-- -->

- There is one button name as approve.

- Click that button.

![](media/image2.png){width="6.5in" height="4.708333333333333in"}

- Pump will approved and display with status authorised.

![](media/image3.png){width="6.5in" height="4.791666666666667in"}
