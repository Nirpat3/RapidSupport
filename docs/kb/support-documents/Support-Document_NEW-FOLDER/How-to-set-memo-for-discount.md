# How to set memo for discount

**How to set memo for discount ?**

When user want to add memo pop up for discount then it can be set from BOF.

Follow below procedure to set memo for discount.

- Log in with web URL.

- Go to setting tab.

- Select memo option.

- When memo open then many of ON/OFF switch display for different option.

- Review there is one option for discount.

![](media/image3.png){width="6.5in" height="3.125in"}

- When discount memo option off from BOF then in pos when give discount from discount then there is no pop up for memo.

![](media/image4.png){width="6.5in" height="4.777777777777778in"}

- Now from BOF , keep discount memo option ON.

![](media/image1.png){width="6.5in" height="3.1805555555555554in"}

- Now open application and open cash register.

- Apply discount on item.

- Click on discount pop up.

- When click on discount button then memo pop up dould open.

![](media/image2.png){width="6.5in" height="4.736111111111111in"}

- Select proper memo text for discount.

- Click on save.

- Memo text would save in discount.
