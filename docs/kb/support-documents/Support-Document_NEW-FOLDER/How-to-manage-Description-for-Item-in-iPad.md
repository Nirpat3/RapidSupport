# How to manage Description for Item in iPad

**How to Manage Description for Item in iPad:**

If you want any extra note for your Item for your reference or you want to give some information to your cashier then there is Remarks and Cashier Note facility available in Item Management.

Here are the steps on how to manage it.

- Open RapidRMS application in your iPad.

- Select Inventory Management from the dashboard.

- Enter your valid passcode to access Inventory Management Module.

- Click on New Item from the bottom of the page to add new Item or click on an item from the list to edit existing Item.

![](media/image2.png){width="6.5in" height="4.75in"}

- Add/Edit Item screen will open.

- In the top of the screen, it has Info, Pricing, Discount, History and Options tabs.

- It has Description section under Info tab.

There are Remark and Cashier Note fields under Description section.

![](media/image1.png){width="6.5in" height="4.736111111111111in"}

**Remarks:** Add any special remarks you want to give for this particular item.

**Cashier Note:** Add any notes you want to provide for the cashier for this particular item.

- Click on **Save** button to apply changes in Item. After saved added remarks and Cashier not will appear in a summary pane of this Item.
