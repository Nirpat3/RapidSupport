# How to review pump info_

**How to review pump info ?**

In pump all detail will display regarding recent pump fuling.

Pump fuel amount and volume will display in window.

Post pay : If in pump any post pay transaction is remain then that will display post pay amount.

Fuel type : Type of fuel display i.e unleaded, premium, diesel, super etc.

Current transaction will also display in list.

Pump number would display in title.
