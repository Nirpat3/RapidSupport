# How can I prevent the user to create new item, edit and delete an existing item from Inventory management

**[How can I prevent the user to create or edit or delete an existing item from Inventory Management?]{.underline}**

- Some of our customers frequently ask a question to create a new item or edit or delete existing item permission from Inventory Management.

- The New RapidRMS allows you to give permission to your user to create a new item or edit or delete an existing item from Inventory Management.

- You can set up and manage a role to create a new item, edit or delete an existing one from Role management.

If you do not know how to edit a role, please click here (<https://www.rapidrms.com/App/how-can-i-update-or-edit-role-from-role-management/>)

If you do not know how to create a new role, please [click here!](role) (Need to add link)

- **[Please follow the below steps:-]{.underline}**

1.  Login into the Back office with your credentials.

2.  Click on the Settings option.

3.  Click on the Staff menu.

![](media/image1.png){width="6.5in" height="2.9992607174103236in"}

4.  It will appear the list of an existing role.

5.  Click on the **action** button next to a role that you wish to edit**.**

> ![](media/image2.png){width="6.5in" height="0.2786482939632546in"}

6.  Select **Edit** option.

> ![](media/image3.png){width="6.5in" height="0.7962139107611549in"}

7.  Click on **+** button next to **POS Rights** option. You may need to scroll down for POS right option.

![](media/image4.png){width="6.5in" height="3.3410542432195975in"}

8.  Click on the **+** button next to the **Inventory Info** option.

> ![](media/image5.png){width="6.5in" height="3.3678904199475066in"}

9.  **Check/Uncheck the permission** you wish to give the user. For example:- If you do not want to give permission to add a new item **uncheck the Item add** role from the page.

> ![](media/image6.png){width="6.5in" height="3.008986220472441in"}

10. Click **Save.**

11. Now, assign the role to the user you want to.

If you do not know how to assign a role to a user, click here!

If you do not know how to create a user, click here!

12. It will appear below the message when users want to add a new item or edit or delete an existing item.

> ![image.png (825×618)](media/image7.png){width="6.5in" height="4.86463145231846in"}
