# How Can I remove or delete item from order

**[How can I remove or delete item from order?]{.underline}**

- If you want to remove or delete any added item, you can remove or delete the item from the purchase order, open order or receiving pending list.

<!-- -->

- Open any order.

- It will appear the item list of order.

- To select an item, click on the radio button available at the end of the record.

- 

- Select the item which you want to remove.

- At the top right corner, click on the delete button.

- It will appear an info message that the item successfully removed.

- Click on the OK button.

- It will remove the item from the list.

![](media/image1.png){width="6.5in" height="4.667190507436571in"}

- There is another way to remove an item by swiping an item.

- You can also remove an item by swiping the item.

- Swipe from left to the right side.

- It will appear delete button at end of the item.

- Click or tap on the delete button.

- It will appear the confirmation message.

- Select the Yes option. It will appear an info message that the item successfully removed.

- Click on the OK button.

- Item will be removed from the list.

![](media/image2.png){width="6.5in" height="4.704306649168854in"}

**Note: - You can follow the same procedure to delete item for Purchase order, Open order, and Receiving Pending.**
