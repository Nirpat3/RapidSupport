# How to review CC batch

**How to review CC batch?**

All credit, debit and EBT transactions listed in cc batch.

- Open rapid RMS application.

- Go to the daily report module.

- Open CC batch.

![](media/image6.png){width="6.5in" height="4.777777777777778in"}

- In cc overview graphical representation display of all transactions with a total number of transactions and total transaction amount.

![](media/image1.png){width="6.5in" height="4.361111111111111in"}

- In current transactions all the transactions of the current device and all registers will be listed with date & time, card number, type, tips, total amount, authorization path, and invoice number.

![](media/image3.png){width="6.5in" height="4.708333333333333in"}

- If you want to filter data register wise then you can filter it by register filter.

![](media/image5.png){width="6.5in" height="4.75in"}

- From card type also you can search transactions.

![](media/image4.png){width="6.5in" height="4.736111111111111in"}

- In device summary, all the transactions of credit, debit and EBT would display after the settle batch.

- When a batch is settled then all entries are removed.

![](media/image2.png){width="6.5in" height="4.722222222222222in"}

- In the device batch, all card transactions display with date & time, card number, card type, amount, tips, total amount, authorization path, and invoice number.

- The transaction can be filtered by card type and also can be searched by invoice#, Auth Code and card number.
