# How to add QSR group

**How to add QSR group?**

In inventory there are groups and sub groups are available for adding modifier category.

For adding new QSR group follow below procedure

- Open inventory management.

- Open side pane available on left side.

- In item master there is one section available name as QSR group.

- Click on that.

- All QSR group will listed.

![](media/image1.png){width="6.5in" height="4.777777777777778in"}

- At bottom bar there is one icon available"+".

- Click on that add button.

- When click on add button then one window will open for adding new QSR group with require detail.

- First is QSR group name. This is text field. Add proper name for QSR group.

- QSR group name should be unique.

- Another field is QSR group code.

- Add any convenient QSR group code.

- QSR group code is compulsory and unique field.

- With these two options QSR group can be save by save button which is available at the bottom of window.

![](media/image2.png){width="6.5in" height="4.833333333333333in"}

- In QSR group there are another useful fields are also there.

- QSR operation: QSR operation is not text field. From available operation few of operations can be select.

- For adding operation in QSR group, click on arrow which available in field QSR operation.

- When open operation window then select operation from available list. Also from add button new operation can be created.

- At a time multiple operations are allow to enter in QSR group.

![](media/image3.png){width="6.5in" height="4.805555555555555in"}

- Select operations which you want to add and click on save. Selected operations are added.

![](media/image4.png){width="6.5in" height="4.75in"}

- Now next field is QSR sub group.

- In QSR group sub group addition can be possible.

- For adding sub group in QSR group, click on arrow which available in field QSR sub group.

- List of available sub group will display.

- Select QSR sub group. Multiple choices can be selected.

- New QSR sub group can be added from Add button.

![](media/image5.png){width="6.5in" height="4.722222222222222in"}

- Click on save. QSR sub group save in QSR group.

- Now next field is modifier selection. By default this option is remain OFF.

![](media/image6.png){width="6.5in" height="4.666666666666667in"}

- If you want to ON this option then there are two radio buttons are display.

- Select one : If you select , 'select one ' option then in item from all this QSR group modifier only one modifier can be selected at a time.

![](media/image7.png){width="6.5in" height="4.666666666666667in"}

- Select min and max: If you select min and max radio button then there is field to enter minimum and maximum value of modifier selection.

- Add min and max value. That value means at the time of selling item you need to select minimum count modifier and you can select up to maximum count of modifier in item.

![](media/image8.png){width="6.5in" height="4.652777777777778in"}

- Click on save button.

- New QSR group will create and listed in QSR group.

**LINKED MODIFIER**

There is one tab available as linked modifier.

In this linked modifier are listed.

- Click on add items.

- When click on add items then all available modifier in system would display.

![](media/image9.png){width="6.5in" height="4.625in"}

- Select modifiers from list.

- Click on save button.

![](media/image10.png){width="6.5in" height="4.722222222222222in"}

- Selected modifiers are linkup with QSR group.

- If want to remove any modifier from QSR group then click on close button and save.

![](media/image11.png){width="6.5in" height="4.736111111111111in"}

- There is one search button is there for searching available modifier from list.
