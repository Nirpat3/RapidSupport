# How merchandise  departments are separately bifurcated in report

**How merchandise department is separately bifurcated in the report?**

There are different departments available with their characteristics in RapidRMS.

All departments' sales display separately with sales, returns, and the discount value in the report.

Total value = Sales value -- Return value -- discount value

**Merchandise Department**

When you sell any merchandise category assigned item. Its sales value appears in the merchandise category in the report.

![](media/image1.png){width="6.5in" height="4.805555555555555in"}

![](media/image2.png){width="6.5in" height="4.791666666666667in"}

If sold items are returned, their amount will appear in the return section.

If it applies any discount, their amount will appear in the discount section.

![](media/image3.png){width="6.5in" height="4.875in"}

![](media/image4.png){width="6.5in" height="4.930555555555555in"}

In total value is set as according.

Total = sales - refund - discount.

Note: In department which value is display with negative sign that means that value is deducted from total amount thats why display with negative sign.
