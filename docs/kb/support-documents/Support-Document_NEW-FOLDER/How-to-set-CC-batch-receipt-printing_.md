# How to set CC batch receipt printing_

**[How to set CC batch receipt printing?]{.underline}**

- If users want to set up to Print **CC Batch Receipt** automatically after users have settled the batch, Please follow the below steps.

1.  Open rapid RMS application.

2.  Click on the setting option from the bottom bar.

3.  Open tender configuration option.

4.  To Print CC Batch Receipt automatically after users have done CC Batch settle.

5.  Turn on the **Print CCBatch Receipt** toggle button.

![](media/image1.png){width="6.5in" height="4.847222222222222in"}

6.  When Users settle a batch and the printer is connected to the application, it will Print CC Batch Receipt automatically.

![](media/image2.png){width="6.5in" height="4.847222222222222in"}

![](media/image3.png){width="6.5in" height="4.708333333333333in"}

![](media/image4.png){width="6.395833333333333in" height="4.028834208223972in"}

7.  If users have not set up a **CC Batch receipt print** option, the User can print the CCBatch receipt **manually** through the print option.

![](media/image5.png){width="6.5in" height="4.791666666666667in"}

8.  It will prompt a confirmation message to **Print CC Batch Receipt**.

9.  Click on the Print option to print **CC Batch Receipt**.

10. If you do not want to print CC Batch Receipt, click on the **OK** button.
