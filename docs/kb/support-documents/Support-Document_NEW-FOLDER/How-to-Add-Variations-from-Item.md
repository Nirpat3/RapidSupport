# How to Add Variations from Item

**[How to Add Variations from Item?]{.underline}**

[Variance is a facility provided by rapidRMS that you can define the variance price of that item. Variation functionality is mainly used in restaurants. If an item has more than one form then the user can define variance for a particular item. For example, in pizza, there is variety in size i.e small, large jumbo, etc. So that all are the variances of pizza.]{.mark}

**[Please note that it allows adding three variations only.]{.mark}**

- [Open RAPID RMS application in IOS.]{.mark}

- [Select inventory management from Dashboard]{.mark}

![](media/image1.png){width="6.5in" height="4.847222222222222in"}

- [Open any item in edit mode or create a new item.]{.mark}

- [Go to the variance field.]{.mark}

- [Click on "\>" the side arrow available in the variance field.]{.mark}

![](media/image2.png){width="6.5in" height="4.847222222222222in"}

- [The variations screen to add variations will appear.]{.mark}

- [Click on add button to add an existing variation.]{.mark}

- [You can also add custom variations by selecting the custom option.]{.mark}

![](media/image3.png){width="6.5in" height="4.902777777777778in"}

- [Item variation group pop up window will appear.]{.mark}

- [Enter the variation group name and click on the save option.]{.mark}

![](media/image4.png){width="4.625in" height="2.9843755468066493in"}

- [A new variation will be created and it will also be added to the variations screen.]{.mark}

- [Now add variation value. Please follow the below example to add variation values.]{.mark}

- [I.e size is the main variation type then small, medium, large can be types of that variation value.]{.mark}

- [Click on Add button to add variation value. Add variation value pop up window will open.]{.mark}

- [Enter the Variation value and click on the save button.]{.mark}

- [You can add more than one variation value for a particular variation.]{.mark}

![](media/image5.png){width="6.5in" height="4.902777777777778in"}

- [You can add more variations values by clicking on the add button at the end of the variations.]{.mark}

![](media/image6.png){width="6.5in" height="4.791666666666667in"}

- [After configuring all types of variation click on the done button.]{.mark}

- [The variation will be added to the item.]{.mark}

- [Now, click on the save button at the bottom of the screen from item master.]{.mark}

![](media/image7.png){width="6.5in" height="4.833333333333333in"}
