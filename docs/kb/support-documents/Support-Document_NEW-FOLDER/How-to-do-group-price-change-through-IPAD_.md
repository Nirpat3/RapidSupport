# How to do group price change through IPAD_

**[How to do group price change through the [iPad]{.mark}?]{.underline}**

- The group price change is mainly used to change or update the same cost and price to multiple items or single item by Department, Vendor, Tag, and by any existing group.

- You can create Group by Department, Vendor, Tag, and by any existing group.

**[Follow the below steps to create new group:-]{.underline}**

- Open rapid RMS application.

- Go to the inventory management module from Dash board,

- Click on three line button available at the top left side corner of the screen.

- Select the Change Group price option.

- 

![](media/image1.png){width="6.5in" height="4.736111111111111in"}

- Click on Add button.

![](media/image2.png){width="6.5in" height="4.694444444444445in"}

- There are four categories available to create a new group.

1.  Group **2**. Tag **3**. Vendor **4**. Department

- Select any category.

- There is a drop-down option available to select Group, Tag, Vendor, and Department.

- When you select an option, it will appear the item list.

- Use the check box to select the item available at the beginning item.

- Click on the check box to select items.

- On the left side, select the All item check box If you want to select all items.

- Click on the Next button.

- Enter the amount to update the price.

![](media/image3.png){width="6.5in" height="4.708333333333333in"}

- Let suppose you want to change the price of any department's item then select the radio button of the department.

- Then select a particular department from the list. All available items(S) of that department would be listed.

- Select item(S) from the list.

- Click on next.

![](media/image4.png){width="6.5in" height="4.722222222222222in"}

- In the second phase, there are two options available. 1) custom price and 2) Variable price.

- Custom price: When you select custom price option then the cost and price of single, case and pack can be the change of selected items.

- Variable price: When you select variable price than items price/cost will be increase or decrease in \$ or % according to enter a value.

- When select variable price then pops up will open.

![](media/image5.png){width="6.5in" height="4.652777777777778in"}

- According to the selected option and value, item price and cost value would reflect in case pack table.

- Apply next.

![](media/image6.png){width="6.5in" height="4.708333333333333in"}

- As we have selected value increase by 8% then the new value will be increase by 8 % from the old value. If the old value is 00 then nothing will be changed.

![](media/image7.png){width="6.5in" height="4.722222222222222in"}

- Now click on finish.

- In inventory now item(S) will available with new price/cost.

- Change record will be listed.

![](media/image8.png){width="6.5in" height="4.694444444444445in"}

- In action button, there is three option available. If u will print pdf then item old and new value record will be displayed.

![](media/image9.png){width="6.5in" height="4.930555555555555in"}

- If you have change the price of any group bymistakenly. You can also revert the price and cost of items.

- To Revert, click on the action button and apply revert. Your price change will reverted and the old price will be set for the item.

- If you have reverted changes, It will display changes in red colour.

![](media/image10.png){width="6.5in" height="4.680555555555555in"}
