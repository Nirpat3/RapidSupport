# How to review centralized ZZ from manager report

**How to review centralized ZZ from manager report?**

In the daily report, centralized ZZ is the reporting section for all applied ZZ.

In centralized ZZ report, only one entry displays the current month. In the current month how many ZZ applied are displayed in one report.

- Open rapid RMS application.

- Go to the manager report.

- Open centralized ZZ.

- In centralized ZZ current month all ZZ display in one report as according to register ZZ applied.

![](media/image2.png){width="6.5in" height="4.652777777777778in"}

- If you want to view previous month ZZ history then you can do it by date filter.

- In centralize ZZ it will not filter by date its only filter by month.

![](media/image4.png){width="6.5in" height="4.763888888888889in"}

- In centralized ZZ you can view all ZZ report history in one report.

- Tap on any ZZ report.

- There is a two-section display.

- History and custom.

![](media/image5.png){width="6.5in" height="4.736111111111111in"}

- In history, all viewed history display according to register.

- In custom, you can select different register and can view ZZ history of applied ZZ by that register.

- Select custom.

- It will display all the registers by which ZZ applied.

- Select some register.

- Apply done.

- Combined register total data will display and that history will be added to the viewed history list.

![](media/image1.png){width="6.5in" height="4.791666666666667in"}

![](media/image3.png){width="6.5in" height="4.736111111111111in"}

- And in ZZ report selected register combined ZZ report data display.
