# How to enter variations pricing

**[How to enter Variations pricing?]{.underline}**

- [There are two ways to add the price of variations from Item Master.]{.mark}

<!-- -->

- [**Variations:-** It will allow the user to enter only variations prices. So when you ring up an item, it will appear variations price only.]{.mark}

- [**Variations + Retail price:-** It will allow the user to enter variations price and Retail price also, So when you ring up an item, it will appear Retail Price of an item and It will appear variations pop up window to select variation and that variation price will be added to the subtotal.]{.mark}

<!-- -->

- [To set a variation price, go to the pricing tab.]{.mark}

- [If you need to set only the variation price, select the radio button of variation.]{.mark}

![](media/image1.png){width="6.5in" height="4.791666666666667in"}

- [It will appear only variations price section in the item.]{.mark}

![](media/image2.png){width="6.5in" height="4.833333333333333in"}

- [If you need Retail price with Variations, Please select the Variations + Retail price radio button.]{.mark}

![](media/image3.png){width="6.5in" height="4.875in"}

- [Retail price and variation price options will appear in the item pricing section.]{.mark}

![](media/image4.png){width="6.5in" height="4.791666666666667in"}

- [Thus, variation prices can be set in the item.]{.mark}

**[Note:-]{.underline}** You will be able to add three variations only. It will not allow you to add more than three variations from the item.
