# How Customer can get hose report from ipad_

**[How Customer can get hose reports from the iPad?]{.underline}**

- Hose report is the detail of fuelling with unit and amount.

<!-- -->

- **[Please follow the below steps:-]{.underline}**

<!-- -->

- Configure store in iPad with the gas module.

- Take some transactions of gas or outside the pump.

- Now go to the daily report module from the Dashboard.

> ![](media/image1.png){width="6.527905730533683in" height="4.807292213473316in"}

- Scroll down from the shift report and go to the bottom section of the shift report.

- The hose report will appear at the end of the section in the report.

- There are four columns displayed in the Hose report **1)** Fuel name with the unit and amount **2)** Start **3)** Sold **4)** total.

  ---------------------------------------------------------------------------------------------------------------
               **Field**              **Description**
  ----------------------------------- ---------------------------------------------------------------------------
    Fuel Name with unit and amount    It displays the fuel name with unit and amount.

                 Start                It displays the fuel status when the shift was started and closed.

                 Sold                 It displays the current shift sold data display according to fuel selling

                 Total                It displays a total of start data and current shift data.
  ---------------------------------------------------------------------------------------------------------------

![](media/image2.png){width="6.78125in" height="5.203125546806649in"}

- **[Please follow the below steps to see hose report from X Report:-]{.underline}**

<!-- -->

- Open Daily Report.

- Select **X report** from the bottom bar.

- Scroll down to view the Hose report from the bottom of the X report.

- It will appear all shift data in the **X Report**.

![](media/image3.png){width="6.5in" height="4.875in"}

- **[Please follow the below steps to see hose report from Manager History:-]{.underline}**

<!-- -->

- Open daily report.

- Go to the manager report.

- It will appear hose report details in the Z report of Shift History, Z history, ZZ history, centralized shift, centralize Z, and centralized ZZ accordingly.

![](media/image4.png){width="6.5in" height="4.875in"}
