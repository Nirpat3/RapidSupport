# How to set modifier item as normal item

**How to set modifier item as normal item?**

Follow below procedure to make modifier as item.

- When modifier item is created then it would be listed in modifier item list.

![](media/image3.png){width="6.5in" height="4.513888888888889in"}

- In modifier item in option section there is one button modifier item that is ON.

- Go to that option and keep that option OFF.

![](media/image2.png){width="6.5in" height="4.944444444444445in"}

- Now you will have display that item into item list section.

![](media/image1.png){width="6.5in" height="4.680555555555555in"}
