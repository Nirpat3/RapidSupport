# How to calculate value in default section of department_

**How to calculate value in default section of department?**

If any department is not assigned to an item, the unassigned department will be assigned to the item automatically and its sales will appear in the default department section in the report.

Sold item sales will appear in the sales section of the department. If sold items are returned, their amount will appear in the return section.

Total = sales value - return value
