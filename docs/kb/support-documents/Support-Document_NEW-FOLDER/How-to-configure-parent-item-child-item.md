# How to configure parent item child item

## **How to configure parent item/child item?** 

- Open rapid RMS application.

- Go to the inventory management module.

- In the item list, open any item in edit mode.

- In the options section, there are two fields: Parent Item and Child Quantity.

![](media/image2.png){width="6.5in" height="4.276042213473316in"}

1)Parent Item: You can select Parent Item of the current item.

2\) Child Quantity: You can specify the quantity of child item.

- In our example:

1\) Open '6pk can Busch' for editing.

2\) Go to the Options tab.

3\) Select '30pk can Busch' as the parent item.

4\) Specify 5 as Child Quantity.
