# How to add QSR operations for modifier

**How to add QSR operations for modifier?**

In any of item when modifier is added then need to specify how much quantity required for modifier addition and for that in inventory need to add operation for modifier.

For adding operation follow below procedure.

- Open Rapid RMS application.

- Go to Inventory management module.

- Open side pan.

![](media/image1.png){width="6.5in" height="4.791666666666667in"}

- There is one tab named as QSR operation.

- There is display list of all available operations.

- There is search bar available to search operation.

**ADD OPERATION**

- Click on add button which is available at bottom bar.

![](media/image2.png){width="6.5in" height="4.666666666666667in"}

- Screen for adding new operation will open.

- Enter detail like operation name, Operation name is compulsory field to create new operation.

![](media/image3.png){width="6.5in" height="4.777777777777778in"}

- After that there is one field name as stock. In stock there is pop up.

- Value is 1,-1, 0, and 0.5-0.5.

- As according to the value selected modifier is selected at the time of modifier added into item.

![](media/image4.png){width="6.5in" height="4.791666666666667in"}

- Now click on save new operation will be added.

**EDIT OPERATION**

- Open any operation in edit mode.

- Do changes.

- Apply save button.

- Operation changes will be save.

![](media/image5.png){width="6.5in" height="4.777777777777778in"}

**DELETE OPERATION**

- Already available operation can be deleted, At the time of new creation operation cannot be deleted.

- Open any operation in edit mode.

- Apply delete button.

![](media/image6.png){width="6.5in" height="4.722222222222222in"}

- Operation will be deleted and disappear from operation list.

![](media/image7.png){width="6.5in" height="4.680555555555555in"}
