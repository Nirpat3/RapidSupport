# How to setup Epson TM T82 printer

**[How to setup Epson TM-T82 thermal printer]{.underline}**

- Your TM-T82 Wireless printer can be set up using an iPad. Follow the steps below to configure your printer for wireless printing.

<!-- -->

- Ensure the **Ethernet cable is firmly connected*** behind the Printer Device.*

![C:\\Users\\siya\\Downloads\\rtaImage.jpg](media/image1.jpeg){width="6.5in" height="5.388660323709536in"}

- **Connect the printer **to the power source using the power cable

- Verify the printer is** turned on ***(there should be a blue light on the front/top of the printer)*

**Verify IP Address & Log into Epson Web Config**

When the printer is connected and turned on a** [test slip](https://help.upserve.com/s/article/How-to-Print-a-Printer-Test-Slip) will print automatically**. Note the IP address and SSID (network) on the test slip. Then follow instructions for connecting your [iPad](https://help.upserve.com/s/article/Setting-up-the-TMM-30-Wireless-Printer#ipad) to the same SSID (network).

**\
[Connect Epson printer]{.underline}**

- Click on settings button from bottom of the dashboard.

- Enter Quick access password.

- Select **Printer** option.

- Click on **Bluetooth / TCP** option.

- Click on TCP option.

- Click on the Scan button.

- It will appear the list of the available printer.

- It will appear Printer IP address next to Status option.

- Select the **EPSON printer** option which status is online.

- You can also click on the **test printer** button to check the printer connection.
