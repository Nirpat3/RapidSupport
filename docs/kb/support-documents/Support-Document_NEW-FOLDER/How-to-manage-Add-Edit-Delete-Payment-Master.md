# How to manage Add Edit Delete Payment Master

**[How to manage(Add/Edit/Delete) payment master in inventory management?]{.underline}**

The payment master is a module in which you can add new payments, Edit and delete an existing payment from the payment master.

1.  Open RAPID RMS application in IOS.

2.  Select inventory management from the Dashboard.

![](media/image1.png){width="6.5in" height="4.5in"}

3.  Select payment master from the Item master section on the left side menu.

> Add New Payment
>
> Edit an existing payment
>
> Delete an existing payment.

**ADD NEW PAYMENT**

- At the bottom, click on add button to create a new payment

![](media/image2.png){width="6.5in" height="4.611111111111111in"}

- When clicking on new payment button then the following screen will open.

![](media/image3.png){width="6.5in" height="4.652777777777778in"}

- Payment name: This is a field for payment name. Enter payment name in this field. i.e credit card could be payment name.

- Payment code: The payment code is used for record-keeping. It can be a numerical value or an alphabet letter. Unique payment code is compulsory to add for adding new payment. i.e For credit card payment code could be CC.

- Payment type: Payment type is a type of payment method by which you can pay ordered value. There is one pop up window for payment type selection. From all, you can select only one payment type at a time for payment.

![](media/image4.png){width="6.5in" height="4.708333333333333in"}

- Surcharge: Surcharge is for extra charge or fee. If you would like to charge the customer on this payment method then select surcharge. Amount of surcharge could be in % or \$. For i.e for credit card payment method want to apply 10\$ surcharge on it.

- Drop amount: Only select drop amount for a particular method if you want to take cash out from the drawer. If the drop amount is selected then any amount charged will be in a negative value.

- Click on the Save button.

![](media/image5.png){width="6.5in" height="4.708333333333333in"}

**EDIT PAYMENT**

You can edit already existing payment detail.

- Click on any payment name for editing payment detail. Payment will open in edit mode.

The following screen will open when clicking any payment in edit mode.

![](media/image6.png){width="6.5in" height="4.694444444444445in"}

- When open payment in edit mode then payment appears with payment name, code and type. If you want to change any detail then edit the detail and click on save button.

- If you don't want to edit then click on cancel button.

**DELETE PAYMENT**

- When want to delete already exist payment then open that payment in edit mode and click on the delete button. And click on yes then payment will be deleted.

- Newly create payment you are not able to delete. The only existing payment you can delete.

![](media/image7.png){width="6.5in" height="4.994792213473316in"}
