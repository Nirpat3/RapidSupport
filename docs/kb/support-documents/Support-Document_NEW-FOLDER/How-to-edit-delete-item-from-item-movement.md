# How to edit delete item from item movement

**How to edit/delete the item from item movement?**

When you want to edit or delete items which are available in item movement list, then the item you edit will update in main inventory and item you delete will delete only from item movement list.

- Open rapid RMS application.

- Go to item movement module.

- Swipe item.

- Two option display edit and delete.

- Apply edit option.

- In item after change applies to save.

![](media/image4.png){width="6.5in" height="4.75in"}

- If you apply delete then item will delete from item movement list only not from inventory.
