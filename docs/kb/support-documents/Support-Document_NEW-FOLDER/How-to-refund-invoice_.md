# How to refund invoice_

**How to refund invoice?**

When a user wants to refund selling invoice then rapid RMS gives a facility to refund invoice.

- Open rapid RMS application.

- Go to the cash register module.

![](media/image1.png){width="6.5in" height="4.527777777777778in"}

- From top left option select invoice.

![](media/image2.png){width="6.5in" height="4.638888888888889in"}

- All available invoices listed according to date and month.

- The open invoice which you want to refund.

- Select all items available in the invoice or select only those items(S) which you want to refund.

- Click on refund option which is available on the top bar.

![](media/image3.png){width="6.5in" height="4.708333333333333in"}

- Items will ring as a refund.

![](media/image4.png){width="6.5in" height="4.680555555555555in"}

- Complete transaction.

- Refund process complete.

Note: In the invoice, if there is already refund items then you cannot select those items.
