# How to apply Z_

**How to apply Z?**

A daily report is module for selling detail reporting.

All shift data display in x report and when applying z then all x data reset and report move into Z history.

Generally, z is applied at the end of the day to review daily report data and detail.

**[Note:-]{.underline}** Reporting might be mismatch if the Shift or day close occurs during a fuel transaction. It is recommended that shift or day close should be performed when all pumps are clear.

- Open rapid RMS application.

- Go to daily report module.

- At the bottom bar, there is one option named "Z report".

![](media/image3.png){width="6.5in" height="4.680555555555555in"}

- Click on Z for apply z.

![](media/image1.png){width="6.5in" height="4.819444444444445in"}

- Apply yes. Z will apply properly.

- At the time of z apply shift will close and ask for the close amount.

![](media/image4.png){width="6.5in" height="4.75in"}

- When applying z then it will ask for print that report.

- If you want to print then you can print with it from here.

![](media/image2.png){width="6.5in" height="4.777777777777778in"}

- When applying z then x is reset. And a new shift open.

![](media/image5.png){width="6.5in" height="4.777777777777778in"}
