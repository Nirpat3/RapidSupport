# How to edit an existing Sub Department

**[How to edit an existing sub-department from Sub-department Master?]{.underline}**

- Open RAPID RMS application.

- Select inventory management module from Dashboard.

![](media/image1.png){width="6.5in" height="4.597222222222222in"}

- Click on three line option available from the top left side corner of the screen.

- Select sub-department master from the list.

- It will appear sub-department list.

![](media/image2.png){width="6.5in" height="4.239680664916885in"}

- Select any sub-department to open in edit mode.

![C:\\Users\\siya\\Downloads\\MicrosoftTeams-image (69).png](media/image3.png){width="6.5in" height="4.184199475065617in"}

- Change the details that you want to update.

- Click on the Save button.

- If you do not want to save any changes click on the close button.

- Three tabs appear beside the sub-department.

  ------------------------------------------------------------------------------------------------------------------
    **Tabs**   **Description**
  ------------ -----------------------------------------------------------------------------------------------------
      Info     It describes the Sub-department details.

     Items     It displays the list of items associated with the sub-department.

   Department  It displays the list of departments that the current sub-department is allocated to the department.
  ------------------------------------------------------------------------------------------------------------------
