# How can I delete receiving pending invoices

**[How can I delete receiving pending invoices?]{.underline}**

- Please follow the below steps:-

<!-- -->

- Open Rapid RMS application.

- Open the purchase order module from the dashboard.

- Select the receiving pending option.

> ![](media/image1.png){width="6.5in" height="4.458333333333333in"}

- It will appear all receiving pending orders which are moved from open order.

- Now swipe right to the left side to delete the invoice.

- It will appear the delete option at the end of an invoice.

> ![](media/image2.png){width="6.5in" height="4.736111111111111in"}

- Click on the delete option.

- It will appear the confirmation message to delete the invoice.

- Select the Yes option.

> ![](media/image3.png){width="6.5in" height="4.722222222222222in"}

- It will appear a message to delete an order or invoice successfully.
