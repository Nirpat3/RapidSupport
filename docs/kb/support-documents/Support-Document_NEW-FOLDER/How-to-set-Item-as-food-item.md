# How to set Item as food item

**How to set Item as food item ?**

In inventory when item is food item then only modifier can be added at the time of ring up.

To set item as food item follow below procedure.

- Open inventory management module.

- Go to Item list.

![](media/image1.png){width="6.5in" height="4.472222222222222in"}

- Open any of item in edit mode.

- In option section there is one option named as "Is food item".

- Default this option is OFF.

- Keep this option ON.

![](media/image2.png){width="6.5in" height="4.638888888888889in"}

- When in any time if food item option is ON and in cash register ring up that item then modifier can be added into that item.
