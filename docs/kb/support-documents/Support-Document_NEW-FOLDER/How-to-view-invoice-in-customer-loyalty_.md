# How to view invoice in customer loyalty_

**How to view an invoice in customer loyalty?**

In customer loyalty, all the invoices are displayed which items are purchased by that customer.

- Open rapid RMS application.

- Go to the customer loyalty module.

![](media/image6.png){width="6.5in" height="4.763888888888889in"}

- View that customer.

- In the invoice, there is two tab display. i) The item list and ii) Invoice list.

**Item list :**

- In the item list, all the items are displayed with image, item name, UPC, department name, Invoice \# and price which are purchased by that customer.

- Item can be searched by item name and UPC.

- If you want to review the item of a particular time duration then it can be possible by date range & time.

![](media/image2.png){width="6.5in" height="4.791666666666667in"}

**Invoice list :**

- In invoice list all the invoices display which is purchased by that customer with date/time, invoice #, qty, payment type, total, tags.

- Invoices can be searched by invoice \# and payment type.

![](media/image3.png){width="6.5in" height="4.722222222222222in"}

- If you want to view the invoice of any time frame then it can be possible by date and time range menu.

![](media/image5.png){width="6.5in" height="4.805555555555555in"}

- There is one action button available in every invoice.

![](media/image4.png){width="6.5in" height="4.777777777777778in"}

- From view invoice detail display.

![](media/image1.png){width="6.5in" height="4.861111111111111in"}

- From print, option invoice can be print if the printer connected to the system.

- From email, option email can be sent to that customer.

![](media/image7.png){width="6.5in" height="4.763888888888889in"}
