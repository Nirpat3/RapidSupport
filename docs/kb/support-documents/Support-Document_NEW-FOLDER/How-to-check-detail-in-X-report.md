# How to check detail in X report

**[How to check detail in X report?]{.underline}**

- At the end of the trading day, you can check total sales data according to multiple shifts open and closed.

<!-- -->

- **[Please follow the below steps:-]{.underline}**

<!-- -->

- Open rapid RMS application.

- Go to Daily report module.

![](media/image1.png){width="6.5in" height="4.75in"}

- At the bottom bar, select **X report** option.

![](media/image2.png){width="6.5in" height="4.777777777777778in"}

- Shift start date, end date, register, current user batch \# etc details appears in X Report.

- Daily sales, gross sales, tax details, payment details, discount details, department wise bifurcation appear in x Report.

- All shifts data appear together in X Report.

- You will be able to review data in chart wise. It represents data in Column and Pie chart wise.

- You can review your chart data by \$ and % wise accordingly.

- It will appear data by \$ wise by default.

- To view data in % percentage wise, select % option from top middle of the page.
