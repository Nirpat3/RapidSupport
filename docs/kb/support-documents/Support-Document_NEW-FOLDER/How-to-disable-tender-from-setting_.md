# How to disable tender from setting_

**[How to disable tender from setting?]{.underline}**

- **[Please follow the below steps to disable tender:-]{.underline}**

<!-- -->

- Open rapid RMS application.

- At the bottom, click on the setting option from the dash board.

- Enter a quick access password.

![C:\\Users\\mit shah\\Downloads\\MicrosoftTeams-image (8).png](media/image1.jpeg){width="6.268055555555556in" height="3.65541447944007in"}

- Select the Tender configuration option from the right side option.

- It will appear the list of all available tender.

- Select tender from the list.

![MicrosoftTeams-image (91).png](media/image2.png){width="6.5in" height="4.875in"}

- As you can see in the above image, the Check payment type or tender type appears.

- I.e. We are going to disable the Check tender type or payment type .

![](media/image3.png){width="6.5in" height="4.8684208223972005in"}

- There will appear the tender disable toggle button.

- By default, the tender disable toggle button is turned off.

- To disable the tender, turn on the tender disable toggle button.

![](media/image4.png){width="6.5in" height="4.791666666666667in"}

- It will not appear tender temporarily while you have turned on tender disable the toggle button.

![](media/image5.png){width="6.5in" height="4.763888888888889in"}
