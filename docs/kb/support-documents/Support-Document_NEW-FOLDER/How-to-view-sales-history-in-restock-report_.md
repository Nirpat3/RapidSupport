# How to view sales history in restock report_

**How to view sales history in the restock report?**

Restock report is a module for the complete lack of item stock which is already sales.

In sales History all the items display which are already sales.

- Open rapid RMS application.

- Go to restock report.

![](media/image1.png){width="6.5in" height="4.638888888888889in"}

- All the items which are sale in the current day is displayed in the list.

![](media/image4.png){width="6.5in" height="4.763888888888889in"}

- If you want to filter item date wise then you can filter by date range option.

- Click on the date range option and it will show as a calendar.

- Select the proper date.

- Selling Item list will filter by selected date range.

![](media/image3.png){width="6.5in" height="4.875in"}

![](media/image2.png){width="6.5in" height="4.777777777777778in"}

- Sometimes if all items do not reload at a time then by refreshing button list will reload and all items display.

![](media/image5.png){width="6.5in" height="4.791666666666667in"}

- If you want to filter item category wise then you can filter by category filter option.

- Click on the filter option available at the top right corner of the screen..

- Select the category and select category option from the list.

- Item list will appear by selected category.

![](media/image6.png){width="6.5in" height="4.888888888888889in"}

- Please select the category from available option.

- All category options will appear in the list.

![](media/image8.png){width="6.5in" height="4.901042213473316in"}

![](media/image7.png){width="6.5in" height="4.875in"}

- Select the option from the category list.

- Click or tap on apply button.

- Selected category option items will appear in the list.
