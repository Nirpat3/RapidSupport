# How to update department services

**[How to update department services?]{.underline}**

- The User cannot create or delete any services. The user can only update these services.

**To update department service, please follow the below procedure:-**

- Open Rapid RMS application.

- Select the inventory management module from the dashboard.

- Enter your quick access password.

![](media/image1.png){width="6.5in" height="4.777777777777778in"}

- Click on the three-line button from the top left side corner of the screen.

- Select the Department option from the left side menu.

- Select Services tab appear beside the department tab.

- The list of all default services will appear in the services tab.

![C:\\Users\\siya\\Downloads\\Department-services-will-work-02.png](media/image2.png){width="6.5in" height="4.70076990376203in"}

- You will be able to update only the enabled service from the list.

- When you enable any service then you can open that service in edit mode only.

- Click on services to open in edit mode.

![](media/image3.png){width="6.5in" height="4.791666666666667in"}

+-----------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **Field Name**  | **Description**                                                                                                                                                                                                                                                                          |

| Department Name | It describes the name of **Service**.                                                                                                                                                                                                                                                    |

If you have only entered the cost of service, it will define the retail price of service as per entered percentage amount in the profit field automatically.                                                                                     |

| EBT             | If you want to apply EBT on any service. Please turn on the toggle button to select any of option from the drop-down.                                                                                                                                                     |

|                 |                                                                                                                                                                                                                                                                                          |

- Change the necessary details you want to update.

- Tap on the Save button.

- If you do not want to save any changes, click on the close button.
