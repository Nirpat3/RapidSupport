# How to deliver an Open order from PO

**[How to deliver an Open order in PO?]{.underline}**

- **[Please follow the below steps for delivery of an order:-]{.underline}**

<!-- -->

- Go to rapid RMS application.

- Open the purchase order module from the dashboard.

- Select the open order option.

![](media/image2.png){width="6.5in" height="4.708333333333333in"}

- All created open orders will appear in the PO list option.

- It will appear open order record Date, time, Open order Number, Title of an open order, User name, and receiving option.

  -------------------------------------------------------------------------------------------------------------------------
   **Column Name**  **Description.**
  ----------------- -------------------------------------------------------------------------------------------------------
        Date        It describes the open order date.

        Time        It describes the open order time.

        Open#       It describes the open order number.

        Title       It describes the open order title which you entered while creating an open order.

        User        It appears the user name who created the open order.

      Receiving     Receiving is a button. It is mainly used to deliver an order when you receive an order from a vendor.
  -------------------------------------------------------------------------------------------------------------------------

![C:\\Users\\siya\\Downloads\\MicrosoftTeams-image (63).png](media/image3.png){width="6.5in" height="4.87373031496063in"}

- There are two ways to deliver an order.

<!-- -->

- By receiving option

- By Delivery option from open order.

<!-- -->

- **By receiving option**

<!-- -->

- There is a receiving option available at end of the record of an open order.

<!-- -->

- Click on receiving option.

- After clicking on receiving option, the set Invoice number pop-up window will appear.

- Enter the valid invoice number. **Note:-** Invoice number is a compulsory field. Without entering Invoice #, it will not allow a user to deliver the order.

- Click on the Save button.

![](media/image4.png){width="6.5in" height="4.38132874015748in"}

- After saving the order, it will move to the delivery pending list.

![](media/image5.png){width="6.5in" height="4.625391513560805in"}

- **By Delivery option from open order.**

<!-- -->

- Open the open order list.

- Select any order which you want to deliver.

- Click on the delivery option available at the bottom of the screen.

- After clicking on the delivery option, the set Invoice number pop-up window will appear.

![](media/image6.png){width="6.5in" height="4.173912948381452in"}

![](media/image7.png){width="6.5in" height="3.9574496937882766in"}

- Enter the valid invoice number. **Note:-** Invoice number is a compulsory field. Without entering Invoice #, it will not allow a user to deliver the order.

<!-- -->

- Click on the Save button.

![](media/image8.png){width="6.5in" height="4.588634076990377in"}

- Order will move to the Receiving pending list.
