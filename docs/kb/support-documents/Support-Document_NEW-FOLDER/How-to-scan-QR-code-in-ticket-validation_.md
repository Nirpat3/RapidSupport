# How to scan QR code in ticket validation_

**How to scan QR code in ticket validation?**

For scanning ticket validation code the ticket validation module is used.

For ticket, validation code follows the below procedure.

- Open rapid RMS application.

- Go to inventory management module and open any item in edit mode.

- Go to the options tab and open pass field.

- In ticket pass value.

- Apply to save.

- In item pass/ticket value configured.

![](media/image2.png){width="6.5in" height="4.75in"}

- Now go to the cash register module.

- Ring up pass item and complete transaction.

- Make sure the printer is connected with a printer.

- After completion of transaction pass receipt will print.

![](media/image4.png){width="3.4895833333333335in" height="4.958333333333333in"}

- Now go to the ticket validation module.

![](media/image1.png){width="6.5in" height="4.833333333333333in"}

- There is one option QR code.

- Now scan that printed receipt.

![](media/image5.png){width="6.5in" height="4.805555555555555in"}

- When a barcode is a scan then ticket pass display.

![](media/image3.png){width="6.5in" height="4.930555555555555in"}
