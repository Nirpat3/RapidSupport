# How to set notification for local network connection in RapidRMS_

There are two different processes to turn on the local network.

1st Process : At the time of Install RapidRMS application.

2nd Process : If someone forgot to do this at the time of Install RapidRMS application then Please follow the 2nd process. It is only helpful when you have an app installed on the iPad.

**[1st Process]{.underline}** :

1.  Install **RapidRMS** application from App Store.

2.  Launch **RapidRMS** from iPad dashboard.

3.  At the time of launching the application, a pop up for local network permission would display like in the image below.\
    \
    ![image](media/image1.jpg){width="6.267716535433071in" height="4.708333333333333in"}

4.  Select \"**OK**\". It would allow RapidRMS to connect with local network devices like printers, PAX or any other local network devices.

5.  If you select "Don't Allow" in notification pop up, then you can not connect Rapid RMS applications with local network devices.

6.  We advise you to select \"**OK\"** to get connected with network devices.

> **[2nd Process]{.underline}** :

1.  Open **Setting** application from iPad.

2.  Go to the **RapidRMS** application.

3.  In the Rapid setting there is an option for local network. Turn on the Local Network toggle.\
    \
    ![image](media/image2.jpg){width="6.267716535433071in" height="4.736111111111111in"}
