# How to define drop amount in report_

**How to define drop amount in report?**

In pos there is one option as available in left side pane name as a drop.

![](media/image1.png){width="6.5in" height="4.777777777777778in"}

When select that option and add value or amount in drawer then that value is consider as drop amount.

![](media/image2.png){width="6.5in" height="4.638888888888889in"}

That added value display in drop amount. Its total of all drop amount during a shift. #shift #monthly #daily report.

This drop amount is in \$.

![](media/image3.png){width="6.5in" height="4.791666666666667in"}
