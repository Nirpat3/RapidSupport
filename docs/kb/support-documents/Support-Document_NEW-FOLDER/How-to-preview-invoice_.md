# How to preview invoice_

**How to preview invoice?**

- Open rapid RMS application.

- Go to the cash register.

- Open invoice list.

- Open any invoice you want to see detail.

- At the bottom bar, there is one option name as a preview.

- Hit that button.

![](media/image1.png){width="6.5in" height="4.722222222222222in"}

- Invoice detail will display.
