# How to deliver order in PO

**How to deliver an order in PO?**

In purchase order after adding items in purchase order list need to create order and after that need to deliver that created order. Follow the below procedure for delivering an order.

- Go to rapid RMS application.

- Open purchase order module from the dashboard.

- Hit on Open order.

![](media/image4.png){width="6.5in" height="4.708333333333333in"}

- All the created order list would display.

- In that date, time, open #, title, user and delivery option would display.

![](media/image5.png){width="6.5in" height="4.239725503062117in"}

There are two ways to deliver an order.

1)  From delivery option directly order can deliver.

- Hit deliver option.

- When click on deliver option then one pop up for enter invoice \# would display.

- Enter the proper invoice#.

- Apply to save.

- Invoice \# is a compulsory field. Without an invoice #, order can not be delivered.

![](media/image3.png){width="6.5in" height="4.38132874015748in"}

- After save order will move to the delivery pending list.

![](media/image1.png){width="6.5in" height="4.625391513560805in"}

2)  Another way is to deliver an order from opening open order.

- Open, open order list.

- Select any of order which you want to deliver.

- Apply delivery option which is available at the bottom of the screen.

- It will ask for a delivery invoice #.

- Enter the invoice number.

![](media/image2.png){width="6.5in" height="4.378947944006999in"}

![](media/image6.png){width="6.5in" height="4.381461067366579in"}

- Order will move to the delivery pending list.

![](media/image7.png){width="6.5in" height="4.588634076990377in"}
