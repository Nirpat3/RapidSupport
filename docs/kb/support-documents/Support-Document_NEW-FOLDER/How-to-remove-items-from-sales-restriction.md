# How to remove items from sales restriction

**How to remove items from sales restriction?**

When item need to remove from sales restriction and want to sell in store the need to remove from blue law in BOF.

For item remove follow below process.

- Log in with bof office web url.

- Make sure e commerce option should ON in store.

- Go to blue law tab.

- Open any one entry in edit mode by action button.

![](media/image2.png){width="6.5in" height="3.8094017935258093in"}

- Go to items tab in it.

- All items which are under that sales restriction would display.

![](media/image1.png){width="6.5in" height="3.619792213473316in"}

- Now select items which you want to remove from list.

- Click on delete.

![](media/image3.png){width="6.5in" height="3.7467344706911634in"}

- Selected item(S) will remove from list.

![](media/image4.png){width="6.5in" height="3.713542213473316in"}
