# How to deposit

**How to deposit?**

When a store has to make a deposit on any item and when after completion of item limit store want to return that item then deposit functionality used.

**Set deposit**

- Go to the inventory management module.

- Open any item in edit mode.

- In option ON fee/deposit.

- Whichever fee/deposit available in-store it will display in the list.

- Select any of deposit.

- 

![](media/image2.png){width="6.5in" height="4.680555555555555in"}

- Save item.

![](media/image3.png){width="6.5in" height="4.694444444444445in"}

**Return item**

- Go to the cash register.

- At the bottom bar, there is one option named as more click on that option.

![](media/image1.png){width="6.5in" height="4.763888888888889in"}

- Click on deposit from all of them.

- Now ring up the item in which deposit applied.

- Item will ring up with deposit amount.

- By this way, the deposit amount will return.
