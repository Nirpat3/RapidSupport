# How to delete an existing tender from POS

**[How to delete an existing tender from POS?]{.underline}**

- Click on the settings option ![C:\\Users\\mit shah\\AppData\\Local\\Microsoft\\Windows\\INetCache\\IE\\BU8Y79T3\\WMF-Agora-Settings_BCBCBC.svg\[1\].png](media/image1.png){width="0.17463910761154855in" height="0.17463910761154855in"} available at the bottom of the DashBoard.

![C:\\Users\\mit shah\\Downloads\\MicrosoftTeams-image (8).png](media/image2.jpeg){width="6.262498906386702in" height="3.5326082677165354in"}

- Enter your quick access password.

![](media/image3.png){width="6.5in" height="3.881519028871391in"}

- Select the Tender configuration option from the right side menu.

![](media/image4.png){width="6.5in" height="4.230227471566054in"}

- On the below right side, click on the payment list button.

- When you click on the Payment list button, It will appear the list of an existing tender.

![](media/image5.png){width="6.5in" height="3.9124989063867015in"}

- Select the tender that you wish to delete from the list.

![C:\\Users\\siya\\Downloads\\MicrosoftTeams-image (89).png](media/image6.png){width="6.4944444444444445in" height="4.239129483814523in"}

- At the bottom, click on the Delete button.

- It will not appear the removed payment in the existing list of the payment.
