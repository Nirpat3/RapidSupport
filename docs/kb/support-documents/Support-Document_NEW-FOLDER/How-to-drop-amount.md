# How to drop amount

**How to drop the amount?**

It allows users to remove excess money from the register to deposit it into a secure location at the store.

- Select the cash register icon. Enter the 4-digits password.

- Select the menu button from the top left-hand corner.\
  \
  ![](media/image1.png){width="6.5in" height="4.567708880139983in"}

- Select Drop.

- A message will prompt you if you would like to open the cash drawer. Select the **Yes** option. The drawer will open.

- The number pad will open to enter the drop amount.

> ![](media/image2.png){width="6.5in" height="4.088542213473316in"}

- Enter amount.

- Click on the drop button.

- When you do the drop transaction, the total amount will appear in the drop sales field in the shift report.
