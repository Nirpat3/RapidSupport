# How to link up modifier in item with category and sub category inclusion

**How to link up modifier in item with category and sub category inclusion ?**

In RCR when ring up food item then in customize window modifier list will display to apply modifier in item but for display this list need to set everything from inventory.

For that follow below procedure.

- Go to inventory and create new item i.e pizza.

- Keep 'is food item section ON from options section.

![](media/image1.png){width="6.5in" height="4.625in"}

- Go to modifier tab and add some modifier.

![](media/image3.png){width="6.5in" height="4.819444444444445in"}

- When ring up that item in RCR then in selected modifiers display in modifier window.

![](media/image5.png){width="6.5in" height="4.597222222222222in"}

- Now as concern with the category go to category tab and modifiers in category.Below are the modifiers of topping category.

![](media/image6.png){width="5.84375in" height="5.604166666666667in"}

- Now go to sub category and in sub category add some modifiers. I.e sub category - special topping.

![](media/image4.png){width="6.5in" height="4.763888888888889in"}

- Now go to cash register and ring up that item.

- Now open modifier pop up when select category and sub category then all modifiers will display which we have set in inventory.

![](media/image2.png){width="6.5in" height="4.5in"}

Note : When modifier is available in item then only it will display at the the time of ring up that item according to category and sub category.
