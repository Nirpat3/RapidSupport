# How to download app from testflight or update

**How to download an app from Test flight or update?**

- Open test flight app from your iPad.

> ![C:\\Users\\siya\\Downloads\\MicrosoftTeams-image (95).png](media/image1.png){width="6.5in" height="4.8765879265091865in"}

- If the Rapid RMS application is already installed from test flight it displays **Open** option from test flight, else it will appear **Install** option to download the application.

- Click on install button to download the Rapid RMS app.

> ![](media/image2.png){width="6.5in" height="3.675696631671041in"}

- If you have already installed the application from App store, Pop up for conformation would display to replace the app with test flight build. Click on install.

> ![](media/image3.png){width="6.5in" height="4.051115485564305in"}

- When you install the application, the app installation process will appear on the dashboard.

> ![](media/image4.png){width="6.5in" height="4.861858048993875in"}

- If there is previous build version available of test flight then it should display with an update option.

- Hit the update button.

> ![](media/image5.png){width="6.5in" height="3.732004593175853in"}

- Test flight latest version build will start downloading.
