# How to restock item from restocking history_

**How to restock items from restocking history?**

In sales history, all the items that are sold are displayed.

In restocking all the items which are restocked from sales history will display, But after restock items will remove from restocking history also.

- Open rapid RMS application.

- Go to restock report module.

- Open restocking history.

- Selected items which you want to remove from the list and already add to stock.

![](media/image1.png){width="6.5in" height="4.722222222222222in"}

- Apply option restocked.

- Without the selection of an item, the restocked icon won\'t be enabled.

- Items will be removed from the list.

![](media/image2.png){width="6.5in" height="4.541666666666667in"}

Note: If item will again sell then it will be listed in sales history and the whole process goes further.
