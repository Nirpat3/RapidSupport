# How to define closing amount in report_

**How to define closing amount in report ?**

This value in report is define the value which user have entered at the time of close shift.

![](media/image2.png){width="6.5in" height="4.722222222222222in"}

If at the time of shift close there is not entered any value then default it should display zero.

![](media/image1.png){width="6.5in" height="4.694444444444445in"}

NOTE : At the timing of shift close amount is not compulsory to enter but if user enter any value then in report it display at the time of shift close.

When apply Z or ZZ at that time also shift close so that value also consider in closing amount.
