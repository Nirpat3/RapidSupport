# How to open close shift from daily report

**[How do I open and close shifts from the daily report?]{.underline}**

- The Shift report is a printable report that provides an overview of each employee\'s activity and all transaction details during their shift.

<!-- -->

- **Please follow the below steps:-**

<!-- -->

- Open rapid RMS application.

- Open Daily report module from Dashboard.

![](media/image1.png){width="6.5in" height="4.75in"}

- The Shift report will open by default when you open the Daily report module.

- If the user has already closed the shift, the open shift button will appear in enable mode and report details will appear blank.

![](media/image2.png){width="6.5in" height="4.777777777777778in"}

- On the top middle of the report, click on the Open shift button.

- It will open Pop up window to enter the opening amount.

- Enter the opening amount if you would like to enter the amount.

- If you would like to open a shift with zero amount, you can open a shift with zero amount.

![](media/image3.png){width="6.5in" height="4.819444444444445in"}

- It will open the shift and the Close shift button will appear in enable mode.

- All transaction details or data will appear taken by the user in the particular shift report.

![](media/image4.png){width="6.5in" height="4.763888888888889in"}

- If you want to close the shift, click on the close shift button at the top middle side of the report.

- Now, it will open Pop up window to enter the closing amount.

- Enter the amount in the closing amount window.

![](media/image5.png){width="6.5in" height="4.777777777777778in"}

- When you click on the close button, it will pop up for the print shift report.

- Click on the Print option available next to the report if you would like to print the report.

![](media/image6.png){width="6.5in" height="4.736111111111111in"}
