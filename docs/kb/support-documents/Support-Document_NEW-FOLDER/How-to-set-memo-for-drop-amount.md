# How to set memo for drop amount

**How to set memo for drop amount ?**

When user want to add memo pop up for drop amount then it can be set from BOF.

Follow below procedure to set memo for drop amount

- Log in with web URL.

- Go to setting tab.

- Select memo option.

- When memo opens then many of ON/OFF switch display for different options.

- Review there is one option for drop amount.

- Keep drop amount memo option OFF.

![](media/image4.png){width="6.5in" height="3.375in"}

- Now open rapid rms application and go to cash register.

- Click on drop amount.

- When click on drop amount then pop up for memo would not open.

![](media/image5.png){width="6.5in" height="4.722222222222222in"}

- Now from BOF keep drop amount option ON.

![](media/image3.png){width="6.5in" height="3.7157994313210847in"}

- Restart application and open cash register.

- Click on drop amount.

![](media/image6.png){width="6.5in" height="4.666666666666667in"}

- When click on drop amount then there should be option for drawer open.

- Hit no for drawer open.

![](media/image7.png){width="6.5in" height="4.791666666666667in"}

- At that time pop up for memo would open.

- Now select proper text for memo and save it.

![](media/image2.png){width="6.5in" height="4.763888888888889in"}

- Memo would save and then after pop up for drop amount would open.

- Set drop amount and click on drop.

![](media/image1.png){width="6.5in" height="4.777777777777778in"}

- Drop amount and memo text would set in drop amount.
