# How to view Transfer order history from Inventory Transfer module

**[How to view Transfer order history from the Inventory Transfer module?]{.underline}**

User can use this function when he/she want to check transfer order history.

Follow below steps to check transfer order history

- Open Rapid RMS app.

- Go to Inventory transfer module.

- Open 'Transfer order history' module.

![](media/image1.png){width="6.5in" height="4.777777777777778in"}

- Now you can see, transfer order history with its detail ex. Invoice #, Remark, Title etc.

- Tap on any row for further process.

![](media/image2.png){width="6.5in" height="4.75in"}

- Now you can see, Transfer Order History with whole transfer items with its qty.

- You can email, print and check info by using the bottom icon. Ex. Info and Export icon.
