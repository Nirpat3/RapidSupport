# How to set up a BOFA Gateway from Back office

- **Please follow the below steps:-**

1.  Login to BOS with your user credentials.

2.  Tap menu ![](media/image1.png){width="0.20499890638670165in" height="0.20499890638670165in"} (located on the top left side)

3.  Tap settings.

4.  Tap store.

5.  Scroll to see the Gateway option. Select the Gateway option.

![](media/image2.png){width="6.538540026246719in" height="3.0043744531933507in"}

6.  Select the Add option.

![](media/image3.png){width="6.539577865266842in" height="2.9947911198600177in"}

7.  Click on the Payment drop menu to select **a BOFA (Bank of America)** payment gateway.

![](media/image4.jpeg){width="6.695138888888889in" height="3.077777777777778in"}

![](media/image5.jpeg){width="6.691666666666666in" height="2.8673611111111112in"}

+----------------+------------------------------------------------------------------------------------+
| > **Fields**   | > **Description**                                                                  |
+================+====================================================================================+
| Description    | Please enter the Payment gateway name. For example:- Bank of America               |
+----------------+------------------------------------------------------------------------------------+
| Operation mode | Select Live mode.                                                                  |
+----------------+------------------------------------------------------------------------------------+
| Allow manual   | If you want to do the process manually while doing the payment process from your   |
|                |                                                                                    |
| processing     | POS System, select **Allow manual processing** checkbox.                           |
+----------------+------------------------------------------------------------------------------------+
| IsDefault      | Select the **Isdefault** check box to use the payment gateway by default from your |
|                |                                                                                    |
|                | POS.                                                                               |
+----------------+------------------------------------------------------------------------------------+
| Gateway        | Select the **Gateway platform** on which platform you want to use the selected     |
|                |                                                                                    |
| Platform       | gateway.                                                                           |
+----------------+------------------------------------------------------------------------------------+

8.  After entering all details, Tap on the **Save** button.
