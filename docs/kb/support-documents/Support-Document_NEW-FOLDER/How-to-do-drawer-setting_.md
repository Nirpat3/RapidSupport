# How to do drawer setting_

**How to do drawer setting?**

When you complete transaction or when you do no sales then drawer will open as per done settings accordingly. You can set cash drawer from settings.

- Open rapid RMS application.

- Go to the setting module.

- Tap to the tender setting.

- It will appear the list of tender list over bottom right side.

- Select the tender for you wish to open cash drawer.

- The default cash drawer is off.

![](media/image1.png){width="6.5in" height="4.75in"}

- If **Open Cash Drawer** option is set off then drawer will not open.

![](media/image2.png){width="6.5in" height="4.763888888888889in"}

- The cash drawer will open if **Open Cash Drawer** option is set **ON** at the time of transaction.

- If **Open Cash Drawer** option is set **ON** and applies no sales or drop amount then drawer will open.

- You can set Open Cash drawer option ON for the tender you wish to open cash drawer at the time of transaction.

![](media/image3.png){width="6.5in" height="4.722222222222222in"}

- In any payment type, if the open cash drawer is set on and drawer is connected with the system then after completing of the transaction drawer will open automatically.

![](media/image4.png){width="6.5in" height="4.819444444444445in"}

- If the option is ON and drawer is not connected then alert message will pop up.

![](media/image5.png){width="6.5in" height="4.763888888888889in"}
