# How to delete an existing department

**[How to delete an existing department?]{.underline}**

- **It is highly recommended to check below steps before deleting department.**

1.  Please make sure that you have not assigned the department to any **item or sub-department**.

2.  If you have assigned department, please make sure that you have moved the item or sub-department to another Department.

- Open RAPID RMS application.

- Select inventory management module from Dashboard.

![](media/image1.png){width="6.5in" height="4.583333333333333in"}

- Click on the three-line button from the top left side corner of the screen.

- Select the department option from the list.

![](media/image2.png){width="6.5in" height="4.75in"}

- It will appear the department list.

- Select any department to open in edit mode.

![C:\\Users\\siya\\Downloads\\MicrosoftTeams-image (72).png](media/image3.png){width="6.5in" height="4.87373031496063in"}

- Click on the Delete button available at the bottom of the screen.

- The confirmation message will appear.

![C:\\Users\\siya\\Downloads\\MicrosoftTeams-image (74).png](media/image4.png){width="6.5in" height="4.8765879265091865in"}

- Select the Yes option.

- If you have assigned the department to any items, it will not allow you to delete the department and will appear the error message as shown in the below image.

![C:\\Users\\siya\\Downloads\\MicrosoftTeams-image (73).png](media/image5.png){width="6.5in" height="4.8765879265091865in"}
