# How to calculate No sales value_

**How to calculate No sales value?**

No sale is the total number of count when No sales is called from cash register.

No Sales option is available on the left side menu from Cash register module.

When user press No Sales button, it will ask you to open cash drawer. If you select **Yes** or **No** option, it will add the count to the total of No sales value. Drop count will be also added to No sales in the shift report.
