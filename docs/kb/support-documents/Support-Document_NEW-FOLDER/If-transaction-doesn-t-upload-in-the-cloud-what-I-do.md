# If transaction doesn’t upload in the  cloud what I do

**If the transaction doesn't upload to the cloud what do I do?**

If the transaction doesn't upload to the cloud then the might be system is not connected properly to the internet.

- Press the home button.

- If the home button is not available, On the Home screen of the iPad, or while in an app, swipe up from the bottom of the screen and pause for a moment while still pressing the screen, then let go when the App switcher appears.

- Go to setting.

- Select the Wi-Fi option.

- Select the network from the list.

- Enter password.

- The internet network will connect to your register or iPad.

- Now, open the Rapid RMS application.

- Select cloud button from top left side of the screen.

<!-- -->

- Click Upload button.
