# How to use shortcut in inventory_

**How to use the shortcut in inventory?**

Shortcuts in inventory management are for instant use. All these options are also available in item detail.

- Open rapid RMS application.

- Go to the inventory management module.

- In item click on shortcut icon.

![](media/image2.png){width="6.5in" height="4.569444444444445in"}

- Pop up open.

- From that select icon which you want to ON.

- Apply save.

- According to shortcut setting set, all will reflect in an item.
