# How to set memo for price override

**How to set memo for price override ?**

When user want to add memo pop up for price override then it can be set from BOF.

Follow below procedure to set memo for price override

- Log in with web URL.

- Go to setting tab.

- Select memo option.

- When memo opens then many of ON/OFF switch display for different options.

- Review there is one option for price override.

- Keep price override option OFF.

![](media/image5.png){width="6.5in" height="3.3472222222222223in"}

- Now open application and go to cash register.

- Ring up item and swipe ring up item for edit.

- When change price item then there is no pop up for memo.

![](media/image2.png){width="6.5in" height="4.791666666666667in"}

- Now from BOF kep price override option ON.

![](media/image6.png){width="6.5in" height="3.361111111111111in"}

- Restart rapid rms application.

- Go to cash register and ring up one item in bill.

- Swipe that item for edit.

- Try to change price.

- When price change then one pop up for memo would open.

![](media/image4.png){width="6.5in" height="4.708333333333333in"}

- Select proper reason for price change.

- Save text.

- Memo would save for that price change.

- Now op up for price change would open.

- Enter price you want to change.

- Click on enter.

![](media/image1.png){width="6.5in" height="4.638888888888889in"}

- Item new price will set.

![](media/image3.png){width="6.5in" height="4.833333333333333in"}
