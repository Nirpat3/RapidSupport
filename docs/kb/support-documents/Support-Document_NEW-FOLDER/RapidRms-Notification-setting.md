# RapidRms Notification setting

**[Rapid RMS Notification setting]{.underline}**

RapidRMS application version 2.5.41 and above version would support push notification functionality.

When you install RapidRMS from the app store or test flight and open the application then a pop-up for notification permission would display.

If you want to customize notification preferences, regarding the RapidRMS app directly then follow the below procedure to get a better experience of notification without any disturbance while operating RCR.

- Install RapidRMS application from test flight or app store.

- Launch RapidRMS from the iPad dashboard.

- At the time of launching the application, one pop-up for notification allowance permission would display.

- You need to hit "Allow" in the notification pop-up for all live updates and petro notifications.

![](media/image1.png){width="6.5in" height="4.388888888888889in"}

- If you select "Don't Allow" in the notification pop-up, then you will not have an automatic live update related to inventory and petro notification.

- To get the live update manually you have to select the SYNC button from the cash register or inventory management.

- We advise you to select ALLOW you to get a live update.

**For customize notification of Rapid RMS:**

- Go to setting-\> Notifications -\> Rapid RMS.
