# How to set memo for tax remove

**How to set memo for tax remove ?**

When user want to add memo pop up for tax remove then it can be set from BOF.

Follow below procedure to set memo for tax remove.

- Log in with web URL.

- Go to setting tab.

- Select memo option.

- When memo opens then many of ON/OFF switch display for different options.

- Review there is one option for tax remove.

- Keep tax remove option OFF from BOF.

![](media/image3.png){width="6.5in" height="3.819161198600175in"}

- Open rapid rms application.

- Go to cash register.

- Ring up any item and swipe item.

- Remove tax.

> ![](media/image7.png){width="6.5in" height="4.763888888888889in"}

- There is no pop up for memo at the time of remove tax.

- Now from side pan try to remove tax from item.

![](media/image6.png){width="6.5in" height="4.805555555555555in"}

![](media/image9.png){width="6.5in" height="4.777777777777778in"}

- There would no pop up for memo for tax removing.

- Now from BOF keep tax remove memo option ON.

![](media/image2.png){width="6.5in" height="3.3194444444444446in"}

- Restart application and go to cash register.

- Now ring up item and swipe item for edit.

- Click on cancel button of tax for remove tax from item.

![](media/image8.png){width="6.5in" height="4.791666666666667in"}

- When click on cancel button then popup for memo would open.

- Select tax or add text for remove tax.

- Click on save.

![](media/image4.png){width="6.5in" height="4.777777777777778in"}

- Memo text would save.

- Another option is, click on side pane and remove tax by tax remove button.

![](media/image1.png){width="6.5in" height="4.763888888888889in"}

- When remove tax then memo pop up for tax removing reason would display.

- Enter proper reason and apply save.

- Added memo text would save in tax remove.
