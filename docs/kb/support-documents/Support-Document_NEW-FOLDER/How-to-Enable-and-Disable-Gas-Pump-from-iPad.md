# How to Enable and Disable Gas Pump from iPad

**[How to Enable and Disable Gas Pump from iPad?]{.underline}**

- **Please follow the below steps:-**

1.  Open Rapid RMS application.

2.  Go to Cash Register module.

3.  In Right side, In Gas Pumps section, tap on any enable/disable pump.

![https://www.rapidrms.com/App/wp-content/themes/jupiter/rapidrmsDoc/images/application/other-settings/enable-and-disable-gas-pump-from-ipad-1.png](media/image1.png){width="6.5in" height="4.877031933508311in"}

4.  Now you will see one form with all details of selected pump.

5.  Now For further process Tap on 'Settings' option.

![https://www.rapidrms.com/App/wp-content/themes/jupiter/rapidrmsDoc/images/application/other-settings/enable-and-disable-gas-pump-from-ipad-2.png](media/image2.png){width="6.5in" height="4.877031933508311in"}

6.  Now you will see Enable and disable option.

7.  If a user wants to pump status enable to disable then tap on 'Disable' and for disable to enable tap on 'enable' option.

8.  After select enable/disable option tap on continue button.

![https://www.rapidrms.com/App/wp-content/themes/jupiter/rapidrmsDoc/images/application/other-settings/enable-and-disable-gas-pump-from-ipad-3.png](media/image3.png){width="6.5in" height="4.877031933508311in"}

9.  Now in the POS screen, Pump status will change according to new setting.
