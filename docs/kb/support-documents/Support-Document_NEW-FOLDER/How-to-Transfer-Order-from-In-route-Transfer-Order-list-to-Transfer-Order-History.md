# How to Transfer Order from In route Transfer Order list to Transfer Order History

**How to Transfer Order from 'In route Transfer Order list' to 'Transfer Order History'?**

In Route Transfer Order List module is used to view the order when in destination store order is in receiving pending section in PO module.

We can just view order in 'In Route transfer order list' module. Only destination store can be done order from receiving Screen.

Follow below steps to Reconcile Receiving qty of source store.

- Open Rapid RMS app.

- Active destination store.

- Go to Purchase order module.

- Now go to Receiving pending module.

![](media/image8.png){width="6.5in" height="4.736111111111111in"}

- In Receiving List, Check to receive an order which was Transferred from source store but still not reconciles from destination store.

![](media/image5.png){width="6.5in" height="4.722222222222222in"}

- Check Order in Receiving List of destination store and 'In-route order' of source store.

> -\> Destination Store -\>Purchase Order module -\> Receiving Pending -\> Receiving List.

![](media/image3.png){width="6.5in" height="4.722222222222222in"}

> -\> Source Store -\> Inventory transfer -\> In-route order -\> In Route transfer Order list.

![](media/image7.png){width="6.5in" height="4.791666666666667in"}

- Now from Receiving Screen of destination store Reconcile Order when destination store receive qty.

- For Reconcile order, tap on 'Reconcile' icon and tap on the 'Yes' option.

![](media/image2.png){width="6.5in" height="4.75in"}

- Now enter remark for delivered and tap on 'Save' button.

![](media/image1.png){width="6.5in" height="4.708333333333333in"}

- After Reconcile, Order will complete and order will move 'Transfer Order History' of source store and Order History of destination store.

> -\> Destination Store -\>Purchase Order module -\>Order History .

![](media/image6.png){width="6.5in" height="4.680555555555555in"}

> -\> Source Store -\> Inventory transfer -\> Transfer order history.

![](media/image4.png){width="6.5in" height="4.722222222222222in"}
