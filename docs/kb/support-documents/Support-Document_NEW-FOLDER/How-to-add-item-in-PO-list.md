# How to add item in PO list

**How to add item in PO order**

There are many ways to add items in PO list.

1)  From the particular module, the item can add directly add into PO list via 'Add to PO' option.

2)  From generating order module.

3)  From PO module.

<!-- -->

1.  **Item added via add to PO option in the item.**

- Go to module .i.e inventory management or cash register or Item movement.

- Inventory management.

- Go to the inventory management module.

- Open any item in edit mode.

- There is one option available named as 'Add to PO'.

![](media/image3.png){width="6.5in" height="4.2440212160979875in"}

- When clicking on Add to PO then that item will add to PO.

2.  **Item added to PO via inventory management**

- Go to the inventory management module.

- Hit the select button.

- Select items from the list which you want to add in PO.

- Apply to add to PO button which is available at the bottom bar.

![](media/image5.png){width="6.5in" height="4.652777777777778in"}

- Selected items will be added to the PO.

![](media/image2.png){width="6.5in" height="4.75in"}

3.  **Cash register.**

- Open cash register module.

- For every item, there is option 'ADD TO PO' for adding item directly in PO list.

![](media/image1.png){width="6.5in" height="4.527777777777778in"}

4.  **Item movement**

- Open item movement module.

- Add some item in the list.

- Select item(S) which you want to add in the purchase order.

- Apply to add to open order option.

- Selected items will added to PO list.

![](media/image4.png){width="6.5in" height="4.739153543307086in"}
