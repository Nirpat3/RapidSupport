# How to settle CC batch_

**[How to settle the CC batch?]{.underline}**

1.  Open rapid RMS application.

2.  Go to the daily report module.

3.  Click on the **CC batch** report option from the bottom bar.

4.  There is a **Settle Batch** button at the top right side corner of the screen.

5.  To settle the batch, click on the **Settle Batch** button.

![](media/image1.png){width="6.5in" height="4.75in"}

6.  It will prompt a confirmation message for settle batch.

7.  Select the **Yes** option.

![](media/image2.png){width="6.5in" height="4.708333333333333in"}

8.  It will appear the **Batch settlement** successful alert.

9.  If you have set up the print option is ON and the printer is connected to your Pos system, It will print the **CC batch receipt** accordingly.

10. If you have set up the printer option OFF, you will be able to print **CC batch receipts** through the print option.
