# How to add qsr sub group in modifier item

**How to add QSR sub group in modifier item?**

There are two ways to add sub QSR group into modifier item.

First way is, from QSR sub group master modifier item can be link and that way QSR sub group can be added in modifier item.

Another way is from modifier item sub group can be added.

Follow below procedure to add QSR sub group into modifier item.

- Open Rapid RMS application.

- Go to inventory management module.

- Open modifier items tab.

- Open any of modifier items in edit mode.

- Now scroll that item there is one field available named as QSR sub group.

- When create modifier item then initially it display blank.

![](media/image1.png){width="6.5in" height="4.75in"}

- To add QSR sub group click on arrow available in that field.

- All available QSR sub group in system will be listed.

- Select any of one radio buttons.

![](media/image2.png){width="6.5in" height="4.875in"}

- Hit save button, QSR sub group will be added to item.

- There is search bar available to search QSR sub group from list.

- Note that at a time only one QSR sub group can be added in one modifier item.

- Added QSR sub group will be display in modifier item.

![](media/image3.png){width="6.5in" height="4.722222222222222in"}

- Added QSR sub group can be remove by cancel button available behind QSR sub group name.
