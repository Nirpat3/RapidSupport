# How to set order status

**How to set order status?**

In the purchase order, we need to set the status of the item at the time of order creation as according to we want to order, return, free goods, swap etc.

- Go to rapid RMS application.

- Open purchase order module from the dashboard.

![](media/image1.png){width="6.5in" height="4.168109142607174in"}

- Click on PO list.

![](media/image2.png){width="6.5in" height="4.348917322834645in"}

- Add some items in PO list.

- In item detail, there is one column name as status.

- Default it would display order.

![](media/image3.png){width="6.5in" height="4.758218503937008in"}

- Click on order.

- One pop up window will open with different status.

- At a time only one status can set in a particular item.

- Set any of status and apply done.

- As according to the status set item would behave in order.

![](media/image4.png){width="6.5in" height="4.135977690288714in"}

- Order: Item would set for receive.

- Return: Item would set for a return.

- Free goods: Item would free in order.

Rather than PO list module order status can change from open order module.
