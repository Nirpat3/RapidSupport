# How to stop all pumps simultaneously_

**How to stop all pumps simultaneously ?**

In POS for fuel and all pumps there is one facility available and that is doing stop all pumps together doesn\'t matter weather condition of pump is working or in ideal mode.

To stop all pump follow below steps :

- Open rapid rms application.

- Configure pos with gas module.

- In RCR there is one button available in header with name 'stop all pumps'.

- If tap trice to that button then all available pumps will be stop together.

- Let suppose pumps are in working condition.

![](media/image2.png){width="6.5in" height="4.847222222222222in"}

- Now tap trice on button available at header with red color.

- All pumps will be stop simultaneously and display as disable.

![](media/image4.png){width="6.5in" height="4.930555555555555in"}

- After few second all pumps will be in ideal condition.

![](media/image1.png){width="6.5in" height="4.791666666666667in"}

- After stop all pumps again in ideal condition state will depend on configuration from setting module.

![](media/image3.png){width="6.5in" height="4.875in"}

- If auto enable option is selected that means after emergency stop all pumps all pumps would get into idealize condition automatically.

- If manually enable option is selected then after emergency stop all pumps they will not automatically enable, manually need to enable all pumps.

<!-- -->

- Thus by this way all pumps can be stop if there is any emergency condition.
