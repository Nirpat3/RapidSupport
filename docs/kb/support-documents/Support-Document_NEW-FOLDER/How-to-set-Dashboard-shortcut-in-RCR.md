# How to set Dashboard shortcut in RCR

**How to set the Dashboard shortcut in RCR?**

When want to go instantly from a cash register in any other module then the dashboard shortcut is there.

- Go to a setting module.

- Open "options".

- There are module shortcut settings.

- Either Keep all options ON or keep option ON which you want to use as a shortcut in RCR.

![](media/image1.png){width="6.5in" height="4.736111111111111in"}

- Now go to the cash register.

- In the bottom bar left side there is one icon available.

- Click on that it will show all set module shortcuts.

- By selecting that item you can directly move to that particular module.
