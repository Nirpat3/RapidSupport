# How sound ON OFF from setting in application

**How sound ON/OFF from setting for rapid?**

In rapid RMS application on every beat or tap if you want to set a beep sound then you can set it from setting module.

- Go to the setting module.

- Open sound.

- Keep sound switch ON.

- Set from list of sound.

- On every tap selected sound will beep.
