# How to see history in clock in out_

**How to see history in clock in-out?**

In clock in-out, if you want to see the history of all entries then you can see all in the history page.

When Reset screen then all the entries will move to history.

- Open rapid RMS application.

- Go to clock in-out module.

- Open history section.

![](media/image3.png){width="6.5in" height="4.708333333333333in"}

- History can be filtered according to time duration. From date range and time menu.
