# How to connect customer display_

**How to connect customer display?**

For customer display its need to set in two devices. In one device store need to configure and another device customer display need to configure.

- Download customer display app.

- Configure it with store credential.

- In both, the iPad store must be the same.

- And also both store wifi needs to be same.

- Open rapid RMS application.

- Go to the setting module.

- Select customer display.

![](media/image4.png){width="6.5in" height="4.736111111111111in"}

- Devices in which customer display is available are displayed in the list.

- Select device.

- In another device when not connect then below situation display.

![](media/image3.png){width="6.5in" height="4.861111111111111in"}

- After connection establishes a screen display with a green line.

![](media/image5.png){width="6.5in" height="4.888888888888889in"}

- Now go to cash register and start the transaction.

- All the transaction would display in the cash register.
