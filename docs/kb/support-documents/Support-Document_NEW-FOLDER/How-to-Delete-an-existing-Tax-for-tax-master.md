# How to Delete an existing Tax for tax master

**[How to Delete an existing Tax for tax master?]{.underline}**

- **It is highly recommended to check below steps before deleting Tax.**

1.  Please make sure that you have not assigned the tax to any **item or department**.

2.  If you have assigned tax, Please make sure that you have removed the tax from item or Department.

- Open RAPID RMS application in IOS.

- Select inventory management from Dashboard.

![](media/image1.png){width="6.5in" height="3.3834022309711287in"}

- Click on three line option from the top left side corner of the screen.

![](media/image2.png){width="6.5in" height="3.738610017497813in"}

- It will appear the list of existing taxes.

![](media/image3.png){width="6.5in" height="4.047619203849519in"}

- Select tax from the tax list.

<!-- -->

- It will open tax in edit mode.

![C:\\Users\\siya\\Downloads\\MicrosoftTeams-image (66).png](media/image4.png){width="6.5in" height="4.87373031496063in"}

- At the bottom, click on the delete button.

- It will appear a confirmation message to delete a tax.

- Select the Yes option.

- Click on the Save button.

- It will move to the list page and the Deleted tax will not appear in the list page*.*

- If you have assigned the tax to any item or department, it will not allow you to delete the tax and will appear an error message.
