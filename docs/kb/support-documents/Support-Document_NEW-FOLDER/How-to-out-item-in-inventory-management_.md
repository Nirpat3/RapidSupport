# How to out item in inventory management_

**How to out item in inventory management?**

If you want to add an item in open order for recording outgoing items then you can add via "OUT".

- Open rapid RMS application.

- Go to the inventory management module.

- Select OUT section.

![](media/image1.png){width="6.5in" height="4.736111111111111in"}

- In section display with clear all, search, hold save, info and export option.

- If you want to add item into in section then there two-three ways.

- From barcode scanning.

- From search option already available item you can add.

![](media/image2.png){width="6.5in" height="4.708333333333333in"}

![](media/image3.png){width="6.5in" height="4.777777777777778in"}

- After adding an item you can enter a quantity.

- After adding item quantity you can put an invoice on hold or can save that invoice.

![](media/image4.png){width="6.5in" height="4.708333333333333in"}

- When putting an item on hold or if save invoice then one pop up display.

- Enter text and save.

![](media/image5.png){width="3.2604166666666665in" height="2.4270833333333335in"}

- When you hold invoice then it will move into open order.

- When you save then invoice goes directly into close order.
