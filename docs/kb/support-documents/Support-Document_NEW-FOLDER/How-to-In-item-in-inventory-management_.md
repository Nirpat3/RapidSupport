# How to In item in inventory management_

**How to In item in inventory management?**

If you want to add item in open order for tracking received item then you can add via "In".

- Open rapid RMS application.

- Go to the inventory management module.

- Select IN section.

![](media/image5.png){width="6.5in" height="4.708333333333333in"}

- In section display with add, search, hold save, info and export option.

- If you want to add item into in section then there are three ways.

- From barcode scanning.

- From add option, you can add a new item.

- From search option already available item you can add.

![](media/image4.png){width="6.5in" height="4.736111111111111in"}

![](media/image6.png){width="6.5in" height="4.694444444444445in"}

- After adding an item you can enter quantity which you have received.

![](media/image3.png){width="6.5in" height="4.722222222222222in"}

- After adding item quantity you can put an invoice on hold or can save that invoice.

![](media/image1.png){width="6.5in" height="4.777777777777778in"}

- When put an item on hold or if save invoice then one pop up display.

- Enter text and save.

![](media/image2.png){width="3.2291666666666665in" height="2.3854166666666665in"}

- When you hold invoice then it will move into open order.

- When you save then invoice goes directly into close order.
