# How to review centralized Z from manager report

**How to review centralized Z from a manager report?**

In the daily report, centralized Z is the reporting section for all applied Z.

In a centralized Z report, entries are displayed according to all z applied.

Day wise one entry is a display and in that one entry, all the Z which are applied would display.

- Open rapid RMS application.

- Go to the manager report.

- Open centralized Z.

- All the Z report entry display day-wise when the last Z applied. In one entry all the z applied display according to register wise.

![](media/image4.png){width="6.5in" height="4.708333333333333in"}

- If you want to view previous month Z history then you can do by date filter.

- In centralize Z it will not filter by date its only filter by month.

![](media/image5.png){width="6.5in" height="4.736111111111111in"}

- In centralized z you can view register wise history.

- Tap on any z report.

- There is a two-section display.

- History and custom.

![](media/image1.png){width="6.5in" height="4.736111111111111in"}

- In history, all viewed history displays according to the register.

- In custom, you can select a different register and can view the history of applied z by that register.

- Select custom.

- It will display all the registers by which z applied.

- Select some registers.

- Apply done.

- Combined register total data will display and that history will be added to the viewed history list.

![](media/image3.png){width="6.5in" height="4.680555555555555in"}

![](media/image2.png){width="6.5in" height="4.777777777777778in"}

- And in the z report selected register combined z report data display.
