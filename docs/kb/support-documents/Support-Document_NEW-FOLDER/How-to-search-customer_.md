# How to search customer_

**How to search a customer ?**

In customer list, if you want to search customer then it can be possible by name, contact no etc.

- Open rapid RMS application.

- Go to the customer loyalty module.

![](media/image4.png){width="6.5in" height="4.763888888888889in"}

- Customer list will display with text customer name, contact #, email, customer #.

- You can search by any of that detail.

![](media/image1.png){width="6.5in" height="4.777777777777778in"}

![](media/image3.png){width="6.5in" height="4.708333333333333in"}

- But especially if you want to search by customer \# and customer loyalty \# then there is one switch.

- Keep that switch ON.

- You can search by customer \# and customer loyalty #.

![](media/image2.png){width="6.5in" height="4.736111111111111in"}
