# Customer display not working still loading page after uninstalling and reinstall

**Customer display not working still loading page after uninstalling and re-install**

In this situation, it could have happened below these three categories: -

- It could be network issue. Check your network and make sure display iPad and POS iPad in same network.

- It could be Bluetooth off.

- It could be server issue.
