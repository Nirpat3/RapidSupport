# How to add modules in BOS

**[How to add modules in BOS?]{.underline}**

- Log in to the Back office with your credentials.

- Now go to settings \> modules. You may need to scroll down to select the settings option.

![](media/image1.png){width="6.5in" height="3.0165518372703413in"}

- It will appear the module list.

- Now select ON/OFF radio button to turn on or turn off the modules from Back office.

- Select the module as per the given instruction from the USA team.

![](media/image2.png){width="6.5in" height="3.304630358705162in"}
