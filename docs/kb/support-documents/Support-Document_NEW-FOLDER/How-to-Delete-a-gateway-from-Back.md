# How to Delete a gateway from Back

**[How to Delete a gateway from Back-office?]{.underline}**

- **Please follow the below steps:-**

1.  Login to BOS with your user credentials.

2.  Tap menu ![hamburger-menu-icon-png-white-24.jpg](media/image1.png){width="0.2075404636920385in" height="0.2075404636920385in"} (located on the top right side)

3.  Tap settings.

4.  Tap store.

5.  Scroll to see the Gateway option. Select the Gateway option.

![](media/image2.png){width="6.5in" height="2.9762259405074367in"}

6.  To delete the gateway, click on the **action** button and select the **Delete** option.

![](media/image3.png){width="6.5in" height="2.9767989938757657in"}

7.  It will prompt a **confirmation message** to delete the payment gateway.

![](media/image4.png){width="6.5in" height="2.981560586176728in"}

8.  Select the **Yes** option.

9.  The payment gateway will be deleted and successful delete message will appear.
