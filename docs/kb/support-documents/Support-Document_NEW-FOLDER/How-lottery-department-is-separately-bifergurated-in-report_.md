# How lottery  department is separately bifergurated in report_

**How lottery department is separately bifergurated in report ?**

In Rapid RMS there are different kind of departments are available with their own characteristics.

In report all departments are separately display with sales value, Return value, discount value and total value.

If item with lottery department then that sales value is consider in sales section of lottery department.
