# How to do post pay transaction_

**How to do post pay transaction ?**

In pump there are two types of transactions, pre-pay and post pay transaction.

In post pay transaction first fueling is completed then after payment is done as according to filling fule.

Follow below steps to do post pay gas transaction.

- Set up fusion.

- Go to pump and take up nozzle at any pump.

- Start fueling.

<!-- -->

- When nozzle is up then in POS requesting display for selecting particular pump.

> ![](media/image3.png){width="6.5in" height="4.819444444444445in"}

- Accept request from POS.

- Now start fueling.

- Dispense fule for sometime.

- Now get down nozzle.

- stop fueling.

- Now in RCR click on that pump. Pump window will open.

- One transaction is listed with fuel amount and volume. Heading of that transaction would be 'post-pay'.

- To complete fuel payment select that transaction.

- Hit continue button.

![](media/image2.png){width="6.5in" height="4.888888888888889in"}

- When continue that fueling, payment ring up in bill screen as an item with name post-pay with volume, fuel type, fuel price and service type.

- Click on grand total.

![](media/image5.png){width="6.5in" height="4.722222222222222in"}

- Complete transaction by any convenient payment type.

![](media/image6.png){width="6.5in" height="4.75in"}

- Open that pump after completing transaction. Transaction will disappear from list.

![](media/image4.png){width="6.5in" height="4.777777777777778in"}

- That transaction would listed in transaction tab history.

![](media/image1.png){width="6.5in" height="4.763888888888889in"}
