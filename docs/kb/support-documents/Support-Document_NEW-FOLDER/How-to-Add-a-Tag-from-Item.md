# How to Add a Tag from Item

**[How to Add a Tag from Item?]{.underline}**

You can specify products in different tags as per your requirement in an item.

**[Here are the steps to add Tags to an item:-.]{.underline}**

- Open the RapidRMS application on your iPad.

- Select Inventory Management from the dashboard.

- Enter your Quick access password.

- Click on Add button to create a new Item from the bottom of the page or click on any item from the list to edit the existing Item.

![](media/image1.png){width="6.5in" height="4.680555555555555in"}

- Add/Edit Item screen will open.

- At the top of the screen, there are Info, Pricing, Discount, History, and Options tabs available.

- The Tag section is available under the Info tab.

![](media/image2.png){width="6.5in" height="4.722222222222222in"}

- Click on the tax field of the tag. Tag dropdown will appear on the right side of the panel.

- Select a tag from the list or enter a new tag in a text field.

- To add a tag click on the ![C:\\Users\\siya\\AppData\\Local\\Microsoft\\Windows\\INetCache\\IE\\A7HR0A3N\\redirected-150nw-1182903607\[1\].jpg](media/image3.jpeg){width="0.21736001749781278in" height="0.19565944881889763in"} option.

- An added tag will appear below the tag field. You can add one or more tags for the same item.

![](media/image4.png){width="6.5in" height="4.777777777777778in"}

- Click on the![C:\\Users\\siya\\AppData\\Local\\Microsoft\\Windows\\INetCache\\IE\\EHUWMOOS\\button-24806_640\[1\].png](media/image5.png){width="0.293494094488189in" height="0.26307852143482063in"} cross button to remove the added tag.

![](media/image6.png){width="6.5in" height="4.763888888888889in"}

- Click on the **Save** button to apply changes in the Item.
