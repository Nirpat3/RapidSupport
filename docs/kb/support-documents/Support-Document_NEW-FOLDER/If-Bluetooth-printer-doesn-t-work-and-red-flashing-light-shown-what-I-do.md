# If Bluetooth printer doesn’t work and red flashing light shown what I do

**If the Bluetooth printer doesn't work and red flashing light showed what I do?**

It could be printer roll problem maybe you didn't set up properly printer roll. If printer roll is correct, and still you will see a red flashing light or printer receipt will not print Move to below step.

- Check if the printer is connected to the iPad via Bluetooth

  - Open setting on the iPad, click Bluetooth and find the printer you would like to connect. If your printer is not listed then click the printer you wish to connect.

> If still printer is not connected then follow below step.

- Power on the printer and hold down the pair red button located on the back of the printer near the power/cash drawer cable for 5 seconds.

- Open setting on the iPad, click Bluetooth and find the printer.

- Click the printer to pair/connect.

- it might be that you have not enabled receipts to print for the customer's payment method. How to enable: -

  - Go to dashboard setting at the bottom screen

  - On the right, click on tender configuration then check in cash, credit, debit, gift card which is below tender type that is that print option enable or not. If it is not then you will have to enable.

- Once you have enabled receipt to print for the tender, return to the cash register module. Click the last invoice in the bottom left corner.

- A popup will appear saying "do you want to print last receipt" click yes

- A receipt will print for the transaction.
