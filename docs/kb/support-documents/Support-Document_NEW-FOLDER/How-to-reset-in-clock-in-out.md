# How to reset in clock in out

**How to reset in clock in-out?**

When the user wants to clear the screen then there is a reset button in click in-out module.

- Open rapid RMS application.

- Go to the clock in out module.

- In clock in out, there are all entries display.

- Apply the reset option.

![](media/image2.png){width="6.5in" height="4.833333333333333in"}

- Confirmation for a reset will display.

![](media/image1.png){width="6.5in" height="4.722222222222222in"}

- The screen will clear and all records move to the history section.
