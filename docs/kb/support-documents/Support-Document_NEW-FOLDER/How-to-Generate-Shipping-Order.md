# How to Generate Shipping Order

**How to Generate Shipping Order?**

Inventory Transfer is used when a store wants to transfer items from one store to another store.

Generate shipping order means when the source store wants to generate shipping order to send items to the destination store.

There are 2 ways to Generate and create transfer/shipping order.

1)  Follow the below steps to Generate transfer order from Generate Order section.

- Open rapid RMS application.

- Open Inventory Transfer module.

- Open Generate Order category.

- Now from Generate Order page, select fields for item selection.

> Ex: Select Date: Daily, Department: Beer.

![](media/image3.png){width="6.5in" height="4.680555555555555in"}

- Now Tap on 'Create Open Order' button.

![](media/image11.png){width="6.5in" height="4.786458880139983in"}

- Now enter the open order title and tap on save button.

- Now Store list will display which linked with this store from Bof side.

![](media/image9.png){width="6.5in" height="3.276042213473316in"}

- From the Store list, select that store from where this store wants to send items.

- Click on done button.

![](media/image1.png){width="6.5in" height="4.666666666666667in"}

- Now Order will Generate Under Inventory Transfer -\> Transfer Order Items module and Po -\> Open Order list.

> -\> Inventory Transfer -\> Open Transfer Order -\> Transfer Order List page -\> Transfer Order Items.

![](media/image5.png){width="6.5in" height="4.765625546806649in"}

-\> Purchase Order -\> Open Order -\> Open Order List -\> Open Order Items.

![](media/image4.png){width="6.5in" height="4.847222222222222in"}

2)  Follow below steps to Create transfer order from Open Transfer order.

- Open Rapid RMS app.

- Go to Open Transfer Order from Po module.

- In the top right side, tap on + Icon.

- Now tap on store selection option.

![](media/image8.png){width="6.5in" height="4.708333333333333in"}

- Enter the Open Order Title and tap on the Save button.

![](media/image6.png){width="6.5in" height="4.638888888888889in"}

- Now select store and click on the done button.

![](media/image7.png){width="6.5in" height="4.638888888888889in"}

- Now transfer order list will create in inventory transfer of source store and one open order will create in Po of destination store.

-\> Inventory Transfer -\> Open Transfer Order -\>Transfer Order List.

![](media/image2.png){width="6.5in" height="4.708333333333333in"}

-\> Purchase Order -\> Open Order -\> Open Order List.

![](media/image10.png){width="6.5in" height="4.736111111111111in"}
