# How to Clock in_

**How to Clock in?**

In-store when there are different users need to work then all users work according to shift.

For shift record in rapid RMS, there is one module clock in out.

For start, a new shift or new record follow below procedure.

- Open rapid RMS application.

- Go to clock in-out module.

![](media/image4.png){width="6.5in" height="4.791666666666667in"}

- Initially, the screen is blank for new user when starting a new shift.

- After start shift will display in the list.

- Click on clock in.

- New clock in will start.

![](media/image3.png){width="6.5in" height="4.694444444444445in"}

- The new clock will display in the list with started time.

![](media/image2.png){width="6.5in" height="4.722222222222222in"}

![](media/image1.png){width="6.5in" height="4.736111111111111in"}

- Entry display with day/date, user name, clock in time, clock out time, and total hours spent.
