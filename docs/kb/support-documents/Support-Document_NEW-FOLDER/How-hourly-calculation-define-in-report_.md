# How hourly calculation define in report_

**How hourly calculations define in report?**

In report there is one section available for hour wise report. Total sales and customer count broken by hourly.

In this three column available

Hours: Display time with one hour difference.

Price: Total value (sales-return) of selling within that hour. Sum of all invoices within that hour.

Count: Display per hour sales count. \# of trnx based on invoices, NOT line item
