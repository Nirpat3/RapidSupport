# How to add modifier item into system

**How to add modifier item into system?**

To add modifier items follow below procedure.

- Open inventory management module from IPAD.

- There is one tab available named as a modifier item. If from BOF modifier flag ON then only the modifier tab will display.

- When tab on modifier then all modifier list display.

![](media/image1.png){width="6.5in" height="4.75in"}

- To add new modifier click on Add option.

- When open page for modifier then it will display page of new modifier item creation.

- Modifier item configuration is almost similar to item. Just differ thing is need to set QSR group, QSR sub group and QSR operation.

![](media/image2.png){width="6.5in" height="4.763888888888889in"}

- Add all detail of modifier should display.

- Info, pricing, discount, history and options section of modifier are same as item.

- When create modifier item then in option section "modifier item" option display ON.

![](media/image3.png){width="6.5in" height="4.791666666666667in"}

- Click on save option.

- New modifier item will be listed in modifier items list.
