# How to set up drawer setting from RapidRMS

**[How to set up drawer setting from RapidRMS?]{.underline}**

**[Please follow the below steps:-]{.underline}**

1.  Open rapid RMS application.

2.  Click on the settings option ![C:\\Users\\mit shah\\AppData\\Local\\Microsoft\\Windows\\INetCache\\IE\\BU8Y79T3\\WMF-Agora-Settings_BCBCBC.svg\[1\].png](media/image1.png){width="0.17463910761154855in" height="0.17463910761154855in"} available at the bottom of the DashBoard.

![C:\\Users\\mit shah\\Downloads\\MicrosoftTeams-image (8).png](media/image2.jpeg){width="6.2625in" height="3.6521741032370953in"}

3.  Select the Tender configuration option.

4.  Open any tender from the list.

5.  The **Open cash drawer** toggle button by default is set **Off**. If the **Open cash drawer** toggle button is set **Off**, you will not be able to open the Cash drawer from any options as shown in the below image.

6.  Turn on the **Open cash drawer** toggle button.

![](media/image3.png){width="6.500905511811023in" height="4.0in"}

![](media/image4.png){width="6.500905511811023in" height="4.0in"}

7.  When you have turned on the Cash drawer toggle button from any tender, it will open when you complete any transaction or when you click on NO sale or Drop amount from the Cash register.

![](media/image5.png){width="6.5in" height="4.722222222222222in"}

8.  If your drawer is connected to your system, It will open after the completion of the transaction automatically.

![](media/image6.png){width="6.5in" height="4.819444444444445in"}

9.  If you have turned on the Open Cash drawer option but your drawer is not connected to the system. It will prompt an alert message as shown in the below image.

![](media/image7.png){width="6.5in" height="4.763888888888889in"}
