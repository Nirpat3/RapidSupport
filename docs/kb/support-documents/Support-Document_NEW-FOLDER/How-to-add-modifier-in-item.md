# How to add modifier in item

**How to add modifier in item?**

In normal item modifier can be added.

For adding modifier into item follow below procedure.

- Open Rapid RMS application.

- Go to inventory management module.

- Open any item in edit mode.

- When open normal item in edit mode then there is one tab available in item named as 'Modifier'.

- When open modifier then below screen will display.

![](media/image1.png){width="6.5in" height="4.569444444444445in"}

- If item modifiers are available then it will display in page. Initially it display blank.

- If modifiers are not available then can be added by add button.

- Click on ADD button.

![](media/image2.png){width="6.5in" height="4.777777777777778in"}

- When click on add button and if there is no data available in qsr group and qsr sub group then recommended and qsr group tab it display blank.

- In all modifier tabs, all available modifiers will display with operation.

![](media/image3.png){width="6.5in" height="4.722222222222222in"}

- Select modifier from list.

- At a time multiple modifier selection can possible.

- After selected modifier click on save.

- Selected modifiers are listed.

- Now apply save option.

- Selected modifiers added in item.

![](media/image4.png){width="6.5in" height="4.652777777777778in"}

- From search bar selected modifier can search.

- As modifiers are added according to that qsr group and sub groups are also added into item.

![](media/image5.png){width="6.5in" height="4.75in"}
