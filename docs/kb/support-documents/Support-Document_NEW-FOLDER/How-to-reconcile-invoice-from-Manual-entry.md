# How to reconcile invoice from Manual entry

**[How to reconcile the Invoice?]{.underline}**

There are two ways to reconcile invoices:-

1)  Reconcile new Invoice from the new option.

2)  Reconcile an existing Invoice from the Recall option.

- **Reconcile new Invoice from the new option**

1.  Open the Manual entry module from DashBoard.

2.  Enter Quick access password

3.  Select the New option from Manual entry.

![](media/image1.png){width="6.5in" height="4.823849518810149in"}

4.  It will prompt you to receive the info window.

5.  Enter Receiving Date, select vendor, and invoice number.

6.  Click on the search button from the bottom bar.

7.  Select the item and click on add button.

8.  After entering the received quantity and updating stock in inventory, you need to reconcile the order.

9.  Click on reconcile button from the bottom bar.

10. It will prompt a confirmation message.

11. Select the Yes option.

- **Reconcile an existing Invoice from the Recall option**

1.  Open the Manual entry module from DashBoard.

2.  Enter Quick access password

3.  Select the Recall option from Manual entry.

![](media/image2.png){width="6.5in" height="4.375022965879265in"}

4.  All hold invoices will appear in the recall list.

![](media/image3.png){width="6.5in" height="4.21875in"}

5.  Select an invoice from the list.

6.  Update quantity, cost, or retail price of an item.

7.  Click on reconcile button from the bottom bar.

8.  Select the Yes option.
