# How to edit customer_

**How to edit customer?**

In rapid RMS under customer loyalty module existing customer list available. If you want to edit any customer details then it can be possible by customer loyalty module.

- Open rapid RMS application.

- Go to the customer loyalty module.

![](media/image1.png){width="6.5in" height="4.763888888888889in"}

- All existing customers listed in customers page.

- Open that customer from view option whom detail you want to edit.

![](media/image2.png){width="6.5in" height="4.736111111111111in"}

- The customer detail page will open.

- Click on edit profile.

![](media/image4.png){width="6.5in" height="4.763888888888889in"}

- Change detail and click on save. New edited detail remains to save in customer.

![](media/image3.png){width="6.5in" height="4.763888888888889in"}
