# How to print report from manager report

**How to print a report from the manager report?**

**Print**

- If you want to print any of manager report then it can be possible by print option.

- For printing, the printer must be connected with the system.

- Open any of report or tap on a report which you want to print.

- Hit the print button.

- The selected report will print.
