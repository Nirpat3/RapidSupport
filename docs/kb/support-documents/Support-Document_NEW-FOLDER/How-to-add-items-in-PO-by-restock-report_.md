# How to add items in PO by restock report_

**How to add items in the PO by restock report?**

Restock report is the module to complete and check the stock of selling items.

All items which are sold from the store will display in sales history.

For add item(S) in PO, the list follows below procedure.

- Open rapid RMS application.

- Go to restock report module.

- Open sales history.

- Select item from list which you want to add to Po.

- Apply option send to PO.

![](media/image1.png){width="6.5in" height="4.805555555555555in"}

- Selected items will add to purchase order.
