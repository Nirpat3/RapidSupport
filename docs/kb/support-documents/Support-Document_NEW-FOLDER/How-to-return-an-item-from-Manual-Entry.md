# How to return an item from Manual Entry

**[How to return an inventory from Manual Entry?]{.underline}**

1.  Open the Manual entry module from DashBoard.

2.  Enter the Quick access password.

After you create [new invoices](file:///C:\Users\siya\Downloads\!qhlogs.doc) or [recall a hold invoice](file:///C:\Users\siya\Downloads\!qhlogs.doc) from manual entry, you will need to click on the new item button from the bottom bar to create a new item. Review these articles to learn how to create a new invoice and recall hold invoice from manual entry.

<!-- -->

- After you [add a new item from manual entry.](How%20to%20add%20an%20item%20from%20manual%20entry.docx)

- Click on an item that you want to return.

- It will open the **Received** tab.

![](media/image1.png){width="6.500905511811023in" height="4.434783464566929in"}

- To return an item, click on the **Return** button from the top right side corner of the screen.

![](media/image2.png){width="6.5in" height="4.7947364391951in"}

- Now enter a quantity in the QTY OH field you want to return.
