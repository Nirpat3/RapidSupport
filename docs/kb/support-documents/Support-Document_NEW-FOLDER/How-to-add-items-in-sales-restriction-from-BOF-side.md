# How to add items in sales restriction from BOF side

**How to add items in sales restriction from BOF side?**

When in Rapid RMS if in particular time duration if you want to restrict item selling then it can be possible by blue law functionality from BOF.

For putting any item(S) under sales restriction follow below procedure.

- Login with back office web URL.

- Make sure e commerce option should ON in store.

- Go to blue law tab.

- Open any one entry in edit mode by action button.

![](media/image1.png){width="6.5in" height="4.210458223972004in"}

- Add all details like description, country, state, date range etc.

- There is one tab name as items.

- Click on that items tab.

![](media/image2.png){width="6.5in" height="3.8281255468066493in"}

- When click on item tab.

- When open item tab then it display blank with following page.

![](media/image3.png){width="6.5in" height="3.6321511373578304in"}

- If any items are already added then it will display in list.

- For adding items in it click on add button.

- When click on add button then all items which are available in store will display.

![](media/image4.png){width="6.5in" height="4.041023622047244in"}

- Select items from list and click on add items option.

- There is one search button available to search items from list.

- Also there is one select all option to select all items together.

- Filter option is also there to filter items.

![](media/image5.png){width="6.5in" height="3.7864588801399823in"}

- Select items which you want to put under sales restriction and click on add items.

- All items will be added under sales restriction.

![](media/image6.png){width="6.5in" height="3.0972222222222223in"}
