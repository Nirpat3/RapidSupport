# How to print and email x report

**[How to Print and Email X Report?]{.underline}**

When the user wants to print x report then it can be printed from the daily report.

**Print x report**

- Open rapid RMS application.

- Go to the Daily report module.

![](media/image1.png){width="6.5in" height="4.75in"}

- Open x report.

<!-- -->

- For printing x report, the printer must be connected with the application.

- Click on the print button which is available at the bottom bar.

![](media/image2.png){width="6.5in" height="4.777777777777778in"}

- You will have a hard copy of the x report available.

**Email x report**

You can preview or email x report.

- Click on PDF/Email option.

![](media/image3.png){width="6.5in" height="4.75in"}

- When applying preview than a preview of x report display.

- And when enter email address then the email of x report will be sent to regarding email address.
