# How to ring up modifier item in RCR

**How to ring up modifier item in RCR ?**

When in store items are as a modifier then also they can be sell as normal item.

To sell modifier as a normal item follow below procedure.

- Open cash register.

- From RCR open item list.

- There is one tab available name as modifier item

- In this tab all available modifier list will display.

- Select single/multiple modifier from list.

![](media/image2.png){width="6.5in" height="4.680555555555555in"}

- Modifiers will ring up in bill screen same as normal item.

![](media/image1.png){width="6.5in" height="4.805555555555555in"}

- All procedure which we can do with normal item can be perform with modifiers item also.

- Now complete payment process.

- Thus by this way modifier items can be sold from RCR.
