# How to ring up item in bill_

**How to ring up the item in the bill?**

In bill ring, up the screen all items which need to sale ring up from different places via department list, favourites and item list etc.

In Dashboard, Select Cash Register module and enter 4 digits quick access.

In POS, screen user can ring up items from a different way.

1.  Department List -\> Select items from department list.

2.  Select items from the favourites list.

3.  To ring up an item, you can scan the item with your scanner.

4.  Ring up an item from the item list.

5.  In the ring up screen click on item list icon.

6.  Select those items which you want to ring up.

- Click on Done button.

- Now popup appears for package type (Single Item, Case, Pack ).

- Select any one type and click on the Save button.

- Observe, selected items appear in ring up the screen.
