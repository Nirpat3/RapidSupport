# How merchandise and non merchandise department differentiated in report_

**How merchandise and non merchandise department differentiated in report?**

In department, there are two types of departments available in rapid.

![](media/image1.png){width="6.5in" height="4.527777777777778in"}

In shift report this kind of bifurcation is not available only in X report, Z history, ZZ history etc.

If department type is merchandise and that belonging items are sale then that all items price are consider in merchandise department section.

![](media/image2.png){width="6.5in" height="4.75in"}

In merchandise department section there are four columns available first is department name, second column is for price value of that department selling third column is define % value of that department selling and forth is for number of count of that department selling.

Department name - Name of that department by which items are sold.

Price - Total sales and refund value of that department selling.

Percentage - It display % amount of total department selling value.

Count - Count is number that how many times selling is form the particular department type.

Other departments which are not part of merchandise are considered in non merchandise and other are payout type departments.

For that both type of departments same kind of bifurcation is there as explained above.
