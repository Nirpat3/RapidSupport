# How to Add a group from Item

**[How to Add a group from Item?]{.underline}**

You can specify products in the different groups as per your requirement.

**[Here are the steps to add groups to an item:-]{.underline}**

- Open the RapidRMS application on your iPad.

- Select Inventory Management from the dashboard.

- Enter your quick access password.

- Click on add button to create New Item from the bottom of the page or click on any item from the list to edit the existing Item.

![](media/image1.png){width="6.5in" height="4.680555555555555in"}

- Add/Edit Item screen will open.

<!-- -->

- At the top of the screen, there are Info, Pricing, Discount, History, and Options tabs available.

- The Group section is available under the Info tab.

![](media/image2.png){width="6.5in" height="4.805555555555555in"}

- To add a group click on the \> option.

- All available groups will be listed on the screen.

- Select from the list and apply the done option.

- The selected group will be added to an item.

- If you want to create a new group then click on add button from the bottom of the screen.

![](media/image3.png){width="6.5in" height="4.666666666666667in"}

- Now screen for adding a new group will open.

- Enter a group name and apply to save.

![](media/image4.png){width="3.1979166666666665in" height="2.3645833333333335in"}

- Group will be added to the item.

- Apply to save in item. The added group remains to save in the item.

![](media/image5.png){width="6.5in" height="4.666666666666667in"}
