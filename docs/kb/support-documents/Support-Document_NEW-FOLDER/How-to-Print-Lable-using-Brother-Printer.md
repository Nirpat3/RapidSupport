# How to Print Lable using Brother Printer

- **How to Print Label using Brother Printer?**

> **Please follow the below Steps:-**

1.  Open Rapid Dashboard page on the Rapid App.

2.  Click on "**SETTING**" button.

![](media/image1.jpeg){width="6.495944881889764in" height="3.549867672790901in"}

3.  Enter your PIN.

![](media/image2.jpeg){width="6.5in" height="2.8958333333333335in"}

4.  From left side menu, select "**PRINTER**" option.

5.  Select "**Brother Printer**".

6.  The Brother Printer list will be there which are available in your Bluetooth frequency.

![](media/image3.png){width="6.5in" height="4.91162510936133in"}

7.  Now select your brother printer with Label size and tap on DONE button.

![](media/image4.png){width="6.5in" height="4.903915135608049in"}

- **If the scan fails to detect printers on the network. Select "MANUALLY" option on the top right side corner.**

![](media/image5.jpeg){width="6.5in" height="4.875in"}

1.  Enter the Label Printer IP.

![](media/image6.jpeg){width="6.5in" height="4.875in"}

**(NOTE: - Refer to the Label printer manual to identify the IP of the Printer, [OR]{.underline} download the brother label app and follow instructions on the app)**

2.  Select the model of the label printer.

3.  Select the label printer size. It is important that you select the correct label print size to avoid error messages.

> **(NOTE: - Please select Label Size: 2.4" x 1.1" or 2.4)**
