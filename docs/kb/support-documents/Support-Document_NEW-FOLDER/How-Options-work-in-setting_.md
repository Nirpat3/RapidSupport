# How Options work in setting_

**How to work Options in the setting ?**

Options is for various kind of setting.

Open rapid RMS application.

Go to the setting module.

Open "Options".

> ![](media/image1.png){width="6.5in" height="4.777777777777778in"}

- NO sales receipt is for print setting at the time of applying no sales. This will affect in RCR.

- When DOB required option ON at that time in RCR in age verification pop up it will not ask for an age like 18 or 21. If option Off then it will ask for age verification.

- All module setting option effect in RCR dashboard shortcut.
