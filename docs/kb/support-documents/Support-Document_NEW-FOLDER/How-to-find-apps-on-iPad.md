# How to find apps on iPad

**How to find apps on the iPad?**

- Tap the home button of the iPad. If you do not have Home button swipe right to find App Library.

- A search bar would display.

- Enter text in the search bar.

- Hit the search button.

- If the application would available then it would display in the list.
