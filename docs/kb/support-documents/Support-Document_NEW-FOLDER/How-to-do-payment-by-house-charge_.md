# How to do payment by house charge_

**How to do payment by house charge?**

The first customer need to set 'House charge' ON from Back office.

Follow below steps to set On 'House charge' option

- Web Login with your store.

- Setting -\> Module -\> Set On 'House charge'.

![](media/image2.png){width="6.5in" height="4.026042213473316in"}

Follow Below steps to payment by house charge

- Now open RapidRMS application.

- Go to the cash register module.

- From the cash register screen, in top screen tap on customer icon.

- Now from customer page select any one customer and tap on Icon for redirect cash register page.

![](media/image1.png){width="6.5in" height="4.819444444444445in"}

- Now ring up items and tap on the 'Grand Total' button for further process.

![](media/image3.png){width="6.5in" height="4.861111111111111in"}

- Now complete Transaction by 'House Charge' payment.

![](media/image4.png){width="6.5in" height="4.75in"}
