# How to resolve Batch Close RBOUT OF BALANCE Error for Elavon Processor

**[How to resolve Batch Close RBOUT OF BALANCE Error for Elavon Processor]{.underline}**

**1) [Batch Settling through Payment Processor Host]{.underline} :-**

1\. Call technical support at 1 (800) 819-6019\
2. Request to settle the batch\
3. Provide MID and TID number of the terminal

**2)** **[How to get MID & TID Number]{.underline} :-**

1\. Go to payment terminal\
2. On Pax screen, touch 4 corners of the screen in clockwise sequence , starting from top left corner, top right corner, bottom right corner, bottom left corner.**\
\
[Note]{.underline}: Repeat the steps until it prompt for password\
\**
1. Enter today's or tomorrow date as MMDDYYYY\
2. Select Host Settings / Config\
3. Identify MID and TID**\
\**

**After Processor Settles the Batch:**

**[Important]{.underline}: Only perform below steps once processor confirms batch settlement. Otherwise you risk to loose transaction.\
\**
1. Go to payment terminal\
2. Follow above steps to access the menu\
3. Select "Other" from the menu\
4. Select Clear all\
5. Restart the Pax terminal
