# How can visualize blue law menu in BOF

**How can visualize blue law menu in BOF?**

For sales restriction within particular time duration need to set that time period sales restriction via blue law menu.

In back office, blue law menu is there available in e-commerce module.

By default e-commerce module is not active.

For activate e-commerce module follow below procedure.

- Log in with web URL.

- Configure store properly.

- Go to setting.

- Open module tab.

- In module all modules are display with ON/OFF button.

- If any module with option ON, then that module will active in store.

![](media/image1.png){width="6.5in" height="3.3194444444444446in"}

- Keep e-commerce option ON.

- Now in store go to e-commerce menu.

- There is one tab names as blue law.

- From that tab sales restriction can be set.
