# How to add qsr operation in modifier item

**How to add qsr operation in modifier item ?**

In modifier item operation can be added.

For adding operation in modifier item follow below procedure.

- Open rapid rms application.

- Go to inventory management module.

- Tap on modifier items tab.

- Open any item in edit mode.

- There is one field named as qsr operation.

- When create new modifier item then by default two operation are there select and none.

![](media/image2.png){width="6.5in" height="4.777777777777778in"}

- If you want to add more operation then click on arrow available in operation field.

- All operation available in system will display in list.

![](media/image3.png){width="6.5in" height="4.763888888888889in"}

- Multiple operation selection can be possible.

- If you want to add new operation then can be also added by new button.

![](media/image1.png){width="6.5in" height="4.819444444444445in"}

- There is search bar available from where you can search listed operation.

- Added operation will display in modifier item.

- If want to remove operation then from cancel button available at operation name, by there operation can be remove.
