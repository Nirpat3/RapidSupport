# How to link up modifier from QSR group

**How to link up modifier from QSR group ?**

In modifier section qsr group is for separate category of modifier.

There are many ways to link up modifier with qsr group.

For link up modifier from qsr group follow below procedure.

- Open inventory management module.

- Open side pane.

- Tap on qsr group.

- Open any group in edit mode.

- When open qsr group then you can observe one tab named as linked modifier.

- If any of modifier linked with this qsr group then it will display.

![](media/image1.png){width="6.5in" height="4.680555555555555in"}

- There is search bar available for searching modifier from list.

- Click on add items to link other modifier into qsr group.

- Already added modifier will display selected.

- There is one search bar for searching modifiers from list.

![](media/image2.png){width="6.5in" height="3.8472222222222223in"}

- Select other modifiers.

- Click on done button and then hit save.

![](media/image4.png){width="6.5in" height="4.708333333333333in"}

- Added modifier will linked in qsr group and will display in linked modifier list.

- If want to remove any modifier from qsr group then follow same procedure, deselect modifier from list and save.

![](media/image3.png){width="6.5in" height="4.763888888888889in"}

- Modifier would remove from list and also in modifier item qsr group will remove.

- And if click on cancel button then modifier will remove then apply save. Modifier linking with that qsr group will be broken.
