# How to cancel authorised pump

**How to cancel authorized pump?**

When pump is in used then it is in authorized condition.

![](media/image1.png){width="6.5in" height="4.888888888888889in"}

- To cancel any pump click on that pump.

- In window there is one cancel button.

- Hit that cancel button.

![](media/image2.png){width="5.833333333333333in" height="5.083333333333333in"}

- By canceling that pump is again in ideal condition.

![](media/image3.png){width="6.5in" height="4.763888888888889in"}
