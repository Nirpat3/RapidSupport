# How to Print label using Brother printer

**[How to Print labels using Brother printer]{.underline}**

- Before printing labels, we need to connect the brother label printer to the POS system.

> If your device is already connected to your POS system. Click [here!](#Print_Labels)

- **[Please follow the below steps:-]{.underline}**

1.  Click on the settings option ![C:\\Users\\mit shah\\AppData\\Local\\Microsoft\\Windows\\INetCache\\IE\\BU8Y79T3\\WMF-Agora-Settings_BCBCBC.svg\[1\].png](media/image1.png){width="0.17463910761154855in" height="0.17463910761154855in"} available at the bottom of the DashBoard.

![C:\\Users\\mit shah\\Downloads\\MicrosoftTeams-image (8).png](media/image2.jpeg){width="6.268055555555556in" height="3.65541447944007in"}

2.  Enter your quick access password.

3.  Under Hardware settings, click on the Printer option.

4.  To connect brother label printer, click on the **Select Brother Printer** option.

![](media/image3.png){width="6.500694444444444in" height="4.879861111111111in"}

5.  The Brother printer list will appear which is available within your Bluetooth frequency.

6.  Select the brother printer from the list.

7.  Select the Label size you wish to print.

8.  Click on the **Done** button**.**

![](media/image4.png){width="6.500694444444444in" height="5.097222222222222in"}

[]{#Print_Labels .anchor}**[Print labels via brother printer from Inventory Management.]{.underline}**

1.  Go to the Inventory management module.

2.  Click on the **Select** option to select the item from the bottom of the screen.

3.  Select the item from the list that label you want to print.

![](media/image5.png){width="6.5in" height="4.322916666666667in"}

4.  Click on the **Label Print** option and select the **Print with Brother printer** option from the bottom bar.

![](media/image6.png){width="6.5in" height="4.114583333333333in"}
