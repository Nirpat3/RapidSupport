# How to define Cancel transaction in report_

**How to define Cancel transaction in report ?**

It is display number of transaction cancel on sales screen.

In RCR there is one button available at bottom bar to cancel transaction.

When casier don't want to complete transaction then from cancel transaction button it can be cancel.
