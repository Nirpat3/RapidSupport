# How to see history in manual entry_

**[How to see history in a manual entry?]{.underline}**

- User can review all closed invoice details and the purchase order details from History. It also appears purchased and returned item entries in each closed invoice records.

<!-- -->

- Open rapid RMS application.

- Go to manual entry module.

- Open the history section.

![](media/image1.png){width="6.5in" height="4.708333333333333in"}

- All the records display in the list.

![](media/image2.png){width="6.5in" height="4.666666666666667in"}

- User can use date range filter option to search particular invoice records.

![](media/image3.png){width="6.5in" height="4.786458880139983in"}

- When user open any invoice record, all purchased or returned items details of particular invoice will appear.

- User can search an item by UPC barcode, and Item number.

![](media/image4.png){width="6.5in" height="4.694444444444445in"}

- It displays all item details like Image, UPC, description, Item number, Qty OH of Single, Case or pack item, Qty of received Single, Case and Pack item, price/cost of an item.

- If user have updated cost of an item with the received quantity. The total sum of cost of an item will appear in the Extended Cost field.

- User can use the print option to view invoice details.

![](media/image5.png){width="6.5in" height="4.791666666666667in"}

- It will appear Invoice number, Vendor name, Date of created invoice, Total products number and the sum of the cost of all products in Item Info section.
