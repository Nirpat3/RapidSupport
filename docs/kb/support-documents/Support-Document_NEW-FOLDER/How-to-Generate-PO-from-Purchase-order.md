# How to Generate PO from Purchase order

**[How to Generate PO from Purchase order?]{.underline}**

- Generate order option is mostly used to create new open order by customizing the filter option. It allows you to create open order by selecting an option from the filter to generate an order. For example, if you want to purchase items that are sold for today.

<!-- -->

- Go to rapid RMS application.

- Open the purchase order module from the dashboard.

![](media/image1.png){width="6.5in" height="4.168109142607174in"}

- Click on generate order option.

![C:\\Users\\siya\\Downloads\\MicrosoftTeams-image (64).png](media/image2.png){width="6.5in" height="4.87373031496063in"}

- Item can be added by customizing the various filter.

  -----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
     **Fields**    **Description.**
  ---------------- ------------------------------------------------------------------------------------------------------------------------------------------------------------
    Select Date    The User can filter selling records by selecting a custom date range and Predefined time range i.e. Daily, Weekly, Monthly, Quarterly, and yearly options.

     Department    The User can filter records by selecting Department.

   Sub Department  The User can filter records by selecting Sub Department.

       Vendor      The User can filter records by selecting Vendor.

       Group       The User can filter records by selecting Group.

        Tag        The User can filter records by selecting Tag.

     Variation     The User can filter records by selecting a variation.

    Minimum Qty    The User can filter records by entering the Minimum quantity.

    Maximum Qty    The User can filter records by entering the Maximum quantity.

   Backorder list  The User can select Items from the backorder list.
  -----------------------------------------------------------------------------------------------------------------------------------------------------------------------------

For example, as shown in the below image, use want to select those items from a particular Department, Sub -- Department, particular vendors, and that minimum qty is 10.

![C:\\Users\\siya\\Downloads\\MicrosoftTeams-image (65).png](media/image3.png){width="6.5in" height="4.87373031496063in"}

- At the bottom, click on Create Open Order button.

- It will appear an open order title Pop up window.

- Do not enter anything and click on the Save button.

- It will appear items in the Open order items list.

![](media/image4.png){width="6.5in" height="4.8852635608048995in"}

[Please follow the below steps to add items to the purchase order:-]{.underline}

- Select any filter option to add an item to the purchase order.

- Now, Select Move to PO list.

- It will appear a message stating that "Generate PO list".

- Now go to the PO list option.

- All items will appear on the PO list.

**Note:-** When you click on the Move to PO button, All items will appear on the PO list. If there is any item available in open order, they will be removed directly. It will appear as filtered items only.
