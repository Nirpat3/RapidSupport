# How to create GiftCard Payment Type in BackOffice

**[How to create GiftCard Payment Type in BackOffice ?]{.underline}**

- **Please follow the below steps:-**

1.  Login to BackOffice with your credentials.

2.  Tap menu ![hamburger-menu-icon-png-white-24.jpg](media/image1.png){width="0.2075404636920385in" height="0.2075404636920385in"} (located on the top left side)

3.  Click on "**SETTINGS**" Section.

In Settings, Click on "**STORE**".

In Store, Click on "**PAYMENT TYPE**".

![](media/image2.png){width="6.5in" height="2.6006113298337707in"}

4.  In Payment Type, Click on "**ADD**" to create new payment type.

![](media/image3.png){width="6.5in" height="2.048931539807524in"}

5.  Please fillup all details to create "**GIFT CARD**" payment type.

[Payment Name]{.underline} : GIFT CARD

[Payment Code]{.underline} : Any code as per your requirement

[Payment Type]{.underline} : Please select "RapidRMS Gift Card" with the help of scroll arrow.

[Drop Amount]{.underline} : its optional.

[Force Drop Amount]{.underline} : its optional.

Click on "SAVE" button.

![](media/image4.png){width="6.5in" height="2.387494531933508in"}

6.  Once you click on "SAVE", you can see your "GIFT CARD" in your payment type list.

![](media/image5.png){width="6.5in" height="2.9324660979877515in"}
