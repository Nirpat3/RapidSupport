# How to create new Fees and Deposits

> **[How to Create New Fees and Deposits?]{.underline}**

[**Fee/Deposit is mainly used when stores want to collect extra charges on any item**.]{.mark}

- [Open Rapidrms application.]{.mark}

- [Go to the Inventory management module.]{.mark}

- [Enter your quick access password.]{.mark}

- [At the top left, click on the three lines (Hamburger) button]{.mark} ![C:\\Users\\mit shah\\AppData\\Local\\Microsoft\\Windows\\INetCache\\IE\\ZY2HW6WI\\Menu-thin-circular-outlined-button.svg\[1\].png](media/image1.png){width="0.2247561242344707in" height="0.2247561242344707in"}. []{.mark}

- Select the Fee and Deposit option from the Item Master menu.

![](media/image2.png){width="6.5in" height="4.888888888888889in"}

- [Select the Fee/Deposit option from the list.]{.mark}

- [It will appear existing Fee/Deposit list.]{.mark}

- [At the bottom, click on Add button.]{.mark}

![](media/image3.png){width="6.5in" height="4.369792213473316in"}

+----------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **[Field]{.mark}**   | **[Description]{.mark}**                                                                                                                                                                                                                  |
+======================+===========================================================================================================================================================================================================================================+
| [Name]{.mark}        | [It describes the Fee/Deposit Name.]{.mark}                                                                                                                                                                                               |
+----------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [Fee/Deposit]{.mark} | [Select the Fee and Deposit Radio button. Note: You can select any one of them.]{.mark}                                                                                                                                                   |
+----------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [Slab]{.mark}        | - [**[Fix:-]{.underline}** It allows you to create a fixed charge to impose a charge on an item or service. Fix charge remains the same regardless of an item\'s price.]{.mark}                                                           |
|                      |                                                                                                                                                                                                                                           |
|                      | - [**[By Amount:-]{.underline}** It allows you to create a charge in a particular range to impose a charge on an item or service. The charges are applicable only when any item or service price is within a specified range.]{.mark}     |
+----------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [Charge Type]{.mark} | - [**[Dollar:-]{.underline}** It allows you to create charges in Dollar charge type.]{.mark}                                                                                                                                              |
|                      |                                                                                                                                                                                                                                           |
|                      | - [**[Percentage:-]{.underline}** It allows you to create charges in Dollar charge type. charge amounts do vary as per the item's price.]{.mark}                                                                                          |
|                      |                                                                                                                                                                                                                                           |
|                      | - [**[Price at POS:-]{.underline}** If you select the Price at POS radio button instead of Dollar and percentage charge type. It will ask ysou to enter the charge amount when you ring up an item or service from Cash Register.]{.mark} |
+----------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| [Taxes]{.mark}       | [The taxes option appears when you create a charge type. If you turn on the tax toggle button, It will allow you to select tax to apply on Fee/Deposit.]{.mark}                                                                           |
+----------------------+-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

- [Enter Fee/Deposit Name.]{.mark}

<!-- -->

- [Select the Fee and Deposit radio button.]{.mark}

- [When you select the Fee and Deposit option, it will appear slab section.]{.mark}

- [There are two charge types available **1)** Fix **2)** By Amount.]{.mark}

**[[Fix:-]{.underline}]{.mark}**

- [Select the radio button to enter the value in dollar or Percentage.]{.mark}

- [Enter the value in the Dollar or Percentage section.]{.mark}

- [Click on the Save button.]{.mark}

- [If you want to apply tax, turn on the toggle button, It will appear the list of tax.]{.mark}

- [Select any existing tax. You can as many taxes as you want to select.]{.mark}

- [Click on the Done button.]{.mark}

- [Selected tax will appear.]{.mark}

![](media/image4.png){width="6.5in" height="4.021739938757655in"}

![](media/image5.png){width="6.5in" height="3.9427088801399823in"}

**[[By Amount:-]{.underline}]{.mark}**

- [If you select the By range section.]{.mark}

- [Click on add]{.mark} ![C:\\Users\\siya\\AppData\\Local\\Microsoft\\Windows\\INetCache\\IE\\GNNPZ8E6\\plus-310108_960_720\[1\].png](media/image6.png){width="0.21779746281714785in" height="0.21779746281714785in"} [button from the end of the add range section.]{.mark}

- [Enter the Start and end range amount.]{.mark}

- [Enter the value in the Dollar or Percentage section.]{.mark}

- [Click on the Save button.]{.mark}

- [If you want to apply tax, turn on the toggle button, It will appear the list of tax.]{.mark}

- [Select any existing tax. You can as many taxes as you want to select.]{.mark}

- [Click on the Done button.]{.mark}

- [Selected tax will appear.]{.mark}

**[Note:-]{.underline}** The Fee/Deposit charges will apply only when any item or service price is set within specified range.
