# How Can I Create New Role from Role Management

**[How Can I Create New Role from Role Management?]{.underline}**

- **What is Role?**

<!-- -->

- The role is rights that you can assign to a user to allow to access certain modules or features of the Rapid RMS application.

<!-- -->

- **What is Role Management?**

<!-- -->

- Role Management helps you manage authorization, which enables you to specify the resources that users in your application are allowed to access Rapid RMS Application. Role management is mainly used to create roles, and assign a role to the user. Such as Admin, Cashier, and so on.

**[Please follow the below steps to create a new role:-]{.underline}**

1.  Login into the Back office with your credentials.

2.  Click on the Settings option.

3.  Click on the Staff menu.

4.  Select Role Management.

![](media/image1.png){width="6.5in" height="2.9992607174103236in"}

5.  Tap on Add tab.

![](media/image2.png){width="2.467361111111111in" height="0.5599442257217848in"}

6.  Enter Role Name.

7.  It will appear the list of all modules, sub-modules, and features of the RapidRMS application.

8.  Select the check box next to the modules.

For example, If you want to let users to access the Inventory Management module. Select the checkbox next to the Inventory Modules.

![](media/image3.png){width="6.500068897637795in" height="2.8152176290463693in"}

If you want to let users access certain features or sub modules of the Inventory Management module.

9.  Click on the plus button available beside module name.

10. It will appear sub-module list.

11. Select the check box next to the sub-module.

![C:\\Users\\siya\\Downloads\\Screenshot_49.png](media/image4.png){width="6.5in" height="2.8582983377077866in"}

12. Tap on the Save button.
