# How to set memo for cancel transaction

**How to set memo for cancel transaction ?**

When user want to add memo pop up for cancel transaction then it can be set from BOF.

Follow below procedure to set memo for cancel transaction.

- Log in with web URL.

- Go to setting tab.

- Select memo option.

- When memo opens then many of ON/OFF switch display for different options.

- Review there is one option for cancel transaction.

- Keep memo button OFF for cancel transaction option from BOF.

![](media/image5.png){width="6.5in" height="3.3333333333333335in"}

- Now open rapid rms application and go to cash register.

- Ring up some items and hit cancel transaction button which is available at button bar in POS.

- There should not be any memo pop up blink for cancel transaction.

![](media/image3.png){width="6.5in" height="4.777777777777778in"}

- Now from BOF keep memo option of cancel transaction ON.

![](media/image6.png){width="6.5in" height="3.7522714348206474in"}

- Restart application, As any change from BOF will reflect after restart application.

- Ring up some items and now hit cancel transaction button which is available at bottom bar.

![](media/image4.png){width="6.5in" height="4.819444444444445in"}

- Give yes for clear screen.

- Pop up for memo would blink for adding reason for cancel transaction.

![](media/image1.png){width="6.5in" height="4.777777777777778in"}

- Select proper reason for cancel transaction or enter valid reason for cancel transaction and save data.

- By this way memo text can be set for cancel transaction.

![](media/image2.png){width="6.5in" height="4.680555555555555in"}
