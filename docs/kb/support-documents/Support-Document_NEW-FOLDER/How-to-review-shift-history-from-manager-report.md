# How to review shift history from manager report

**[How to review Shift History from Manager Report?]{.underline}**

- **Please follow the below steps:-**

In shift history all the shifts of current month is display with related data.

1.  Open rapid rms application.

2.  Go to daily report.

3.  Open manager report.

4.  Tap to shift history which is placed at the first column.

![review-shift-history-from-manager-report](media/image1.png){width="6.5in" height="4.799121828521435in"}

5.  In shift history, all the shifts are displayed with whole data.

6.  In right-side, all shifts are listed with open close time and date, register & branch name, open/close status and with total shift amount.

![review-shift-history-from-manager-report](media/image2.png){width="6.5in" height="4.731060804899387in"}

7.  Whichever shift you select, in left side all detail information of that particular shift will display.

8.  If you want to see another month gift then there is one menu named as a date.

9.  Select month and year in the scrollbar.

10. And apply to save.

11. Selected month shifts will display in the list.

![review-shift-history-from-manager-report](media/image3.png){width="6.5in" height="4.685970034995625in"}

12. In shift history, all the shifts are listed but if you want to review a particular register's shift then you can filter by the register.

13. Apply filter.

14. Select register.

15. Selected registered shift would display.
