# How to create Open order

**[How to create Open order from PO?]{.underline}**

- Go to rapid RMS application.

- Click on the Purchase order module from the dashboard.

![](media/image1.png){width="6.5in" height="4.168109142607174in"}

- Click on the PO list option.

![](media/image2.png){width="6.5in" height="4.694444444444445in"}

- Now click on the search option from the bottom of the screen.

- Select items which you want to add to the purchase order from the list.

- After selecting items, click on add + button from the bottom of the screen.

![](media/image3.png){width="6.5in" height="4.8668318022747155in"}

- Enter the qty which you want to order for the item.

- Select the item to create open order.

- After selecting items click on "**create open order"** from the bottom of the screen.

- It will appear an open order title pop-up window to enter the Open order name.

- It is a mandatory to enter the Open order name.

![](media/image4.png){width="6.5in" height="4.1630435258092735in"}

- Enter the Open order name.

- Click on the Save button.

- It will move to Open order and all items details will appear in Open order items list.

![](media/image5.png){width="6.5in" height="4.010310586176728in"}
