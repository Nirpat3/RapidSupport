# How to void in clock in out

**How to void the clock in-out?**

When there is any shift which is not required then the user can void that shift and also if there is an entry which is void before then that shift can be unavoidable.

- Open rapid RMS application.

- Go to the Clock in-out module.

  - Swipe entry.

- There is one option void, Apply that option.

![](media/image1.png){width="6.5in" height="4.611111111111111in"}

- After void that entry display with void watermark.

![](media/image2.png){width="6.5in" height="4.722222222222222in"}

- If you want to unvoid entry then again swipe that entry.

![](media/image4.png){width="6.5in" height="4.694444444444445in"}

- After unvoid entry will display as earlier.

![](media/image3.png){width="6.5in" height="4.791666666666667in"}
