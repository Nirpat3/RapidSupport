# How to identify which email id associated with their app store

**[How to identify which email id is associated with their app store.]{.underline}**

- Start IPad.

- Hit the app store icon from the dashboard.

![](media/image1.png){width="6.5in" height="3.6703346456692914in"}

- Click on the account icon displayed in the top right corner.

![](media/image2.png){width="6.5in" height="4.820312773403325in"}

- Account detail for app store mail id would display.

![](media/image3.png){width="6.5in" height="4.208725940507437in"}
