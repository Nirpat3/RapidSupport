# How to suspend pump_

**How to suspend pump ?**

Suspend pump that means if user want to hold continue pump then there is one option to hold pump and that is suspend.

- Open RCR and approve one pump.

- Now start fueling.

- When pump is dispense and want to hold that pump fueling then click on that pump.

![](media/image2.png){width="6.5in" height="4.777777777777778in"}

- From pop up click on suspend option.

![](media/image1.png){width="6.5in" height="4.819444444444445in"}

- When click suspend then fuel dispensing will stop.

- Pump will display with status suspend.

![](media/image3.png){width="6.5in" height="4.805555555555555in"}
