# How to set up a drawer connection with the Printer

**[How to set up a Cash drawer connection with the Printer?]{.underline}**

- **Please follow the below steps:-**

1.  First, you need **RJ cable** to connect cash drawer to printer. The RJ cable looks similar to phone cables. You may call **Cash drawer cable.**

2.  To connect Cash drawer to printer you need to plug in RJ cable at the back of the printer.

3.  The Cash Drawer cable **will be labeled** on each end to indicate which end of the cable plugs into the Cash Drawer and which end plugs into the Printer.

4.  Plug in one end of RJ cable directly below the Cash drawer.

5.  Plug in another end of the cable back to the Printer where is written **letter DK**.

![Star TSP100 cash drawer port](media/image1.png){width="6.5in" height="1.7031342957130358in"}

6.  On other printers, the cash drawer port may be labeled differently. Please refer to the printer manual to confirm where the cash drawer port is.
