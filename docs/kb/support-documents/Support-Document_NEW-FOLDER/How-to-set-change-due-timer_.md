# How to set change due timer_

**How to set change due to timer?**

Change due timer is for setting time duration after complete transaction how much change due to screen display.

Change due timer can be set from the setting module.

- Open rapid RMS application.

- Go to the setting module.

- Open tender configuration.

- There is one option change due to timer.

- Keep that option ON.

![](media/image1.png){width="6.5in" height="4.736111111111111in"}

- There is one menu for set timing.

- How much time is set in change due to timer, in cash register after complete transaction change screen display for that much time.
