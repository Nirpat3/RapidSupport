# How to create a New Sub Department

**[How to create a New Sub Department?]{.underline}**

- Sub-department is a master from which you can create a new sub-department, Edit and delete an existing Sub-department from Inventory Management.

- You can access the Sub-department master from the Inventory Management module.

> **[Please follow the below steps to Create New Sub Department:-]{.underline}**

- Open RAPID RMS application.

- Select inventory management module from Dashboard.

![](media/image1.png){width="6.5in" height="4.597222222222222in"}

- Click on three line option available from the top left side corner of the screen.

- Select sub-department master from the list.

- It will appear sub-department list.

<!-- -->

- To create a new sub-department, click on Add button from the bottom of the screen.
