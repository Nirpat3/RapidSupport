# RJR ScanData File Submission Process

Version: 1.0

# **[RJR FILE SUBMISSION PROCESS]{.underline}**

- Once you finish registration process with RJR then you will receive welcome email from RJR with process details along with TEST FILE request. Please refer below screensort.

![](media/image1.png){width="6.5in" height="2.842409230096238in"}

- Now, you have to submit RJR TEST FILE from your store.

Please follow the below steps and screenshot to download the file.

1.  Login to BackOffice with your credentials.

2.  Tap menu ![hamburger-menu-icon-png-white-24.jpg](media/image2.png){width="0.2075404636920385in" height="0.2075404636920385in"} (located on the top left side).

3.  Click on "**SCAN DATA**" Module.

4.  Click on "**RJR FILE**" option.

5.  Select Date Period.

6.  Click **on Download RJR.txt** Tab.

You can change file name (file name example: ABCRetail_11162018_TEST).

![](media/image3.png){width="6.5in" height="3.078946850393701in"}

- Once file downloaded in your computer then you can submit your file with the following web portal.

<https://mfthub.msa.com/Login>

![](media/image4.png){width="6.5in" height="2.7613265529308837in"}

**[NOTE]{.underline}** : Please take your Sales Representative help to submit your file.
