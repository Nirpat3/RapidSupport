# How to view customer profile_

**How to view customer profile?**

When customers want to review the updated profile then it can be possible from the customer loyalty module.

- Open rapid RMS application.

- Go to the customer loyalty module.

![](media/image1.png){width="6.5in" height="4.763888888888889in"}

- Hit view option from the customer list.

![](media/image2.png){width="6.5in" height="4.763888888888889in"}

- A customer profile will open and updated data will display.

![](media/image3.png){width="6.5in" height="4.791666666666667in"}
