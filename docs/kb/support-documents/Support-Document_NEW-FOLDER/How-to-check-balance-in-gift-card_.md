# How to check balance in gift card_

**How to check the balance in a gift card?**

In store, if gift card is available then the balance of that gift card can be check from cash register module.

For gift card balance check follows below procedure.

- Open rapid RMS application.

- Go to the cash register.

- At the bottom bar, there is one option named as more click on that option.

- Click on a gift card from all of them.

![](media/image1.png){width="6.5in" height="4.819444444444445in"}

- When hit gift card then there are two options available.

- From that click on check balance.

![](media/image2.png){width="6.5in" height="4.680555555555555in"}

- Enter gift card number which gift card is available in store.

> **NOTE**: If you enter a number which is not active then it will ask you to activate this entered number card. If hot yes then a new gift card will be added.
>
> ![](media/image3.png){width="6.5in" height="4.819444444444445in"}

- After entering the gift card number click on check balance.

- Balance amount will display.

![](media/image4.png){width="6.5in" height="4.680555555555555in"}

- If you want to print balance receipt then it can be possible by print option.
