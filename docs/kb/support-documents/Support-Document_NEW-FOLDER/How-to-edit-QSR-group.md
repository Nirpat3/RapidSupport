# How to edit QSR group

**How to edit QSR group ?**

From inventory group can be edit and modifier available in group also can edit other detail of QSR group.

For edit Group go to below procedure.

- Open inventory management.

- Go to QSR group tab.

- There should be list of available qsr group.

- Open any group in edit mode for that click on that group name from list it will be open.

- All detail of that group will display.

![](media/image1.png){width="6.5in" height="4.694444444444445in"}

- Change detail.

- Click on save.

- Edited detail will save and group will edit.
