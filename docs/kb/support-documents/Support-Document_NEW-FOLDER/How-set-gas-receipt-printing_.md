# How set gas receipt printing_

**How set gas receipt printing?**

When in cash register if the transaction is of gas and after completion of the gas transaction when a user wants to pint receipt then it can be set from the setting module.

- Open rapid RMS application.

- Go to the setting module.

- Open tender configuration.

- There is one option as a print gas receipt.

- Keep that option ON.

- In bill when the transaction is of gas and printer is connected then after the completion of payment gas receipt will print.
