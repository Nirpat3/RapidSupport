# How can I email and preview any report from Manager Report

**[How can I email/preview any report from the manager report?]{.underline}**

- **Preview/Email**

<!-- -->

- When users want to email or preview any report, user can open that report from the manager report.

- Select report.

- Click on pdf/email option.

![](media/image1.png){width="6.5in" height="4.722222222222222in"}

- When the user selects the email option, the Email popup window will open.

- Enter proper email id.

- Enter Email Add and click on the send button.

- The selected report will be sent to enter an email address.

![](media/image2.png){width="6.5in" height="4.708333333333333in"}

- When users select the preview option, the report overview will appear.

![](media/image3.png){width="6.5in" height="4.791666666666667in"}
