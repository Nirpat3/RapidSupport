# How to undo restocked history_

**How to undo restocked history?**

When items are restocking from sales history then it will move from sales history to restocking history.

But if you want to move again that item to sales history then you can do that reverse process by Undo option.

- Open rapid RMS application.

- Go to restock module.

- Open restocking history.

- Initially undo option is disable as you select item(S) it will enable.

- Select items from the list.

- Apply the undo option.

![](media/image2.png){width="6.5in" height="4.680555555555555in"}

- Items will remove from restocking history and listed in sales history again.
