# How to set up a Ingenico gateway from Back office

**[How to set up an Ingenico gateway from Back-office?]{.underline}**

- **Please follow the below steps:-**

1.  Login to BOS with your user credentials.

2.  Tap menu ![hamburger-menu-icon-png-white-24.jpg](media/image1.png){width="0.2075404636920385in" height="0.2075404636920385in"} (located on the top right side)

3.  Tap settings.

4.  Tap store.

5.  Scroll to see the Gateway option. Select the Gateway option.

![](media/image2.png){width="6.5in" height="2.9866404199475065in"}

6.  Select the Add option.

![](media/image3.png){width="6.5in" height="2.9767989938757657in"}

7.  Click on the Payment drop menu to select **a Rapid Pay Tech** payment gateway.

![](media/image4.png){width="6.5in" height="4.535982064741908in"}

![](media/image5.png){width="6.5in" height="2.9866404199475065in"}

  ---------------------------------------------------------------------------------------------------------------------------------------------------------------------
  **Fields**                **Description**
  ------------------------- -------------------------------------------------------------------------------------------------------------------------------------------
  Description               Please enter the Payment gateway name.

  Gateway ID                Please enter 23 for Gateway Id

  User name                 Please enter your store email Id into Username.

  Password                  Please enter your store password into Password.

  Operation Mode            Select Live mode.

  Allow manual processing   If you want to do the process manually while doing the payment process from your POS System, select **Allow manual processing** checkbox.

  IsDefault                 Select the **Isdefault** check box to use the payment gateway by default from your POS.

  Gateway Platform          Select the **Gateway platform** on which platform you want to use the selected gateway.
  ---------------------------------------------------------------------------------------------------------------------------------------------------------------------

8.  After entering all details, Tap on the **Save** button.
