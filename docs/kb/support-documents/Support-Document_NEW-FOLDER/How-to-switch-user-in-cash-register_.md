# How to switch user in cash register_

**How to switch user in the cash register?**

In cash register when want to switch user then cash register allows switch user.

- Open rapid RMS application.

- Go to the cash register module with one user.

- When you want to switch user then click on top right corner button.

- Apply the switch user option.

![](media/image1.png){width="6.5in" height="4.763888888888889in"}

- Enter the passcode for another user.

- Click on Enter.

- It will switch to another user.
