# How to Cancel transaction_

**How to Cancel transaction ?**

When user do not want to complete transaction then cashier can cancel the whole bill.

Open rapid RMS application.

Go to the cash register.

Ring up items.

At the bottom bar there is one option"cancel transaction."

Hit that option.

![](media/image2.png){width="6.5in" height="4.722222222222222in"}

It will display confirmation pop up.

Apply yes.

![](media/image1.png){width="6.5in" height="4.777777777777778in"}

All ring up item will remove and transaction cancel

Note : In offline mode cancel transaction will display disable. Without internet transaction can not be cancel
