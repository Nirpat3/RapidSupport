# How to add QSR sub group

**How to add QSR sub group ?**

In inventory there are groups and sub groups are available for adding modifier category.

For adding new QSR sub group follow below procedure

- Open inventory management.

- Open side pane available on left side.

- In item master there is one section available name as QSR sub group.

- Click on that.

- All QSR sub group will listed.

![](media/image4.png){width="6.5in" height="4.736111111111111in"}

- At bottom bar there is one icon available"+".

- Click on that add button.

- When click on add button then one window will open for adding new QSR sub group with require detail.

![](media/image6.png){width="6.5in" height="4.597222222222222in"}

- First is QSR sub group name. This is text field. Add proper name for QSR sub group.

- QSR sub group name should be unique.

- Another field is QSR sub group code.

- Add any convenient QSR sub group code.

- QSR sub group code is compulsory and unique field.

- With these two option QSR sub group can be save by save button which is available at the bottom of window.

![](media/image3.png){width="6.5in" height="4.583333333333333in"}

- There are two radio button for selecting modifier in sub group.

- If you select pick only one then in item only one sub group can be added.

- If you select multiple choice then in item you can add multiple modifier from this sub group belonging.

![](media/image8.png){width="6.5in" height="4.583333333333333in"}

- In QSR group there are another useful fields are also there.

- QSR operation : QSR operation is not text field. From available operation few of operations can be select.

- For adding operation in QSR group, click on arrow which available in field QSR operation.

- When open operation window then select operation from available list. Also from add button new operation can be created.

- At a time multiple operations are allow to enter in QSR group.

![](media/image7.png){width="6.5in" height="4.805555555555555in"}

- Select operation which you want to add and click on save.selected operations are added.

- In QSR group there are another useful fields are also there.

- QSR operation : QSR operation is not text field. From available operation few of operations can be select.

- For adding operation in QSR group, click on arrow which available in field QSR operation.

- When open operation window then select operation from available list. Also from add button new operation can be created.

- At a time multiple operations are allow to enter in QSR group.

![](media/image7.png){width="6.5in" height="4.805555555555555in"}

- Select operation which you want to add and click on save.selected operations are added.

![](media/image1.png){width="6.5in" height="4.736111111111111in"}

- Now next field is selection of QSR group.

- For qsr sub group, qsr group selection is compulsory field.

- Every sub group is belonging to any one of qsr group.

- At a time only one qsr group selection is possible.

- Click on side arrow which is available in field of qsr group.

- All available qsr group will listed.

- Select any one group.

![](media/image9.png){width="6.5in" height="4.819444444444445in"}

- There is also search bar available to search qsr group from list.

- Also you can add new qsr group from add button.

- Added qsr group will added to qsr sub group.

![](media/image2.png){width="6.5in" height="4.611111111111111in"}

- Now click on save button.

- All added detail will save in qsr sub group.

- Added qsr sub group is listed in selected qsr group detail.

![](media/image5.png){width="6.5in" height="4.638888888888889in"}

Note : In one qsr sub group only one qsr group can be added means its one to one relation, BUt in one qsr group multiple sub group can be possible this is one to many relationship.
