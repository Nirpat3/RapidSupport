# How to delete item from invoice_

**How to delete an item from the invoice ?**

- When from counting invoice if you want to delete item then you can remove that item by swipe item.

- From recall, invoice item can be deleted.

- From reconcile and history item can not be deleted.

- Before reconcile if you want to delete item then only you can remove the item.
