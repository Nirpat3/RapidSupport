# How to view counting item status_

**How to view counting item status?**

When in-store from any of place item record for counting then you can see the status of that item.

- Open the inventory count module.

- Go to recall or join or any module in which counted item list available.

- Click on view option.

![](media/image1.png){width="6.5in" height="4.736111111111111in"}

- There are four options display at the bottom bar.

- In all, all the items display which is considered in count.

![](media/image2.png){width="6.5in" height="4.722222222222222in"}

- In Unmatched, Item(S) is a display of which stock available in-store(QTY OH) not match with counted item qty.

![](media/image3.png){width="6.5in" height="4.847222222222222in"}

- In Matched, Item(S) is a display of which stock available in-store(QTY OH) match with counted item qty.

![](media/image4.png){width="6.5in" height="4.791666666666667in"}

- In Uncounted, Item(S) display which is available in store but not considered in count.
