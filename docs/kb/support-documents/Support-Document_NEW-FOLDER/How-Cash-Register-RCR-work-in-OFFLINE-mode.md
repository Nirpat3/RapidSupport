# How Cash Register (RCR) work in OFFLINE mode

**[How Cash Register (RCR) work in OFFLINE mode?]{.underline}**

# **Overview**

RapidRMS supports Cash Register (RCR) module in offline mode. Rest of the modules work only in online mode.

- **What is OFFLINE mode ?**

**Offline Mode:** When communication is lost with RapidServer, it is called offline mode. It happens in four cases:

1.  Switch off Wifi from device settings

2.  Switch off router

3.  When Internet connectivity is down

4.  When RapidServer is down

NOTE : In 1^st^ scenario, Printer and Drawer will be not working.

- **When App goes in offline mode, only following icons are available in Dashboard:**

1.  Cash Register

- Retail

- Petro

2.  Clock-In/Out

3.  Sign-In/Out

4.  Settings
