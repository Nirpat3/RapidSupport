# Close order in inventory management

**[Close order in Inventory Management]{.underline}**

When from open order save invoice then it moves in close order.

- Open inventory management module.

- Go to close order.

![](media/image1.png){width="6.5in" height="4.791666666666667in"}

- Close order can display weekly or custom date.

![](media/image2.png){width="6.5in" height="4.722222222222222in"}

![](media/image3.png){width="6.5in" height="4.708333333333333in"}

- In close order item cannot be added, delete or item qty cannot be changed.

- You can preview or email invoice.

- Close order is like in/out history.
