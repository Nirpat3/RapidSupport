# How to Logout from PO module

**[How to Logout from PO module?]{.underline}**

- To logout from Purchase order module.

- There is a Logout option available in the Purchase order.

- Click on Log out option.

![](media/image1.png){width="6.5in" height="4.75in"}

- It will close the Purchase order module and you will redirect to the Dashboard screen.
