# How to review centralized shift from manager report

**How to review centralized shift from manager report?**

In daily report, centralized shift is the reporting section for all centralized shift.

In a centralized shift report, entries are displayed day wise.

Day wise one entry is a display and in that one shift entry, all the shifts which are open and close on that particular day would display.

- Open rapid RMS application.

- Go to the manager report.

- Open centralized shift.

- All the shifts are displayed day wise.

![](media/image5.png){width="6.5in" height="4.736111111111111in"}

- If you want to view previous month shift history then you can do by date filter.

- In centralize shift it will not filter by date its only filter by month.

![](media/image3.png){width="6.5in" height="4.805555555555555in"}

- In centralized shift you can view register wise history also.

- Tap on any shift entry.

- There is a two-section display.

- History and custom.

![](media/image1.png){width="6.5in" height="4.75in"}

- In history all viewed history display according to register.

- In custom you can select different register and can view shift history of one day completed by that register.

- Select custom.

- It will display all the registers by which shift are open and close.

- Select some register.

- Apply done.

- Combined register total data will display and that history will added to viewed history list.
