# How to create a new tender from POS

**[How to create a new tender from POS?]{.underline}**

- Click on the settings option ![C:\\Users\\mit shah\\AppData\\Local\\Microsoft\\Windows\\INetCache\\IE\\BU8Y79T3\\WMF-Agora-Settings_BCBCBC.svg\[1\].png](media/image1.png){width="0.17463910761154855in" height="0.17463910761154855in"} available at the bottom of the DashBoard.

![C:\\Users\\mit shah\\Downloads\\MicrosoftTeams-image (8).png](media/image2.jpeg){width="6.262498906386702in" height="3.5326082677165354in"}

- Enter your quick access password.

![](media/image3.png){width="6.5in" height="3.881519028871391in"}

- Select the Tender configuration option from the right side menu.

![](media/image4.png){width="6.5in" height="4.230227471566054in"}

- On the below right side, click on the payment list button.

- When you click on the Payment list button, It will appear the list of an existing tender.

![](media/image5.png){width="6.5in" height="3.9124989063867015in"}

- To create a new tender, click on add button from the bottom of the screen.

![C:\\Users\\siya\\Downloads\\MicrosoftTeams-image (88).png](media/image6.png){width="6.500905511811023in" height="4.36956583552056in"}

  ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
     **FIeld**    **Description**
  --------------- ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
   Payment Name   It describes the **name** of the tender. Payment name is a mandatory field.

   Payment Code   It describes the code of the tender. Payment code is a mandatory field.

   Payment type   It describes the payment type. By default, it is selected other payment type.

    On/Off Memo   It allows the user to enter any remarks or memos for the tender.

    Drop Amount   It allows user to remove excess money from the register to deposit it into a secure location at the store. When you have selected the drop amount check box, it allows the user to drop by the tender. By default, the drop amount check box is unchecked.
  ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

- Enter Payment Name.

- Enter Payment Code

- Select payment type from the drop-down list. Ex:- Credit, Debit, Cash, Check, etc.

- Tap or click on the save button.
