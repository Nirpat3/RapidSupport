# How can Admin or Store view accept or reject the user request from the Back office

**[How can Admin or Store view, accept or reject user requests from the back-office?]{.underline}**

> **Please follow the below steps:-**

1.  Open Back office.

2.  Login with your store or admin credentials.

3.  Go to settings.

4.  Click on **Staffs** and select the **User list** option.

5.  It will appear User request in the list.

6.  It will appear Unique Employee number, First name, and Last name of every user on the list.

7.  To accept a user request, click on the action button and select Accept/Add button.

8.  If you do not want to accept the user request, click on the action button and select the Reject button.
