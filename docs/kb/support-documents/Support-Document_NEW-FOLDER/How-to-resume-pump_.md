# How to resume pump_

**How to resume pump ?**

Suspended pump can be resume.

When for a while if any pump is suspended then you can resume that pump from RCR.

- Open RCR and go to gas section.

- If any pump is suspended then it will display with suspended symbol.

![](media/image3.png){width="6.5in" height="4.763888888888889in"}

- Click on that pump.

- Window will open.

- Select resume from that option.

![](media/image1.png){width="6.5in" height="4.736111111111111in"}

- Pump will again start.

![](media/image2.png){width="6.5in" height="5.0in"}
