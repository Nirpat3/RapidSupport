# How does Vendor Inventory work

**[How does Vendor Inventory work?]{.underline}**

Vendor Inventory is an option to view which items are associated with a particular vendor.

- Open rapid RMS application.

- Go to the inventory management module.

- Click on the three-line button available at the top left side corner of the screen.

- Scroll down to select the Vendor Inventory option.

- Select the Vendor Inventory option.

- Click on the "\>" arrow available at the top right side corner of the screen.

![](media/image1.png){width="6.5in" height="4.402777777777778in"}

- It will appear a list of all vendors of the store.

- When you select any vendor, It will appear the list of items that are associated with the vendor.

- Click on a vendor to select the vendor from the list.

- When you select any vendor the radio button will appear in yellow available at the end of the record.

- You can select more than one vendor from the vendor list.

![](media/image2.png){width="6.5in" height="4.3369564741907265in"}

- At the bottom, click on the Done button.

![](media/image3.png){width="6.5in" height="3.743173665791776in"}

- It will appear the list of items like Item details like UPC, Item Name, Qty OH, Minimum and Maximum stock limit of the item.
